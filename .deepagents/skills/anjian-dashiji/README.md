# 案件大事记整理方法论

## 简介

本skill提供案件大事记的系统整理方法，用于：

- 从案情材料中提取时间信息
- 识别和分类案件相关事件
- 提炼事件核心内容
- 按时间顺序组织事件脉络
- 生成规范的案件大事记表格

## 使用场景

**适用场景**：
- 律师梳理案件事实和时间线
- 法官整理案件经过
- 法律工作者制作案情摘要
- 案件分析的前期准备工作

**不适用场景**：
- 法律评价和分析（仅整理事实）
- 证据审查（不涉及证据效力判断）
- 请求权分析（属于另一模块）

## 输出示例

```markdown
# 案件大事记

| 时间 | 事件 | 主要内容 |
|------|------|----------|
| 2023年1月15日 | 合同签订 | 甲乙双方签订《房屋买卖合同》，约定甲将某房屋出售给乙，价款200万元 |
| 2023年2月1日 | 首付款支付 | 乙向甲支付首付款60万元 |
| 2023年3月15日 | 过户登记 | 双方办理房屋过户登记手续，房屋登记至乙名下 |
| 2023年4月1日 | 尾款催告 | 甲催告乙支付剩余房款140万元 |
| 时间未知 | 纠纷产生 | 乙以房屋存在质量问题为由拒绝支付尾款 |
```

## 快速使用指南

### 最简流程（简单案件）

1. 通读案情材料
2. 逐句提取时间和事件
3. 按时间排序
4. 生成表格

### 完整流程（复杂案件）

1. 分类阅读（按材料类型分别处理）
2. 初步提取所有时间节点
3. 时间标准化处理
4. 事件分类和内容提炼
5. 时间排序和冲突处理
6. 生成最终表格

## 与其他模块配合

本skill为案件整理的基础模块，分析时应配合：

- **请求权基础模块**：在大事记基础上进行请求权分析
- **证据分析模块**：在大事记基础上审查证据
- **案由选择模块**：根据大事记识别的法律关系选择案由

使用流程：
1. 使用本skill整理案件大事记
2. 输出时间线和事件脉络
3. 为后续法律分析提供事实基础
