# 事实整理法

从案件材料中提取和整理案件事实的规范方法。

## 核心原则

**简要叙述原则**：案件概要中的事实部分应是一段简要叙述，而非详细时间线。详细时间线由大事记模块处理。

## 整理方法

### 一、摘要式叙述

将案件事实以一段话简要说明，突出核心要素：

```markdown
## 案件事实

[以一段话简要叙述案件事实，包含：当事人关系、交易/事件性质、关键行为、争议起因]

示例：
碧桂园公司与凯利公司于2017年7月15日签订《资产转让合同》，约定凯利公司将三亚蓝月湾海景酒店公寓项目转让给碧桂园公司，转让价款暂定7亿元，碧桂园公司支付诚意金3.2亿元，凯利公司应于2017年10月30日前完成目标地块规划调整及用地性质变更。碧桂园公司于2017年8月7日支付3.2亿元。凯利公司未能在约定期限完成规划调整，主张因政府政策调整属于不可抗力。碧桂园公司于2017年11月12日发函解除合同，要求退还诚意金及违约金。凯利公司在收到3.2亿元后向股东圣方公司、张伟男转账共计5371万元，引发股东责任争议。
```

### 二、关键要素提取

摘要叙述应包含以下关键要素：

| 要素 | 说明 | 必要性 |
|------|------|--------|
| 当事人关系 | 双方主体身份、合同关系 | 必要 |
| 交易/事件性质 | 买卖、借贷、侵权等 | 必要 |
| 核心义务 | 合同约定的主要义务 | 必要 |
| 关键行为 | 签约、付款、违约等关键行为 | 必要 |
| 争议起因 | 纠纷产生的原因 | 必要 |
| 涉案金额 | 如有具体金额应注明 | 如有 |

### 三、篇幅控制

- 一般控制在200-400字
- 涉及多笔交易或复杂关系的案件可适当延长
- 避免冗长细节，保持"概要"性质

### 四、剪裁规则

事实叙述有明确的"写"与"不写"边界：

**必须写（4要素）**：
- 当事人关系：谁与谁是何种法律关系
- 交易/事件性质：买卖、借贷、侵权等
- 核心行为：签约、履约、违约、侵权等关键行为
- 争议起因：纠纷产生的直接原因

**禁止写（3排除）**：
- 具体日期序列：关键时间点可融入叙述（如"签约后""随后"），不罗列"×年×月×日..."的日期链条。详细时间由大事记模块处理
- 金额明细：提及核心金额即可（如"3.2亿元"），不逐笔列明付款、转款、利息的具体构成
- 证据名称：证据的整理和分析由证据策略模块负责

**原则**：读完事实段落，读者应知道"谁和谁做了什么交易，为什么打官司"，不需要知道"哪天做了什么事"

### 五、信息来源标注

根据材料类型区分表述可靠性：

| 材料类型 | 叙述方式 |
|----------|----------|
| 判决书、裁定书 | 采信的事实按客观事实叙述 |
| 当事人单方陈述（咨询阶段、起诉状等） | 涉及争议的关键事实前加"据XX陈述""XX主张"，区分陈述与被证事实 |
| 混合材料（部分有书证、部分凭口述） | 有书面证据的事实按客观事实写，仅凭口述的加来源标注 |

**示例**：
- 错误：公司要求圣某进行人脸识别并录制声明。（将单方陈述当作客观事实）
- 正确：据圣某陈述，入职约两个月后，公司要求其进行人脸识别并录制"我要成为个体工商户"的声明。

## 整理步骤

### 第一步：识别核心要素

- 当事人是谁
- 发生了什么交易/事件
- 主要义务是什么
- 争议从何而起

### 第二步：串联叙述

以时间逻辑串联核心要素，形成连贯叙述：

```
[当事人关系] → [交易性质] → [核心义务] → [关键行为] → [争议起因]
```

### 第三步：精简表述

- 删除无关细节
- 合并重复信息
- 使用简洁语言

### 第四步：概念锁定

- 保持原文表述
- 不擅自具体化抽象概念

## 与大事记模块的分工

| 模块 | 内容 | 格式 |
|------|------|------|
| **案件概要** | 简要叙述案件事实 | 一段话（200-400字） |
| **大事记模块** | 详细时间线梳理 | 表格（时间、事件、主要内容） |

案件概要提供"全景图"，大事记提供"时间轴"，两者配合使用。

## 特殊情况处理

### 多笔交易的情况

涉及多笔交易的，可分段简要叙述：

```markdown
## 案件事实

**交易背景**：[第一笔交易简要叙述]

**争议交易**：[引发争议的核心交易简要叙述]
```

### 事实争议的情况

当事实存在争议时，分别简要说明各方主张：

```markdown
## 案件事实

[无争议的事实简要叙述]

**争议事实**：原告主张...，被告抗辩...
```

## 检查清单

整理完成后检查：

- [ ] 是否以一段话简要叙述？
- [ ] 是否包含核心要素（当事人、交易性质、核心义务、关键行为、争议起因）？
- [ ] 篇幅是否控制在合理范围？
- [ ] 是否避免了冗长细节？
- [ ] 是否保持了原文概念锁定？
