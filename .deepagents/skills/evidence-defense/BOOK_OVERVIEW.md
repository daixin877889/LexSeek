## 1. 结构 (Structural)

### 类型
**实操手册 / 方法论书籍** — 系统论述刑事证据辩护的技术方法，兼具理论阐释与实战案例

### 一句话主旨
**本书系统阐述如何通过质疑控方证据的合法性、真实性、关联性来攻破证据锁链，实现有效刑事辩护。**

### 骨架 (主要论点及其关系)

1. **证据辩护导论**：确立证据辩护在刑辩中的核心地位，介绍十项核心证据规则
2. **物证、书证的审查**：实物证据的鉴真、合法性审查、保管链条、关联性、补正与排除
3. **证人证言、被害人陈述的审查**：证人资格、传闻证据、意见证据、真实性审查、出庭发问
4. **犯罪嫌疑人、被告人供述的审查**：合法性审查（讯问程序）、真实性审查、翻供处理、口供补强
5. **鉴定意见的质证**：鉴定主体资质、检材鉴真、鉴定程序、鉴定方法、常见鉴定类型的专项质证
6. **勘验、检查、辨认、侦查实验笔录的审查**：程序合法性、笔录内容完整性、辨认规则、侦查实验条件
7. **视听资料、电子数据的审查**：来源合法性、提取制作程序、真实性鉴定、完整性校验
8. **技术调查、侦查证据的审查**：适用范围与审批、移送举证程序、庭外核实
9. **非法证据排除**：非法取证认定、排除程序启动、调查程序、同步录音录像运用
10. **证据的综合审查与运用**：证明责任、证明标准、间接证据定案、印证规则、孤证不能定案、口供补强

**论点之间的关系**: **并列 + 递进** — 十章按证据种类并列展开，最后一章综合运用，形成"分论→总论"的递进结构

### 作者要解决的核心问题
**"如何在刑事辩护中系统运用证据规则，质疑控方每一项证据，攻破证据锁链，实现无罪或罪轻辩护？"**

## 2. 解释 (Interpretive)

### 关键术语 (作者本人的用法)

| 术语 | 作者的定义 | 和常识用法的差异 |
|---|---|---|
| 证据能力 | 证据能否在法庭出示的资格，取决于合法性 | 常人常混淆"证据能力"与"证明力"，作者强调两者分离 |
| 证明力 | 证据对案件事实的证明程度，取决于真实性、关联性 | 常人以为"能证明"就是有证明力，作者强调需综合审查 |
| 鉴真 | 验证实物证据同一性、初始性，解决证据来源问题 | 常人忽视证据来源审查，作者强调鉴真是证据能力的前提 |
| 印证 | 两个以上证据的事实信息相互验证 | 常人以为"能证明"就够了，作者强调必须有多证据印证 |
| 证明责任 | 举证义务及举证不能的败诉风险，控方承担 | 常人以为被告也要举证，作者强调无罪推定下控方独担 |
| 证明标准 | 证据证明案件事实需达到的程度，刑事为"排除合理怀疑" | 常人以为"可能性大"就行，作者强调刑事需达99%程度 |
| 孤证不能定案 | 单个证据不能独立认定有罪事实 | 常人以为"有证据就行"，作者强调必须有证据链 |
| 口供补强 | 口供必须有其他证据补强才能定案 | 常人以为"认罪就够"，作者强调口供不能单独定案 |
| 非法证据排除 | 以非法方法取得的证据不得作为定案依据 | 常人以为"证据是真的就行"，作者强调合法性是前提 |

### 核心命题 (用自己的话)

1. **证据辩护是刑事辩护的基础**：无论实体辩护还是程序辩护，都必须以证据辩护为依托
2. **证据审查要从"三性"入手但不止于三性**：合法性、真实性、关联性是起点，但需综合运用证据规则
3. **实物证据必须鉴真**：来源不明、保管链条断裂的物证书证不得作为定案依据
4. **言词证据必须印证**：孤证不能定案，口供必须有其他证据补强
5. **非法证据必须排除**：刑讯逼供等非法方法取得的证据，不得作为定案依据
6. **证明责任由控方承担**：被告人不负证明责任，控方必须排除合理怀疑地证明犯罪
7. **同步录音录像是最优证据**：当笔录与录音录像不一致时，以录音录像为准

### 论证链
作者通过"理论阐述 → 法律条文援引 → 案例演示"的三段式论证路径：
- 每章先阐述证据审查的理论框架和要点
- 再援引《刑事诉讼法》《最高法院刑诉解释》等法律依据
- 最后用大量真实案例（判决书摘录）演示质证技巧和法院采信/排除的结果

## 3. 批判 (Critical) ★

### 作者的时代局限
- 本书基于中国现行《刑事诉讼法》及司法解释，法律条文可能随立法修订而变化
- 部分案例引用的是2012年版司法解释，需注意2021年新版的变化
- 电子数据审查部分可能随技术发展需要更新（如区块链存证等新技术）

### 作者的立场盲点
- 作者为刑辩律师团队，立场天然偏向辩护方，对控方角度论述较少
- 强调"攻破证据锁链"，但对如何建构辩方证据体系论述相对较少
- 案例多为辩护成功案例，可能存在"幸存者偏差"

### 未被证明的假设
- 假设法官会严格依法审查证据（实践中法官可能更"务实"）
- 假设同步录音录像能真实反映讯问情况（实践中录像也可能被选择性录制）
- 假设证人会如实作证（实践中证人可能受多种因素影响）

### 最强反对意见
**如果检察官反驳本书**：证据规则的严格运用会增加控方负担、降低诉讼效率，某些"瑕疵证据"补正后仍可采信，过度强调程序可能放纵犯罪。
