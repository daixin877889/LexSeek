# 刑事证据审查技能包

## 概述

本技能包提供系统化的刑事证据审查方法论，帮助用户从合法性、真实性、关联性三个维度质证控方证据。

**核心原则**：先审查证据能力（合法性），再审查证明力（真实性、关联性）。

## 使用方法

### 1. 确定证据类型

首先判断需要审查的证据属于哪一类：
- 实物证据（物证、书证）→ 先鉴真，后质证
- 鉴定意见 → 六步质证框架
- 证人证言 → 四维度真实性审查
- 勘验笔录 → 六项审查要点
- 辨认笔录 → 六个排除情形

### 2. 选择对应框架

根据证据类型选择 `references/` 中的对应模块，按步骤执行审查。

### 3. 综合判断

单一证据审查完成后，综合运用证明规则判断能否定案。

## 详细内容

详见以下文件：
- `SKILL.md` - 核心原则与执行步骤
- `INDEX.md` - 技能关系图与学习路径
- `PROMPT.md` - 质证提示词模板
- `references/` - 各模块详细说明

## 注意事项

1. **先鉴真后质证**：实物证据必须先验证同一性
2. **合法性优先**：非法证据即使真实也应排除
3. **证明责任在控方**：辩方只需提供线索
4. **存疑有利于被告人**：无法排除合理怀疑时作出有利认定
