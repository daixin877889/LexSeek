# 刑事证据审查技能包

## 概述

本技能包提供系统化的刑事证据审查方法论，帮助用户从合法性、真实性、关联性三个维度质证控方证据，攻破证据锁链。

## 核心原则

证据审查遵循三性框架：
- **合法性**：证据能力门槛，决定能否在法庭出示
- **真实性**：证明力维度，决定证据内容是否可靠
- **关联性**：证明力维度，决定证据与案件事实的联系

先审查证据能力（合法性），再审查证明力（真实性、关联性）。

## 技能构成

本技能包含六大模块，详见 `references/` 目录：

| 模块 | 内容 | 适用场景 |
|---|---|---|
| 基础审查框架 | 三性审查、鉴真、鉴定质证、证人审查、勘验审查 | 审查各类证据的基础方法 |
| 程序审查专项 | 辨认审查、录音录像运用、非法证据排除 | 程序合法性专项审查 |
| 证明责任与标准 | 举证责任、证明标准、孤证识别、存疑有利 | 证明层面的判断规则 |
| 证据定案规则 | 间接证据、印证规则、一对一处理 | 综合判断能否定案 |
| 排除规则 | 各类强制性排除情形 | 判断证据应否排除 |
| 证据权重与优先 | 客观优于主观、录像优于笔录、鉴真前提 | 证据矛盾时的取舍规则 |

## 使用方法

### 1. 确定证据类型

首先判断需要审查的证据属于哪一类：
- 实物证据（物证、书证）→ 先鉴真，后质证
- 鉴定意见 → 六步质证框架
- 证人证言 → 四维度真实性审查
- 勘验笔录 → 六项审查要点
- 辨认笔录 → 六个排除情形

### 2. 选择对应框架

根据证据类型选择 `references/` 中的对应模块，按步骤执行审查。

### 3. 综合判断

单一证据审查完成后，综合运用证明规则判断能否定案：
- 是否孤证？
- 是否相互印证？
- 能否排除合理怀疑？

## 调用时机

当用户出现以下表述时激活本技能：

**典型触发信号**：
- "这份证据有问题吗"
- "怎么质证这个证据"
- "证据能不能用"
- "程序有问题吗"
- "能定案吗"

**不应激活**：
- 纯信息查询（"什么是证据"）
- 与证据审查无关的请求

## 学习路径

**新手入门**：
1. 基础审查框架 → 理解三性审查是所有证据审查的起点
2. 证明责任与标准 → 理解控方独担举证责任

**进阶实战**：
3. 程序审查专项 → 掌握攻破控方证据的核心武器
4. 排除规则 → 判断哪些证据必须排除

**高级综合**：
5. 证据定案规则 → 判断证据链能否支撑定案
6. 证据权重与优先 → 处理证据矛盾时的取舍

## 注意事项

1. **先鉴真后质证**：实物证据必须先验证同一性，未鉴真的证据不具备证据能力
2. **合法性优先**：非法证据即使真实也应排除，不能因"印证"而采信
3. **证明责任在控方**：辩方只需提供线索，控方证明合法性
4. **存疑有利于被告人**：证据无法排除合理怀疑时，作出有利于被告人的认定
