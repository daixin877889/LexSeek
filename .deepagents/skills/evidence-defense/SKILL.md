# 刑事证据审查

## 核心原则

### P1: 先审查证据能力，再审查证明力

证据能力（合法性）是门槛问题，决定证据能否在法庭出示。
证明力（真实性、关联性）是分量问题，决定证据能证明多少。

无证据能力的证据无需继续审查真实性、关联性。

### P2: 控方独担证明责任

被告人不负自证其罪的责任。
控方必须"排除合理怀疑"地证明犯罪事实。
非法证据排除程序中，辩方只需提供线索，控方证明合法性。

### P3: 孤证不能定案

单个证据不能单独作为认定有罪事实的根据。
虽有多份证据，但与主要事实有关联的只有一项，实质仍是孤证。
仅适用于对被告人不利的事实。

### P4: 存疑有利于被告人

证据无法排除合理怀疑时，作出有利于被告人的认定。
金额存疑时以二者重合部分认定。
事实存疑时认定该项事实不存在。

### P5: 非法证据应当排除

刑讯逼供、威胁、引诱、欺骗等非法方法取得的证据必须排除。
程序违法导致证据来源不明时强制排除，不可补正。

### P6: 客观证据优于主观证据

言词证据与物证矛盾时以物证为准。
笔录与录像不一致时以录像为准。

## 关键要点

### 证据审查要点

详见 `references/基础审查框架.md`

1. **三性审查**: 合法性 → 真实性 → 关联性
2. **鉴真两步**: 独特性辨认 → 保管链验证
3. **鉴定六步**: 主体 → 检材 → 程序 → 方法 → 内容 → 关联
4. **证人四维度**: 利害关系 → 证言矛盾 → 印证情况 → 常理判断
5. **勘验六项**: 主体 → 见证人 → 时间 → 内容 → 签名 → 附件

### 程序审查要点

详见 `references/程序审查专项.md`

1. **辨认六情形**: 非侦查人员主持、辨认前见对象、未个别进行、未混杂、有暗示、其他存疑
2. **录像运用**: 申请调取 → 详细观看 → 形成文字稿 → 发现差异 → 揭露非法
3. **排除流程**: 提出申请 → 说明理由 → 启动调查 → 调查程序 → 裁决结果

### 证明规则要点

详见 `references/证明责任与标准.md`

1. **举证责任**: 控方承担，被告人不负证明责任
2. **证明标准**: 全面性 → 合法性真实性 → 排他性（刑事约99%，死刑排除一切）
3. **孤证识别**: 形式判断 → 实质判断 → 关联判断
4. **存疑处理**: 事实存疑 → 金额存疑 → 数量存疑

### 定案规则要点

详见 `references/证据定案规则.md`

1. **间接证据五条件**: 查证属实 → 相互印证 → 证据链完整 → 排除怀疑 → 推理合逻辑
2. **印证判断**: 寻找重合点 → 检验相互验证（解决真实性不解决合法性）
3. **一对一处理**: 本质是孤证 → 概率仅50% → 存疑有利被告人

### 排除规则要点

详见 `references/排除规则.md`

1. **来源不明**: 强制排除，不可补正
2. **非法取证**: 刑讯逼供、威胁、引诱、欺骗、非法拘禁
3. **传闻证据**: "听说"、"传说"表述，不可采
4. **见证人不适格**: 辅警、聘用人员、侦查机关人员
5. **鉴定超范围**: 金额认定、折算等法律裁判事项
6. **口供无补强**: 只有被告人供述不能定罪

### 权重规则要点

详见 `references/证据权重与优先.md`

1. **客观优于主观**: 物证优于言词，鉴定优于供述
2. **录像优于笔录**: 实质性差异时以录像为准
3. **鉴真是前提**: 未鉴真无证据能力，不能进入质证

## 执行步骤

当技能被激活后，按以下步骤执行：

### 步骤1: 判断证据类型

确认证据属于物证、书证、鉴定意见、证人证言、勘验笔录、辨认笔录等哪一类。

### 步骤2: 选择审查框架

- 实物证据 → 先鉴真（`references/基础审查框架.md` 鉴真部分）
- 鉴定意见 → 六步质证（`references/基础审查框架.md` 鉴定部分）
- 证人证言 → 四维度审查（`references/基础审查框架.md` 证人部分）
- 程序证据 → 程序审查专项（`references/程序审查专项.md`）

### 步骤3: 执行审查

按选定的框架逐项审查，记录问题点。

### 步骤4: 判断排除必要性

若发现合法性问题，判断是否应排除（`references/排除规则.md`）。

### 步骤5: 综合判断能否定案

若单一证据通过审查，综合判断证据链能否支撑定案（`references/证据定案规则.md`）。

## 边界

### 不应在以下情况使用

- 纯信息查询（"什么是证据"）→ 直接回答概念定义
- 与证据审查完全无关的请求
- 民事案件（民事举证责任和证明标准不同）

### 常见失败模式

详见 `references/排除规则.md` 各排除情形对应的陷阱

1. **拘泥三性忽视规则**: 机械套用三性框架，不综合运用孤证、补强等规则
2. **混淆能力与证明力**: 质证时混淆"能不能用"和"能证明多少"
3. **忽视鉴真环节**: 直接质证内容，未先审查同一性和保管链
4. **以印证采信非法证据**: 印证解决真实性不解决合法性
5. **举证责任倒置**: 要求被告人证明刑讯逼供
