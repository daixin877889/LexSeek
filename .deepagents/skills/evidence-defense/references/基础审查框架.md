# 基础审查框架

## 一、证据"三性"审查

审查任何刑事证据时，依次检验三项属性。

### 1. 合法性（证据能力门槛）

**审查内容**：
- 取证程序是否合法
- 有无刑讯逼供、程序违法
- 见证人是否适格
- 批准手续是否齐全

**判断标准**：
- 合法性问题 → 证据能力受质疑
- 非法取证 → 应排除
- 程序瑕疵 → 判断可否补正

### 2. 真实性（证明力维度）

**审查内容**：
- 证据内容是否真实
- 有无伪造、篡改、污染
- 保管链条是否完整

**判断标准**：
- 来源清楚、保管完整 → 真实性可靠
- 来源不明、保管断裂 → 真实性存疑

### 3. 关联性（证明力维度）

**审查内容**：
- 证据与案件事实有无实质联系
- 能否证明待证事实

**判断标准**：
- 直接关联待证事实 → 证明力强
- 间接关联或无关联 → 证明力弱或无

### 执行步骤

1. **判断证据类型** → 确认属于哪一类证据
2. **审查合法性** → 若有问题建议排除
3. **审查真实性** → 判断内容是否可靠
4. **审查关联性** → 判断能否证明待证事实
5. **综合运用规则** → 结合孤证、补强、印证等规则

## 二、实物证据鉴真

实物证据（物证、书证）必须先鉴真才能进入质证程序。

### 鉴真解决什么

- **同一性**：这份物证就是当时提取的那份吗？
- **载体真实性**：载体是否真实、未被篡改？
- **形式关联性**：与案件有无形式联系？

### 鉴真两步法

#### 第一步：独特性辨认

通过外部特征识别证据同一性：
- 形状、颜色、签名、编号
- 现场照片、标签位置
- 特殊标记

**判断标准**：
- 有独特标记 → 可辨认
- 无标记或标记变化 → 同一性存疑

#### 第二步：保管链验证

追踪证据从提取到出示的全流程：
- 提取笔录 → 扣押清单 → 保管记录 → 送检记录 → 鉴定意见 → 出示

**审查要点**：
- 各环节记录是否完整
- 时间间隔是否合理
- 标签位置有无变化
- 性状有无变化

**判断标准**：
- 保管链完整 → 同一性可靠
- 保管链断裂 → 同一性无法保证

### 执行步骤

1. **审查独特性辨认标记** → 确认有无现场编号、签名、拍照等
2. **审查保管链条** → 追踪各环节记录
3. **对比关键信息一致性** → 对比各环节记载的特征
4. **判断证据能力** → 说明是否具备证据能力

## 三、鉴定意见质证

质证鉴定意见的系统框架，分六步审查。

### 六步质证框架

#### 1. 主体审查

**审查内容**：
- 鉴定机构是否在司法鉴定名录
- 鉴定人专业领域是否与鉴定事项匹配

**判断标准**：
- 无资质 → 鉴定意见无效
- 专业不对口（如昆虫学专家鉴定鸟类）→ 无效

#### 2. 检材审查

**审查内容**：
- 检材来源是否清楚
- 保管链是否完整
- 是否被污染

**判断标准**：
- 检材未鉴真 → 鉴定意见无证据能力
- 检材被污染 → 结果不可靠

#### 3. 程序审查

**审查内容**：
- 鉴定意见是否送达被告人
- 有无复核程序
- 是否依法告知

**判断标准**：
- 未送达 → 程序违法
- 无复核 → 程序瑕疵

#### 4. 方法审查

**审查内容**：
- 鉴定方法是否科学
- 抽样数量是否足够
- 是否符合专业规范

**判断标准**：
- 方法不科学 → 结果不可靠
- 抽样不足（如仅2%）→ 结论不可靠

#### 5. 内容审查

**审查内容**：
- 结论是否明确（不含"可能"、"推断"）
- 是否超出鉴定范围（如金额认定、折算）

**判断标准**：
- 模糊表述 → 不得作为定案根据
- 超范围鉴定 → 无效

#### 6. 关联审查

**审查内容**：
- 与案件事实有无关联
- 与其他证据是否矛盾

**判断标准**：
- 无关联 → 证明力不足
- 与其他证据矛盾 → 需分析原因

### 执行步骤

1. **审查主体资质**
2. **审查检材鉴真** → 若检材未鉴真建议排除
3. **审查鉴定程序**
4. **审查鉴定方法**
5. **审查鉴定内容** → 判断是否超范围
6. **综合判断** → 说明能否作为定案根据

## 四、证人证言真实性审查

审查证人证言真实性的四个维度。

### 四维度审查

#### 1. 利害关系

**审查内容**：
- 证人与当事人有无利害关系
- 是否是被害人亲属、被告人仇人等

**判断标准**：
- 有利害关系 → 需其他证据印证
- 无利害关系 → 相对客观

#### 2. 证言矛盾

**审查内容**：
- 证言前后是否一致
- 与其他证据是否矛盾

**判断标准**：
- 前后矛盾 → 真实性存疑
- 与其他证据矛盾 → 需分析原因

#### 3. 印证情况

**审查内容**：
- 能否与其他证据相互印证

**判断标准**：
- 有印证 → 更可信
- 无印证 → 需谨慎

#### 4. 常理判断

**审查内容**：
- 是否符合常识、常理

**判断标准**：
- 不符合常理 → 需质疑
- 如500米距离、夜晚能否"亲眼看到"

### 执行步骤

1. **审查利害关系**
2. **审查证言矛盾** → 若前后矛盾真实性存疑
3. **审查印证情况**
4. **审查常理判断**
5. **建议证人出庭** → 若为关键证人

## 五、勘验检查笔录审查

审查勘验检查笔录的系统要点。

### 六项审查要点

#### 1. 主体审查

**审查内容**：
- 是否侦查人员（非技术人员）
- 是否持证（《刑事案件现场勘查证》）
- 是否≥二人

**判断标准**：
- 非侦查人员主持 → 无效
- 无证勘验 → 无效
- 一人勘验 → 无效

#### 2. 见证人审查

**审查内容**：
- 是否与案件无关
- 是否适格（非辅警、聘用人员）

**判断标准**：
- 辅警、聘用人员 → 不适格

#### 3. 时间审查

**审查内容**：
- 是否及时进行

**判断标准**：
- 延迟勘验 → 可能现场被破坏

#### 4. 内容审查

**审查内容**：
- 是否现场制作
- 是否全面记载

**判断标准**：
- 事后补写 → 真实性存疑

#### 5. 签名审查

**审查内容**：
- 相关人员是否签名
- 签名人员是否实际到场

**判断标准**：
- 签名人员未到场 → 无效

#### 6. 附件审查

**审查内容**：
- 现场图、照片、录像是否与笔录记载一致

**判断标准**：
- 笔录与附件不一致 → 真实性存疑

### 执行步骤

1. **审查勘验主体**
2. **审查见证人**
3. **审查时间**
4. **审查内容**
5. **审查签名**
6. **审查附件**
7. **给出结论**
