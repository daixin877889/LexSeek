# 证明责任与标准

## 一、举证责任分配

刑事诉讼证明责任分配的核心原则。

### 核心原则

控方独担证明责任，被告人无自证其罪的义务。

### 分配规则

#### 基本规则

**控方承担**：
- 证明被告人有罪的责任
- 无罪推定的直接体现
- 必须"证据确实、充分"地证明犯罪

#### 被告人不负证明责任

**被告人权利**：
- 被告人否认犯罪是辩解，不是举证
- 无需证明自己无罪、不在场、无作案动机
- 无自证其罪的义务

#### 例外情形

**法律规定被告人承担**：
- 如巨额财产来源不明罪（被告人需说明来源）

#### 排除非法证据程序

**举证责任**：
- 辩方只需提供线索
- 检察院承担证明取证合法的责任

### 执行步骤

1. **判断证明事项** → 确认需要证明什么
2. **判断责任归属** → 控方还是被告人
3. **判断举证情况** → 举证方是否完成举证
4. **给出结论** → 举证不能的后果

## 二、证明标准判断

刑事证明标准"证据确实、充分"的三要件。

### 三要件

#### 1. 全面性

**要求**：
- 定罪量刑事实都有证据证明
- 每个待证事实都需要证据支撑
- 不能有事实无证据

#### 2. 合法性和真实性

**要求**：
- 定案证据均经法定程序查证属实
- 证据必须经过质证程序
- 查证属实才能作为定案依据

#### 3. 排他性

**要求**：
- 综合全案排除合理怀疑
- 结论具有唯一性
- 无矛盾或矛盾已排除

### 证明标准对比

| 类型 | 标准 |
|---|---|
|刑事 | 约99%（排除合理怀疑）|
|死刑 | 排除**一切**合理怀疑 |
|民事 | 约51%（优势证据）|

### 执行步骤

1. **判断全面性** → 每个事实有无证据
2. **判断合法性和真实性** → 证据是否查证属实
3. **判断排他性** → 能否排除合理怀疑
4. **判断案件类型** → 是否死刑案件
5. **给出结论**

## 三、孤证识别

判断是否为孤证的方法。

### 核心原则

孤证不能定案：单个证据不能单独作为认定案件事实的根据。

### 三步判断

#### 1. 形式判断

**方法**：
- 只有一个证据形式上呈现

**常见情形**：
- 只有证人证言
- 只有被害人陈述
- 只有被告人供述

#### 2. 实质判断

**方法**：
- 虽有多份证据，但与主要事实有关联的只有一项

**常见情形**：
- 三份证人证言都转述同一人说法（同出一源）
- 多份证据都源自同一来源

#### 3. 关联判断

**方法**：
- 其他证据与主要事实无实质关联

**常见情形**：
- 有物证照片但照片与作案无关
- 只有证人证言关联作案事实

### 适用范围

- 仅适用于**对被告人不利的事实**
- 不适用于有利事实

### 执行步骤

1. **列出与主要事实有关的证据**
2. **判断证据数量**
3. **判断证据来源** → 是否同出一源
4. **给出结论**

## 四、存疑有利于被告人

存疑有利于被告人原则的具体运用。

### 核心原则

当证据无法排除合理怀疑时，应作出有利于被告人的认定。

### 适用情形

#### 1. 事实存疑

**处理方法**：
- 直接认定该项事实不存在

**示例**：
- "一对一"证据 → 认定指控事实不存在

#### 2. 金额存疑

**处理方法**：
- 以二者重合部分认定

**示例**：
- 指控10000元存疑，被告人承认5000元 → 认定5000元

#### 3. 数量存疑

**处理方法**：
- 以能证明的最低数量认定

**示例**：
- 违禁品数量存疑 → 以最低数量认定

### 执行步骤

1. **判断存疑类型** → 事实、金额还是数量
2. **判断指控证据** → 能否排除合理怀疑
3. **判断辩方主张**
4. **给出认定结论**
