# 反射式迭代检索

法律检索采用三周期递进模式，逐层聚焦。

## 检索原理

**核心思想**：从宽泛到精准，从案情到法条，逐周期反射聚焦。

```
周期一：宽泛扫描 → 识别核心框架
    │
    └──→ 周期二：精准定位 → 提炼专业术语
              │
              └──→ 周期三：精确匹配 → 关联司法解释
```

## 周期一：案情解构与宽泛检索

**目标**：识别核心主体、争议标的，建立初步框架

**操作要点**：
- 提取当事人关系
- 识别争议核心
- 确定法律领域
- 宽泛检索相关法律

**检索范围**：
- 民法典相关编章
- 基础法律概念
- 一般性规定

**输出**：
- 初步法律框架
- 核心主体关系图
- 争议焦点清单

## 周期二：反思与二次扫描

**目标**：基于周期一结果，提炼专业关键词进行精准检索

**操作要点**：
- 分析周期一检索结果的相关性
- 提炼专业法律术语
- 调整检索策略
- 二次扫描

**检索范围**：
- 具体法条条款
- 专业法律术语
- 相关裁判观点

**输出**：
- 精准法条定位
- 专业术语对照
- 相关案例索引

## 周期三：精确匹配与关联

**目标**：精准定位具体法条，关联司法解释

**操作要点**：
- 锁定具体条文
- 检索配套司法解释
- 检索指导案例
- 验证时效性

**检索范围**：
- 具体法条条文
- 相关司法解释
- 指导案例/公报案例
- 最新修正案

**输出**：
- 精确法条引述
- 司法解释配套
- 类案裁判观点
- 时效性说明

## 与攻防推演的关系

反射式迭代检索为攻防策略推演提供法律支撑：

```
攻防策略推演
    │
    ├──→ 被告抗辩策略
    │         │
    │         └──→ 需要法律支撑 → 反射式检索
    │
    └──→ 原告反制预判
              │
              └──→ 需要法律支撑 → 反射式检索
```

## 执行步骤

1. **周期一启动**：基于案情进行宽泛检索
2. **反思调整**：分析检索结果，提炼专业关键词
3. **周期二执行**：精准定位法条
4. **关联深化**：检索司法解释、类案
5. **时效检验**：确认法律时效性
6. **输出清单**：形成完整法律依据清单

## 注意事项

1. **避免跳跃**：不跳过周期一直接进入精准检索
2. **保持开放**：周期一保持开放心态，不过早限定
3. **及时调整**：根据检索结果调整策略
4. **时效检验**：始终关注法律的时效性
5. **交叉验证**：法条、解释、案例相互印证
