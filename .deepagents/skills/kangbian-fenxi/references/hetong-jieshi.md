# 合同解释优先原则

合同解释优先于法律评价，是抗辩分析的基本方法。

## 核心原则

**先探究当事人意思表示，再进行法律评价。**

法律评价以意思表示解释为前提：
- 不理解当事人的真实意思，就无法正确适用法律
- 法律评价是"戴着手套的操作"，需先脱下手套

## 解释的对象

### 条款含义解释

针对合同具体条款：
- 文义解释：条款字面含义
- 体系解释：条款在合同整体中的位置
- 目的解释：条款的目的功能

### 交易目的解释

针对合同整体目的：
- 当事人签订合同的真实目的
- 交易的商业背景
- 行业惯例与习惯

### 补充解释

针对合同漏洞：
- 补充当事人未约定事项
- 依据诚实信用原则
- 参照交易习惯

## 解释的方法

### 文义解释优先

从条款字面含义出发：
- 一般含义优先
- 专业含义需证据支持
- 避免脱离文义的扩张解释

### 体系解释补充

将条款置于合同整体：
- 条款间的逻辑关系
- 前后条款的一致性
- 避免矛盾解释

### 目的解释校正

以合同目的校正解释：
- 解释应符合合同目的
- 目的解释可校正文义解释
- 但不得突破文义边界

### 习惯解释补充

参照交易习惯：
- 行业通行做法
- 当事人之间的交易习惯
- 地区惯例

## 解释在抗辩中的应用

### 事实风险点挖掘

通过合同解释发现风险：

| 解释发现 | 风险类型 |
|----------|----------|
| 条款含义模糊 | 解释争议风险 |
| 条款相互矛盾 | 合同解释风险 |
| 重要事项未约定 | 合同漏洞风险 |
| 文义与目的不符 | 意思表示风险 |

### 抗辩策略构建

解释结论作为抗辩基础：
- 对方理解的条款含义不合理
- 合同目的不支持对方主张
- 交易习惯得出不同结论

## 解释的边界

### 不得突破文义

解释不得脱离条款文义：
- 文义是解释的边界
- 超出文义的解释需特别理由
- 法官不能替当事人订立合同

### 尊重当事人自治

解释应尊重意思自治：
- 当事人明确约定的从约定
- 真实意思优先于书面表达
- 但需有证据支撑

### 诚实信用原则

解释应符合理性人标准：
- 合理解释优于不合理
- 商业诚信标准
- 公平原则

## 与合同效力检视的关系

合同解释是效力检视的前置步骤：

```
合同解释优先
    │
    ├──→ 成立检视：解释是否存在合意
    │
    ├──→ 有效检视：解释意思表示是否真实
    │
    └──→ 生效检视：解释条件条款的含义
```

## 执行步骤

1. **识别争议条款**：确定需要解释的条款
2. **文义解释**：确定字面含义
3. **体系解释**：置于合同整体理解
4. **目的解释**：以合同目的校正
5. **习惯补充**：参照交易习惯
6. **形成解释结论**：确定当事人真实意思
7. **进入法律评价**：基于解释结论进行法律判断

## 注意事项

1. **解释是事实问题**：解释当事人意思表示是事实判断，非法律评价
2. **解释需证据支撑**：对意思表示的主张需有证据
3. **避免价值判断渗透**：解释阶段不进行价值判断
4. **解释优先于评价**：不理解意思表示就无法正确适用法律
