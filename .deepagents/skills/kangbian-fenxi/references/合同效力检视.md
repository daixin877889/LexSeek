# 合同效力体系检视

合同效力检视采用三阶段递进结构，每阶段判断性质不同。

## 检视结构

```
成立（事实判断）
    │
    └──→ 有效（价值判断）
              │
              └──→ 生效（履行效力限制）
```

## 核心区分：三种判断的性质差异

| 概念 | 解决的问题 | 判断性质 | 拘束力类型 |
|------|------------|----------|------------|
| **成立** | 有没有 | 事实判断 | 形式拘束力（不得擅自变更解除） |
| **有效** | 好不好 | 价值判断 | 实质拘束力（请求履行、保有给付） |
| **生效** | 履行效力是否齐备 | 效力限制 | 履行效力（可被条件限制） |

**类比理解**：
- "是女性" → 事实判断（成立），确认有没有
- "是漂亮的女性" → 价值判断（有效），确认好不好
- "可以请求履行" → 履行效力（生效），确认能否行使

三者叠加构成完整效力体系：成立→有效→生效→未消灭→无抗辩→请求权成立。

## 第一阶段：成立检视

**性质**：事实判断

**核心功能**：确认合同是否存在

**检视内容**：
- 主体是否存在
- 意思表示是否作出
- 合意是否达成

**成立后果**：形式拘束力
- 当事人不得擅自变更解除
- 即使合同未生效，当事人也不能"反悔"
- 例：十几个亿的合同未生效，当事人也不能说"我不跟你玩了"

**举证规则**：谁主张合同存在谁举证

**判停条件**：合同不成立 → 无需检视后续阶段

## 第二阶段：有效检视

**性质**：价值判断

**核心原则**：已成立的合同推定为有效，质疑者负担举证之责

**核心功能**：确认合同是否值得法律保护

**检视内容**：
- 行为能力是否具备
- 意思表示是否真实
- 是否违反法律强制性规定
- 是否违背公序良俗

**有效后果**：实质拘束力
- 当事人可请求履行
- 已履行给付可保有
- 违反构成违约责任

**效力瑕疵类型**：

| 瑕疵类型 | 法律效果 | 举例 |
|----------|----------|------|
| 无效 | 自始不生效力 | 违反强制性规定、恶意串通 |
| 可撤销 | 撤销后自始无效 | 欺诈、胁迫、重大误解 |
| 效力待定 | 追认后有效 | 限制行为能力、无权代理 |

**举证规则**：质疑合同有效者承担举证责任

**特别注意**：尽量减少无效合同是立法趋势
- 限制效力规范级别（必须是法律行政法规的强制性规定）
- 管理性强制性规定不影响合同效力
- 警惕公序良俗滥用成为无效合同的"逃逸路径"

## 第三阶段：生效检视

**性质**：履行效力限制

**核心区别**：有效≠生效

合同有效是法律评价结论，生效是履行效力的开启。

**核心功能**：确认能否请求履行

| 限制类型 | 检视要点 | 举例 |
|----------|----------|------|
| 附条件 | 条件是否成就 | 附生效条件合同 |
| 附期限 | 期限是否届至 | 附始期合同 |
| 需批准 | 是否获得批准 | 外商投资合同、保险公司股权转让 |

**生效后果**：履行效力发生
- 条件成就前：附条件义务不得请求履行
- 条件成就后：附条件义务可以请求履行
- 未附条件义务：因成立有效而生效，违反构成违约

**特别提示**：条件限制的是"特定义务的履行效力"，非"整个合同"

详见 [附条件分析.md](./附条件分析.md)

**批准生效合同特殊规则**：部分条款可独立生效
详见 [批准生效合同.md](./批准生效合同.md)

## 与请求权基础的关系

合同效力检视是请求权基础检视的前置环节：

| 请求权检视阶段 | 合同效力对应 |
|----------------|--------------|
| 步骤1：积极要件 | 合同成立要件 |
| 步骤2：未发生抗辩 | 合同有效抗辩 |
| 步骤3：消灭抗辩 | 合同撤销/解除 |

## 执行步骤

1. **成立检视**：判断是否存在合意
   - 未成立 → 终止，不存在效力问题
2. **有效推定**：推定已成立合同有效
3. **效力质疑**：检视质疑方提出的瑕疵事由
4. **生效检视**：检验条件/期限/批准限制
5. **结论**：明确合同处于何种效力状态

## 常见误区

1. **混淆有效与生效**：有效是静态评价，生效是动态履行
2. **忽视成立阶段**：直接讨论效力，跳过事实判断
3. **笼统判断附条件合同**：未精细化分析条件影响的义务范围
4. **举证责任倒置**：让合同有效的主张者承担举证责任
