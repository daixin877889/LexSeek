# 攻防策略推演

抗辩策略推演采用"三分法分类→被告-原告-普遍性"结构。

## 核心框架：三分法抗辩分类

抗辩策略必须按请求权检视阶段分为三类，逐类检视，不可遗漏：

### 第一类：权利妨碍抗辩（请求权根本没有产生）

判断标准：一项抗辩是否通过使请求权自始不成立。

| 子类型 | 说明 | 举例 |
|--------|------|------|
| 合同未成立 | 缺乏要约/承诺、意思表示不一致 | 双方未就价格达成一致 |
| 行为能力欠缺 | 缔约时无行为能力或限制行为能力 | 未成年人缔约、精神障碍者缔约 |
| 意思表示瑕疵 | 欺诈、胁迫、重大误解、通谋虚伪 | 受欺诈签订的合同 |
| 违反强制性规定 | 合同内容违法或违背公序良俗 | 标的物为禁止流通物 |
| 无权代理 | 代理人无代理权且本人不追认 | 超越授权范围的代理行为 |

### 第二类：权利消灭抗辩（请求权已产生但已消灭）

判断标准：是否存在使已产生的请求权归于消灭的事由。

| 子类型 | 说明 | 举例 |
|--------|------|------|
| 清偿/履行 | 债务已按约定履行完毕 | 货款已付清 |
| 免除 | 债权人免除债务 | 书面免除函 |
| 抵销 | 双方互负债务相互抵销 | 互负到期金钱债务 |
| 提存 | 因债权人原因将标的物提存 | 债权人拒不受领 |
| 混同 | 债权债务归于同一人 | 债务人继承债权人遗产 |
| 合同解除 | 已依法或依约解除合同 | 行使约定解除权 |
| 给付不能 | 履行已客观不能 | 特定物灭失 |

### 第三类：权利限制抗辩（请求权存在但暂不能行使）

判断标准：是否存在暂时阻却请求权行使的事由。

| 子类型 | 说明 | 举例 |
|--------|------|------|
| 诉讼时效抗辩 | 请求权已过诉讼时效 | 3年普通时效届满 |
| 同时履行抗辩权 | 对方未履行时己方可拒绝履行 | 未收到货拒付尾款 |
| 先履行抗辩权 | 应先履行方未履行 | 供货方未交货，付款方可拒付 |
| 不安抗辩权 | 对方经营恶化可中止履行 | 对方已进入破产程序 |
| 留置权抗辩 | 合法占有对方财产可留置 | 修理方留置修理物 |
| 违约金过高 | 主张调减违约金 | 违约金超过损失30% |

## 推演结构

```
权利妨碍抗辩（请求权根本没有产生）
    │
    ├── 被告抗辩 → 原告反驳 → 同类情形检验
    │
权利消灭抗辩（请求权已产生但已消灭）
    │
    ├── 被告抗辩 → 原告反驳 → 同类情形检验
    │
权利限制抗辩（请求权存在但暂不能行使）
    │
    ├── 被告抗辩 → 原告反驳 → 同类情形检验
```

## 第一视角：被告抗辩策略

**核心公式**：事实风险 + 法律依据 = 抗辩理由

### 事实风险点挖掘

从案件材料中识别对原告不利的事实：
- 前后矛盾之处
- 证据缺失环节
- 程序瑕疵问题
- 时效性问题

### 抗辩构建要素

| 要素 | 内容 | 来源 |
|------|------|------|
| 事实锚点 | 具体案件事实 | 案件材料 |
| 法律支撑 | 支撑抗辩的法条 | 反射式检索 |
| 论证逻辑 | 连接事实与法律 | 法律推理 |
| 效力评级 | 强效力/中等效力/弱效力 | 综合评估 |

### 效力评级标准

| 效力 | 定义 | 典型情形 |
|------|------|----------|
| 强效力 | 事实基础坚实、法律依据明确、论证逻辑严密 | 对方自认的事实+直接的对抗法条 |
| 中等效力 | 有一定依据但存在薄弱环节 | 事实有争议或法条适用需解释 |
| 弱效力 | 理论可行但依据不足 | 证据缺失或法条适用存在争议 |

## 第二视角：原告反制预判

**目标**：预判原告如何应对被告抗辩，逐类逐项进行。

### 反制策略类型

| 类型 | 内容 | 适用场景 |
|------|------|----------|
| 事实补救 | 解释不利事实、补充证据 | 事实争议 |
| 法律反击 | 引用对抗法条 | 法律争议 |
| 策略调整 | 变更诉讼请求、调整主张 | 策略层面 |

### 反制分析框架

针对被告的每项抗辩：
1. 分析抗辩的法律依据
2. 识别可能的反制法条
3. 评估事实补救空间
4. 判断反驳效力（强效力/中等效力/弱效力）

### 风险提示

为原告提供败诉风险评估：
- 高风险：抗辩可能成立，需重点应对
- 中风险：抗辩有一定依据，需准备应对
- 低风险：抗辩依据不足，但仍需注意

## 第三视角：同类情形检验

**核心问题**：若本案这样处理，同类案件是否也应如此处理？

### 验证方法

1. **同类案件推演**
   - 假设存在相同事实的案件
   - 适用相同策略
   - 检验结论是否合理

2. **制度目的检验**
   - 检验策略是否符合制度设计目的
   - 是否存在规避制度的嫌疑

3. **反例测试**
   - 构造极端案例
   - 检验是否存在不合理推论

### 验证结论

| 结论 | 说明 | 策略调整 |
|------|------|----------|
| 通过 | 策略具有普适性 | 维持策略 |
| 部分通过 | 策略需限定条件 | 限定适用范围 |
| 未通过 | 策略仅适用于本案特殊情形 | 重新构建策略 |

## 与其他方法的关系

攻防策略推演与其他方法的关系：

```
合同解释优先 → 挖掘事实风险点
       │
       └──→ 合同效力检视 → 确定抗辩类型归属
                 │
                 └──→ 反射式检索 → 提供法律支撑
                           │
                           └──→ 三分法分类推演
                                 │
                                 ├── 权利妨碍抗辩
                                 ├── 权利消灭抗辩
                                 └── 权利限制抗辩
                                       │
                                       └──→ 同类情形检验
```

## 执行步骤

1. **事实风险挖掘**：运用合同解释优先原则审查案件材料
2. **抗辩类型归类**：将每个抗辩点归入权利妨碍/权利消灭/权利限制三类之一
3. **逐类推演**：按三类顺序逐一展开被告抗辩
4. **法律支撑定位**：运用反射式检索定位法条
5. **反制策略预判**：原告视角逐项预判反驳
6. **同类情形检验**：每项抗辩完成普适性验证
7. **效力评估**：为每项抗辩和每项反驳标注效力等级
8. **汇总输出**：形成抗辩效力汇总表

## 注意事项

1. **不预设立场**：客观分析双方优势劣势
2. **全面覆盖**：三类抗辩逐类检视，不遗漏可能的抗辩角度
3. **空类标注**：如某类抗辩不存在，标注"经审查，本案不存在权利XX抗辩事由"
4. **逻辑闭环**：每项策略需有事实锚点和法律支撑
5. **诚实评估**：如实评估风险等级，不夸大不隐瞒
6. **可操作性**：策略需具体可行，有明确行动指引
