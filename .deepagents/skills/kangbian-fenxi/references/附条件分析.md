# 附条件合同精细分析

条件限制的是"特定义务的履行效力"，而非"整个合同"。

## 核心原则

**条件是履行效力的限制，不是合同效力的限制。**

附条件合同的正确理解：
- 合同已经成立并有效
- 条件成就与否仅影响特定义务的履行
- 条件不成就≠合同无效

## 分析框架

### 步骤一：识别合同的全部义务

列出合同项下的全部义务：
- 主给付义务
- 从给付义务
- 附随义务
- 不作为义务

### 步骤二：识别哪个义务附了条件

判断标准：
- 条款明示：条款直接规定某义务以某条件为前提
- 推定判断：根据交易性质、合同目的推定
- 区分判断：不同义务可能附不同条件

### 步骤三：分析条件对各义务的影响

逐项分析：

| 义务 | 是否附条件 | 条件内容 | 条件成就影响 |
|------|------------|----------|--------------|
| 义务A | 是/否 | 如附条件 | 履行/不履行 |
| 义务B | 是/否 | ... | ... |

## 常见误区与纠正

### 误区一：条件成就合同才生效

**错误理解**：条件成就前合同未生效，当事人无需履行任何义务

**正确理解**：
- 合同已成立有效
- 条件仅限制特定义务的履行
- 条件成就前，未附条件的义务仍需履行

### 误区二：条件不成就合同无效

**错误理解**：条件不成就导致合同无效

**正确理解**：
- 条件不成就仅是特定义务履行条件未满足
- 合同有效存在
- 当事人可能承担其他义务（如缔约过失责任）

### 误区三：整个合同都附条件

**错误理解**：以某一条件成就作为整个合同生效的前提

**正确理解**：
- 需精细化区分各义务
- 不同义务可能适用不同条件
- 同一合同可能存在多个条件

## 与合同效力检视的关系

附条件分析是合同效力检视第三阶段（生效检视）的深化：

```
合同效力检视
    │
    └──→ 生效检视
              │
              └──→ 附条件合同分析
                        │
                        └──→ 义务-条件匹配分析
```

## 执行步骤

1. 确认合同已成立有效（已完成效力检视前两阶段）
2. 识别合同项下全部义务
3. 逐项判断义务是否附条件
4. 分析条件对各项义务的独立影响
5. 结论：明确哪些义务因条件未成就而暂不履行
