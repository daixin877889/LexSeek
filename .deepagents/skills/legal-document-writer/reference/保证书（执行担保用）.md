# 保证书（执行担保用）

## 文书概述

保证书（执行担保用）是在执行程序中，被执行人申请暂缓执行时，担保人（被执行人本人或案外人）向人民法院提交的书面保证文件。保证人承诺在被执行人于暂缓执行期限届满后仍不履行义务时，由法院直接执行保证人的财产。执行担保是执行程序中暂缓执行的法定条件之一。

**适用场景**：执行程序中，被执行人申请暂缓执行，由担保人提供担保。

---

## 角色设定

撰写此文书的 AI 角色为**执行案件代理人/担保人律师**。文风需严谨、明确、无歧义。执行担保保证书涉及法院直接执行保证人财产的严重后果，措辞必须精确，不可有丝毫模糊。

---

## 结构规范

保证书（执行担保用）按以下顺序排列：

1. 标题
2. 致送法院名称
3. 保证正文（案由 + 暂缓执行申请理由 + 保证承诺）
4. 署名 + 日期

**注意**：保证书一般采用直接致送法院的书信格式，结构较其他文书更为简洁。

---

## 各板块写作指南

### 保证正文

保证正文必须包含以下要素，缺一不可：

1. **案件识别信息** — 执行案号、当事人姓名/名称和案由
2. **暂缓执行的原因** — 简要说明被执行人申请暂缓执行的理由（如正在筹集资金、正在与申请执行人协商和解、需要时间变卖财产等）
3. **申请暂缓执行的期限** — 写明申请暂缓执行的具体期限
4. **保证承诺** — 核心条款，必须有明确的承诺：「如被执行人XXX在你院决定暂缓执行的期限届满后仍不履行义务，你院可以直接执行本人/本单位的财产」
- 致送：`{{受理法院名称}}`

### 保证人责任

- 执行担保中的保证人承担的是独立的执行担保责任
- 法院在暂缓执行期限届满后，被执行人不履行义务的，可直接裁定执行保证人的财产，无需另行诉讼
- 保证人应充分了解并自愿承担此责任
- 保证书一经提交法院，如法院准许暂缓执行，保证即告成立

### 署名

- 保证人签名或盖章（法人须加盖公章 + 法定代表人签名）
- 日期为当前日期：`{{落款年月日}}`

```
执行担保保证书

请暂缓执行的期限）。本人/本单位自愿提供保证。如被执行人×××在 
你院决定暂缓执行的期限届满后仍不履行义务，你院可以直接执行本 
人/本单位的财产。 
                                      保证人: 
{{落款年月日}}
```

## 质量自检清单

1. 标题是否以 `# 保证书` 开头？
2. 是否写明了执行案号和案由？
3. 是否写明了被执行人申请暂缓执行的原因和期限？
4. 是否包含了「你院可以直接执行本人/本单位的财产」的核心承诺？
5. 保证人与被执行人的关系是否明确（如系被执行人本人提供担保则无需说明，如系案外人则建议简要说明）？
6. 全文是否简洁、意思表示明确、无歧义？
