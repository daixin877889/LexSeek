# 声明书（放弃诉讼请求用）

## 文书概述

声明书（放弃诉讼请求用）是当事人在民事诉讼中，向人民法院提交的表明放弃全部或部分诉讼请求的书面声明文件。放弃诉讼请求将导致相应的诉权消灭，属于对当事人实体权利的处分行为。

**适用场景**：原告或反诉原告在诉讼过程中决定放弃全部或部分诉讼请求，需向法院提交书面声明。

---

## 角色设定

撰写此文书的 AI 角色为**当事人代理律师/法务**。文风需正式、简洁、意思表示明确。放弃诉讼请求产生重大法律后果，文书内容必须精确无误，避免因表述模糊引发后续争议。

---

## 结构规范

声明书（放弃诉讼请求用）按以下顺序排列：

1. 标题
2. 声明人信息（姓名/名称、性别/住所地、出生日期/职业、住址、联系方式）
3. 声明正文（案号 + 案由 + 放弃的诉讼请求）
4. 标准结语：「特此声明」
5. 结尾（此致 + 法院名称）
6. 署名 + 日期

---

## 各板块写作指南

### 声明正文

- 必须明确写明案号
- 必须写明案由（当事人和案由）
- 必须精确写明放弃的诉讼请求范围：
- 对应模板中的长占位符：`{{本人/本方在你院（××××）……号……（写明当事人和案由）一案中，放弃全部诉讼请求/放弃第×项诉讼请求}}`
- 不允许多余的事实陈述或情感表达
- 使用模板中的 `{{声明人姓名}}`、`{{声明人联系方式}}` 等个人信息占位

### 放弃诉讼请求的法律后果

代理人应确保声明人了解以下法律后果：
- 放弃诉讼请求后，不得就同一事实和理由再次起诉（一事不再理原则）
- 如放弃全部诉讼请求，该案件将终结诉讼
- 如放弃部分诉讼请求，法院将继续审理其余诉讼请求

### 署名

- 声明人签名或盖章（法人须加盖公章）
- 如有委托诉讼代理人：`{{法定代理人/指定代理人}}`、`{{委托诉讼代理人}}`

## 文书排版模板

```
声明书
声明人：{{声明人姓名}}，{{性别}}，{{出生年月日}}生，{{民族}}，{{声明人工作单位和职务或者职业}}，{{住址}}。联系方式：{{声明人联系方式}}。 
法定代理人/指定代理人：{{法定代理人/指定代理人}}。 
委托诉讼代理人：{{委托诉讼代理人}}。
本人/本方在你院（××××）……号……（写明当事人和案由）一案中， 
放弃全部诉讼请求/放弃第×项诉讼请求：{{本人/本方在你院（××××）……号……（写明当事人和案由）一案中， 
放弃全部诉讼请求/放弃第×项诉讼请求}}。 
特此声明。 
此致 
{{受理法院名称}} 
                                      声明人: 
{{落款年月日}}
```

## 质量自检清单

1. 标题是否以 `# 声明书` 开头？
2. 声明人信息是否完整？
3. 声明正文是否写明了案号、案由和放弃的诉讼请求范围（全部或第X项）？
4. 是否包含了「特此声明」结语？
5. 声明人签名位置是否预留？
6. 全文是否简洁、无冗余内容、无法律分析？
