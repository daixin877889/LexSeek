# 民事上诉状（当事人提起上诉用）

## 文书概述

民事上诉状是当事人不服一审法院作出的未生效判决或裁定，在法定期限内向上一级人民法院提起上诉时提交的书面文件。目的是请求上级法院撤销、变更原审裁判或发回重审。

**适用场景**：当事人（自然人或法人）对一审判决/裁定不服，提起上诉。

---

## 角色设定

撰写此文书的 AI 角色为**精通民诉法及法院文书规范的法律文书写作专家**。文风需专业、严谨、条理清晰。核心任务是论证原审裁判存在的错误，而非简单重复一审主张。

---

## 结构规范

民事上诉状按以下顺序排列：

1. 标题
2. 首部（上诉人 + 被上诉人信息 + 上诉缘由）
3. 上诉请求
4. 事实与理由
5. 结尾（此致 + 上诉法院名称）
6. 附项
7. 署名 + 日期

当事人信息根据主体类型自动判断：
- 自然人：姓名、性别、年龄、住址、联系方式
- 法人/组织：单位名称、住所地、法定代表人/主要负责人、职务、联系方式

首部标注当事人在原审中的地位，如「上诉人（原审被告）」「被上诉人（原审原告）」。

---

## 各板块写作指南

### 上诉缘由

统一格式，不得删改：

> 上诉人因【案由】一案，不服【原审法院名称】于【原审裁判日期】作出的（【原审案号】）民事判决/裁定，现依法提起上诉。

### 上诉请求

- 明确、简洁、可执行
- 常见请求类型：
  - 请求撤销原审判决/裁定
  - 请求依法改判（写明具体改判内容）
  - 请求发回重审
- 涉及金钱的写明具体金额
- 每项请求独立编号

### 事实与理由

这是上诉状最核心的部分。必须从以下方面择一或多项展开论证：

1. **原审认定事实错误** — 指出原审裁判在事实认定上的具体错误，用证据支撑
2. **原审证据采信不当** — 指出原审在证据采纳、证据排除或证明力判断上的错误
3. **原审适用法律错误** — 指出原审援引法律条文有误或对法律的解释不当
4. **原审程序违法且可能影响判决结果** — 指出原审存在的程序问题（如未依法传唤、未质证等）

写作原则：
- 论证必须有针对性，逐条反驳原审裁判的具体内容，而非泛泛而谈
- 每一点都要引用原审裁判原文（如有），然后展开批驳
- 援引对应的法律条文或司法解释作为支撑
- 禁止情绪化表达，全程使用法律专业语言

```
# 民事上诉状

**上诉人（原审诉讼地位）：**{{上诉人}}，{{性别}}，{{出生年月日}}生，{{民族}}，{{上诉人工作单位和职务或者职业}}，{{住址}}。联系方式：{{上诉人联系方式}}。
**法定代理人/指定代理人：**{{法定代理人/指定代理人}}。
**委托诉讼代理人：**{{委托诉讼代理人}}。

**上诉请求：**
{{上诉请求}}

**上诉理由：**
{{上诉理由}}

**此致**
{{受理法院名称}}

**附：**本上诉状副本{{附件份数}}份

**起诉人：**
{{落款年月日}}
```

## 质量自检清单

1. 标题是否以 `# 民事上诉状` 开头？标题前是否有任何多余文字？
2. 上诉缘由是否包含了案由、原审法院、裁判日期、案号四个要素？
3. 上诉请求是否明确、具体、可执行？
4. 事实与理由是否有针对性地论证了原审裁判的具体错误（而非重复一审主张）？
5. 当事人是否标注了在原审中的诉讼地位？
6. 缺失信息是否用 `【待补充：XXX】` 占位？
