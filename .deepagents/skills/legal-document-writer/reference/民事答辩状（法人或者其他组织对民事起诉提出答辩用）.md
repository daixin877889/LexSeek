# 民事答辩状（法人或者其他组织对民事起诉提出答辩用）

## 文书概述

民事答辩状是民事诉讼中法人或其他组织作为被告，收到原告起诉状副本后，在法定期限内向人民法院提交的书面答辩意见。

**适用场景**：法人或其他组织（公司、事业单位、社会团体等）作为被告的书面答辩。

---

## 角色设定

撰写此文书的 AI 角色为**被告方法务/代理律师**。文风需理性、克制、精准。与公民答辩状相同，核心原则是「答」（逐项回应）与「辩」（逐一辩驳），但法人答辩需特别注意主体信息完整性和公章要求。

---

## 结构规范

民事答辩状（法人用）与公民用结构相同，但当事人信息格式不同：

1. 标题（全文唯一的一级标题 `# 民事答辩状`）
2. 首部（案号 + 答辩人法人信息 + 代理人信息）
3. 正文（逐项回应原告诉讼请求，表态 + 辩驳理由）
4. 尾部（致送法院 + 答辩人单位盖章 + 日期）

答辩人（法人）信息格式：
- 单位名称、住所地、统一社会信用代码
- 法定代表人/主要负责人：姓名、职务、联系方式
- 委托诉讼代理人（如有）

---

## 各板块写作指南

与公民答辩状完全相同的写作原则，详见 [民事答辩状（公民对民事起诉提出答辩用）.md](reference/民事答辩状（公民对民事起诉提出答辩用）.md)：

**原则一：「答」（回应）**
- 针对原告每一项诉讼请求，逐一、明确地表明态度
- 使用标准法律术语：「认可」「不认可」或「部分认可」

**原则二：「辩」（辩驳）**
- 每一项不认可的诉请，立即给出完整的事实与法律理由
- 论证紧密围绕「为什么不认可」

正文组织逻辑（自动择一）：
- **逻辑 A（总分结构）**：总述（法律关系定性 + 整体态度 + 抗辩立场）→ 逐项分述
- **逻辑 B（分总结构）**：逐项展开（表态 + 理由）→ 总结（概括立场 + 请求驳回）

法人答辩特别注意：
- 涉及公司行为的，说明相关内部决议程序是否合规
- 涉及合同履行的，引用合同条款原文
- 涉及金额的，逐笔列明计算过程和依据

```
**答辩人：**{{答辩人姓名}}，{{住址}}。
**法定代表人/主要负责人：**{{法定代表人/主要负责人}}，{{写明职务}}，联系方式：{{联系方式}}。
**委托诉讼代理人：**{{委托诉讼代理人}}。

**证据和证据来源，证人姓名和住所：**
{{证据和证据来源，证人姓名和住所}

**此致**
{{受理法院名称}}

**附：**本答辩状副本{{附件份数}}份

**答辩人：**
{{落款年月日}}
```

## 质量自检清单

1. 全文是否只有一个一级标题 `# 民事答辩状`？
2. 答辩人（法人）信息是否包含：名称、住所地、统一社会信用代码、法定代表人、联系方式？
3. 是否针对原告每一项诉讼请求逐一表态？
4. 每一项「不认可」是否都跟随了完整的事实与法律理由？
5. 落款是否同时具备「公司盖章」和「法定代表人签名或盖章」？
6. 缺失信息是否用 `【待补充：XXX】` 占位？
