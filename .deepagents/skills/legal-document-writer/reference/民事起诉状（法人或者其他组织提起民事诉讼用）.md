# 民事起诉状（法人或者其他组织提起民事诉讼用）

## 文书概述

民事起诉状是法人或其他组织因民事权利受到侵害或发生争议，以单位名义向人民法院提起诉讼时提交的书面文件。

**适用场景**：法人或其他组织（公司、事业单位、社会团体等）作为原告提起民事诉讼。

---

## 角色设定

撰写此文书的 AI 角色为**资深法律顾问/公司律师**。文风需理性、克制、精准。与公民起诉状不同，法人起诉状需特别注意主体信息的完整性和公章要求。

---

## 结构规范

民事起诉状（法人用）与公民用结构相同，但当事人信息格式不同：

1. 标题
2. 当事人信息（原告法人 → 被告 → 第三人）
3. 案由
4. 诉讼请求
5. 事实与理由
6. 结尾（此致 + 法院名称）
7. 落款（具状人单位盖章 + 法定代表人签名 + 日期）
8. 附件（副本 + 证据清单）

法人原告信息格式：
- 名称、住所地、统一社会信用代码
- 法定代表人/主要负责人：姓名、职务、联系方式
- 委托诉讼代理人（如有）

**重要**：法人起诉状落款必须同时具备「（公司盖章）」和「法定代表人（或主要负责人）：（签名或盖章）」，两者缺一不可。

---

## 各板块写作指南

### 诉讼请求

与公民起诉状相同的要求：
- 明确具体、全面完整、于法有据
- 涉及金钱写明金额、计算方式及起止日期
- 最后一项惯例为「本案诉讼费用由被告负担」

### 事实与理由（司法三段论）

与公民起诉状采用相同的写作方法：
- **第一层：法律关系建立** — 原被告之间在何时建立了何种法律关系
- **第二层：违约/侵权事实** — 被告实施了何种违约或侵权行为，逐项对应法律构成要件
- **第三层：法律后果推导** — 援引法律条文，论证被告应承担的法律责任

法人案件中特别注意事项：
- 写明合同的签订主体、签订时间、合同编号
- 涉及公司内部决议的（如担保需股东会决议），说明相关程序
- 涉及金额的计算，写明计算公式和依据

### 落款

- 必须分两行标注：
  - 第一行：「具状人：【单位名称】（公司盖章）」
  - 第二行：「法定代表人（或主要负责人）：【姓名】（签名或盖章）」
- 两者不可缺一，否则法院可能不予受理

## 文书排版模板

```
# 民事起诉状

**原告：**{{原告姓名}},{{住址}}。
**法定代表人/主要负责人：**{{法定代表人/主要负责人}}，联系方式：{{联系方式}}。
**委托诉讼代理人：**{{委托诉讼代理人}}。
**被告：**{{被告}}。

**诉讼请求：**
{{诉讼请求}}

**事实和理由：**
{{事实和理由}}

**证据和证据来源，证人姓名和住所：**
{{证据和证据来源，证人姓名和住所}}

**此致**
{{受理法院名称}}

**附：**本起诉状副本{{附件份数}}份

**起诉人：**
{{落款年月日}}
```

## 质量自检清单

生成前逐项确认：

1. 标题是否以 `# 民事起诉状` 开头？标题前是否有任何多余文字？
2. 原告（法人）信息是否包含：名称、住所地、统一社会信用代码、法定代表人、联系方式？
3. 落款是否同时具备「公司盖章」和「法定代表人签名或盖章」？
4. 「诉讼请求」是否明确、具体、可执行？
5. 「事实与理由」是否采用司法三段论？
6. 文书结束后是否有任何对话式说明文字？
7. 缺失信息是否用 `【待补充：XXX】` 占位？
