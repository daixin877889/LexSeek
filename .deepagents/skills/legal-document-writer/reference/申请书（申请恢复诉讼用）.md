# 申请书（申请恢复诉讼用）

## 文书概述

恢复诉讼申请书是民事诉讼中，在诉讼中止的原因消除后（如当事人死亡已确定继承人、丧失诉讼行为能力者已确定法定代理人、等待的另一案件已审结等），当事人向人民法院提交的书面申请文件。目的是请求法院恢复审理已被裁定中止的案件。

**适用场景**：中止诉讼的原因已经消除，当事人请求法院恢复诉讼程序。也适用于和解协议未获履行而申请恢复执行的情形。

---

## 角色设定

撰写此文书的 AI 角色为**精通民诉法诉讼中止与恢复制度的诉讼律师**。文风需简洁、程序导向，聚焦于中止原因已消除的事实。

---

## 结构规范

恢复诉讼申请书按以下顺序排列：

1. 标题
2. 申请人信息
3. 法定代理人/委托诉讼代理人信息（如有）
4. 请求事项
5. 事实和理由
6. 结尾（此致 + 法院名称）
7. 申请人署名 + 日期

---

## 各板块写作指南

### 请求事项

- 简明扼要
- 格式：「请求贵院依法恢复对（{{案号}}）{{案由}}一案的审理」
- 或指明中止类型：「请求贵院依法恢复对（{{案号}}）{{案由}}一案的诉讼程序，并继续审理」

### 事实和理由

1. **陈述原诉讼及中止情况** -- 案号、案由、中止诉讼的裁定时间及原因
2. **说明中止原因已消除** -- 具体证明中止事由已不再存在的事实
3. **援引法律依据** -- 引用《民事诉讼法》第一百五十六条的规定

常见恢复诉讼的情形：中止诉讼的原因消除，包括但不限于：
- 等待另一案审结 -> 该案已生效
- 当事人死亡 -> 继承人已确定并参加诉讼
- 当事人丧失诉讼行为能力 -> 法定代理人已确定
- 当事人因不可抗拒事由不能参加诉讼 -> 事由已消除
- 先决问题已解决

写作原则：
- 应附相关证明材料（如另一案的生效判决书、继承公证书等）
- 法院收到申请后，经审查属实即可恢复诉讼
- 文风应当程序化，聚焦于"原因消除"这个核心事实

## 文书排版模板

```
申请书
申请人：{{申请人姓名}}，{{性别}}，{{出生年月日}}生，{{民族}}，{{申请人工作单位和职务或者职业}}，{{住址}}。联系方式：{{申请人联系方式}}。 
法定代理人/指定代理人：{{法定代理人/指定代理人}}。 
委托诉讼代理人：{{委托诉讼代理人}}。 
请求事项： 
{{请求事项}} 
事实和理由： 
{{事实和理由}} 。 
此致 
{{受理法院名称}} 
                                      申请人: 
{{落款年月日}}
```

---

## 质量自检清单

1. 标题是否以 `# 申请书` 开头？
2. 请求事项是否明确请求恢复对特定案号案件的诉讼程序？
3. 事实和理由是否具体说明了中止原因及该原因已消除的事实？
4. 是否注明了原中止裁定的案号和时间？
5. 缺失信息是否用 `{{字段}}` 占位？
