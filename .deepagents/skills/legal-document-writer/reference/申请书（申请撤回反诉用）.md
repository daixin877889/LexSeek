# 申请书（申请撤回反诉用）

## 文书概述

撤回反诉申请书是民事诉讼中反诉原告（本诉被告）在提起反诉后，在法院作出判决之前，自愿放弃反诉请求，向人民法院提交的书面申请文件。目的是请求法院准许撤回已提起的反诉。

**适用场景**：反诉原告（本诉被告）反诉后，因双方和解、履行完毕、证据变化、诉讼策略调整等原因，决定不再继续反诉，申请撤回。

---

## 角色设定

撰写此文书的 AI 角色为**精通民诉法反诉制度的诉讼律师**。文风需简洁、明确，体现自愿性与合法性。

---

## 结构规范

撤回反诉申请书按以下顺序排列：

1. 标题
2. 申请人信息（反诉原告）
3. 法定代理人/委托诉讼代理人信息（如有）
4. 请求事项
5. 事实和理由
6. 结尾（此致 + 法院名称）
7. 申请人署名 + 日期

---

## 各板块写作指南

### 请求事项

- 简明扼要
- 格式：「请求贵院依法准许申请人撤回对{{被反诉人姓名}}的反诉」
- 如属已受理尚未审理：「请求贵院依法准予申请人撤回已提起的反诉」

### 事实和理由

1. **陈述反诉案件基本情况** -- 本诉案号、反诉提起时间、反诉请求、案由
2. **说明撤回反诉的原因** -- 如双方和解、被反诉人已履行义务、申请人改变诉讼策略
3. **强调撤回反诉符合法律规定** -- 反诉人在判决前有权撤回、不违反法律规定、不损害他人合法权益

写作原则：
- 法院是否准许撤回反诉需审查是否违反法律规定和公序良俗
- 撤回反诉不影响本诉的继续进行
- 撤回反诉后，反诉人不得再次基于同一事实和理由提起反诉
- 本诉原告（被反诉人）对撤回反诉有异议的，需记录在案

```
法定代理人/指定代理人：{{法定代理人/指定代理人}}。 
委托诉讼代理人：{{委托诉讼代理人}}。 
请求事项： 
{{请求事项}} 
事实和理由： 
{{事实和理由}} 。 
此致 
{{受理法院名称}} 
                                      申请人: 
{{落款年月日}}
```

## 质量自检清单

1. 标题是否以 `# 申请书` 开头？
2. 申请人是否明确为"反诉原告"？
3. 请求事项是否明确请求撤回反诉？
4. 事实和理由是否体现了撤回的自愿性和合法性？
5. 缺失信息是否用 `{{字段}}` 占位？
