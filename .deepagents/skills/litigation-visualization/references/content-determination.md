# 图表内容确定方法

## 确定内容的三个步骤

### 第一步：全面罗列

将所有可能相关的信息先罗列出来，不要急于筛选和取舍。

**罗列的内容包括**：
- 当事人的所有行为和事件
- 时间节点
- 涉及的金额、数量等数据
- 相关的法律规定和司法解释
- 可能的影响因素

**注意事项**：
- 暂时判断信息的重要性，先罗列再说
- 可以分多张纸罗列，避免遗漏
- 保持逻辑顺序，按时间或分类罗列

### 第二步：逻辑整合

按照一定的逻辑对罗列的信息进行归类整理。

**常见的逻辑整合方式**：

1. **按时间顺序整合**：适用于时间要素重要的案件
2. **按主体分类整合**：适用于主体较多的案件
3. **按事件性质整合**：如"申请行为"和"裁判行为"
4. **按争议焦点整合**：围绕核心争议组织信息

**整合的目标**：
- 发现信息之间的关联
- 形成清晰的结构
- 突出核心内容

### 第三步：精简内容

删除无关信息，突出核心要点。

**精简的原则**：

1. **保留与争议焦点相关的信息**：这是最重要的标准
2. **保留关键事实**：如关键时间节点、关键行为
3. **删除无关细节**：如与争议无关的背景信息
4. **删除重复信息**：避免同一信息重复出现

**精简的度**：
- 不能过于简化导致信息不全
- 不能过于复杂失去可视化意义
- 以"必要而充分"为目标

## 不同对象的内容侧重

### 提交给法官的图表

**内容侧重**：
- 与案件核心争议有关的案件事实
- 对己方有利的事实和证据
- 法律适用的关键时间节点或关系
- 避免过多主观评价

**注意事项**：
- 保持客观公正
- 在事实基础上突出己方观点
- 避免对法官进行"说教"

### 提交给客户的图表

**内容侧重**：
- 完整的案件事实梳理
- 法律关系的分析和论证
- 案件的优势和风险
- 可能的诉讼策略

**注意事项**：
- 让客户理解案件情况
- 让客户理解律师的工作价值
- 客观提示风险

### 内部讨论使用的图表

**内容侧重**：
- 尽量翔实
- 增加事实细节
- 包含所有可能相关的信息
- 标注问题和不确定性

**注意事项**：
- 全面性优先
- 不必考虑美观
- 可以标注待核实的信息

## 内容确定的常见误区

### 1. 过度精简

为了图表简洁而删除重要信息，导致信息不全。

### 2. 保留过多无关信息

"全面罗列"后不舍得删除，导致图表拥挤混乱。

### 3. 忽视受众需求

没有考虑不同对象的需求差异，用同一套内容。

### 4. 主观筛选

根据主观判断而非客观标准筛选内容，可能遗漏重要事实。

## 内容确定的实践建议

1. **团队讨论**：团队一起讨论哪些信息重要，避免个人盲点
2. **对照争议焦点**：不断问自己"这个信息与争议焦点相关吗？"
3. **多次迭代**：第一版可能不完美，多次修改迭代
4. **请他人看**：请不了解案件的人看图表，能否理解核心事实
