# 两张图工作法

## 核心理念

推行"两张图"工作法，即每个案件要画两张图：
- 一张"案件事实图"
- 一张"法律关系图"

## 案件事实图

### 定义

案件事实图是以时间线为基础，客观、真实地反映案件中各主体行为、事件发生先后顺序及相互关系的图表。

### 主要特点

1. **客观性**：以客观事实为基础，避免主观判断
2. **时间导向**：以时间轴为核心框架
3. **行为主体分层**：将不同主体的行为在空间上区分开来
4. **关键事实突出**：突出与案件争议相关的核心事实

### 适用场景

1. 事件发生的先后顺序对案件定性有重要影响
2. 需要梳理复杂的程序性事实
3. 需要对比不同主体在同一时间段的行为
4. 需要展示时间区间概念（如保证期间、诉讼时效等）

### 绘制要点

1. 确定时间轴方向（横向或纵向）
2. 在时间轴上标注关键时间节点
3. 合理利用纵向空间进行行为分层或对比
4. 使用括号、箭头或变换线条颜色表达时间区间
5. 颜色区分不同主体或不同性质的行为

### 配色方案

- 不同主体用不同颜色区分（如甲方用蓝色，乙方用绿色）
- 违约行为用红色标注
- 正常履行行为用蓝色或绿色
- 保持同一案件多张图表的配色一致性

## 法律关系图

### 定义

法律关系图是以主体间的关系为核心，反映各方当事人在法律层面上的权利义务关系、交易结构、股权架构等的图表。

### 主要特点

1. **关系导向**：以主体间的关系为表达核心
2. **结构化**：通过图表结构体现关系性质
3. **主观观点融入**：可融入对案件的法律分析和论证
4. **逻辑清晰**：通过箭头、连线等表达关系方向和性质

### 适用场景

1. 案件主体数量较多（三个以上）且需要梳理关系
2. 各主体之间的法律关系复杂
3. 关系本身是案件的争议焦点
4. 需要展示交易结构、股权架构、权利义务关系
5. 需要向客户解释法律分析思路

### 绘制要点

1. 以主体为节点，用图形表示
2. 用箭头表示关系方向（如资金流向、权利义务方向）
3. 通过连线标注关系性质（如借贷、担保、买卖）
4. 用不同颜色或线条粗细区分不同性质的关系
5. 保持图表布局均衡，避免交叉混乱

### 配色方案

- 不同主体节点使用不同颜色
- 主要关系用粗线，次要关系用细线
- 确定关系用实线，潜在关系用虚线
- 核心争议关系用醒目颜色（如红色或橙色）

## 两张图的配合使用

### 不同呈送对象的选择

| 呈送对象 | 推荐图表类型 | 原因 |
|---------|-------------|------|
| 法官 | 优先案件事实图 | 法官需要客观事实进行裁判 |
| 客户 | 优先法律关系图 | 客户需要理解法律分析和策略 |
| 团队内部 | 两张图并用 | 全面讨论需要完整信息 |

### 制作顺序建议

1. **先画案件事实图**：客观梳理事实，确保准确性
2. **后画法律关系图**：在事实基础上进行法律分析
3. **交叉验证**：两张图应相互印证，避免矛盾

### 配色一致性原则

- 案件事实图和法律关系图中，相同主体应使用相同颜色
- 相同类型的行为或关系应使用相同的颜色编码
- 建立配色方案：为每个主体/行为类型分配固定颜色

## 两张图工作法的价值

1. **事实梳理**：帮助厘清混乱的案件事实
2. **法律分析**：通过图表发现法律关系的本质
3. **庭审效果**：在法庭上更好地向法官传达观点
4. **客户沟通**：让客户理解分析和案件情况
5. **案件复盘**：便于案件结束后的总结和经验积累
