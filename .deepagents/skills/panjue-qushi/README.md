# 判决趋势预测分析方法论

## 简介

本skill提供基于法官裁判视角的判决趋势预测方法论，用于：

- 模拟法官的裁判思维路径（八步法审理视角）
- 分析原被告主张的法律合理性
- 预判被告可能的抗辩
- 分配举证责任（罗森贝克法则）
- 判断证明标准
- 进行涵摄判断和要件成就分析
- 核查案件中涉及的定量计算
- 审查法律时效性与新旧法适用
- 预测判决趋势方向

## 使用场景

**适用场景**：
- 律师分析案件走向和诉讼策略
- 律师模拟法官裁判思路预判判决
- 法官模拟裁判思路
- 法律工作者评估案件可行性
- 当事人了解案件风险

**不适用场景**：
- 需要证据真实性评估的案件
- 需要量化胜诉率预测的咨询
- 涉及事实调查无法律争议的案件

## 快速使用指南

### 最简流程（简单案件）

1. 确认请求权基础分析结果
2. 识别要件事实清单
3. 进行涵摄判断
4. 输出预测结论

### 完整流程（复杂案件）

1. 调用请求权基础模块获取要件事实清单
2. 模拟法官裁判路径（八步法预判）
3. 涉及数字时执行定量核查
4. 执行法律时效审查和增量更新检索
5. 触发诉讼时效审查条件时执行
6. 新旧法对比分析
7. 司法解释适用分析
8. 法律合理性审查（含举证分配、抗辩预判）
9. 涵摄判断（要件成就状态）
10. 证明标准分析
11. 定性分析输出预测

## 与请求权基础模块的关系

判决趋势预测模块是请求权基础分析的下游模块：

- **请求权基础模块**：识别请求权、确定要件事实清单、分配举证责任
- **判决趋势预测模块**：基于要件事实清单进行涵摄判断、要件成就分析、趋势预测

关键衔接点：
| 衔接内容 | 来源 | 用途 |
|----------|------|------|
| 要件事实清单 | 请求权基础模块 | 涵摄判断的检验清单 |
| 举证责任分配 | 请求权基础模块 | 统一使用罗森贝克法则 |
| 争议焦点 | 请求权基础模块 | 法律合理性审查的要素来源 |

使用时应先完成请求权基础分析，再调用本模块进行趋势预测。
