# 定量分析核查法

当案件材料中涉及任何关键的数字、款项、比例、日期或期限计算时，必须强制执行分步核查规则。

## 执行步骤

### 第一步：要素提取与公式陈列

**动作**：从案情中明确提取所有参与计算的基础数字，清晰列出计算公式。严禁模糊处理。

**必须提取的要素**：
- 单价、数量
- 利率、本金
- 已支付金额、总金额
- 日期、期限

**公式陈列示例**：
"为核查房款支付比例，需先计算总房款。计算公式为：`总房款 = 单价 × 面积`。基础数据为：单价 = 2650元/平方米，面积 = 104.4平方米。"

### 第二步：计算过程展示

**动作**：一步一步展示计算过程，明确标出最终结果。严禁直接跳跃到结论数字。

**示例**：
"计算过程：`总房款 = 2650 × 104.4 = 276660`元。计算得出的总房款为 276,660 元。"

### 第三步：结论得出与实质性判断

**动作**：基于计算结果进行逻辑判断。如果计算结果与陈述事实存在微小差异，必须引入"实质性判断"原则。

**实质性判断三要素**：

1. **精确比较**：计算差异的具体数值和比例
2. **商业实践判断**：差异是否属于商业实践中可被接受的微小零头或计算误差
3. **法律影响判断**：差异是否影响合同目的实现或构成根本违约

**示例**：
"1. 精确比较：计算得出的总房款为 276,660 元。原告已支付 27.66万元，即 276,600元。两者差异为 `276660 - 276600 = 60`元。

2. 实质性判断：差异金额60元，占总价276,660元的比例约为0.02% (`60 / 276660 ≈ 0.0002`)。在房屋买卖等大额交易中，此等比例的差异通常被视为可忽略的计算尾差或交易零头，不影响'已付清全款'这一核心事实的认定，也不构成根本性违约。

3. 最终结论：因此，尽管存在60元的微小差异，但在法律和事实上应认定原告已**实质上履行了全部付款义务**。"

## 核查清单

| 检查项 | 要求 |
|--------|------|
| 基础数字提取 | 所有参与计算的数字都已明确提取 |
| 公式陈列 | 计算公式已清晰列出 |
| 计算过程 | 逐步展示，无跳跃 |
| 差异分析 | 存在差异时进行实质性判断 |
| 结论融入 | 结论自然融入后续法律分析 |

## 注意事项

- 差异的实质性判断必须结合交易性质（大额交易vs小额交易）
- 比例计算是判断重要性的参考指标
- 结论应融入后续法律分析，作为认定事实的关键依据
