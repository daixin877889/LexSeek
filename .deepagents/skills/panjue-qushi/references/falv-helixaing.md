# 法律合理性审查法

法律合理性审查以法官裁判视角为主线，审查原被告主张的法律合理性，并融入举证责任分配分析。

**注意**：详细的抗辩分析由独立的抗辩分析模块处理，本模块仅需识别举证风险点，为预测提供依据。

## 核心原则

### 仅基于陈述原则

审查对象仅限于当事人陈述的"事实主张"和"主要理由"，绝对不涉及对任何一方证据的采信、评估或考量。

### 初步评估性质

法律合理性是纯粹法律逻辑和框架内的初步评估，假设所陈述内容为真，判断诉请在法律逻辑上能否成立。

### 动态叠加原则

审查不应只看静态的原告主张，还要考虑举证责任分配对结果的影响，识别关键举证风险点。

## 第一步：审查原告主张的法律合理性

针对每一个核心法律要素/争议焦点（来源于请求权基础分析）：

### 审查要素

1. **事实主张**：原告陈述的事实内容
2. **主要理由**：原告提出的法律依据和论证逻辑
3. **要件匹配**：原告主张能否归入请求权基础的要件事实
4. **法律合理性**：假设原告所述为真，其诉请在法律逻辑上能否成立

### 审查结论选项

| 结论 | 含义 |
|------|------|
| 完全不具法律合理性 | 假设为真，诉请在法律逻辑上完全不能成立 |
| 部分具有法律合理性 | 假设为真，诉请部分能在法律逻辑上成立 |
| 完全具有法律合理性 | 假设为真，诉请在法律逻辑上完全成立 |

## 第二步：识别举证风险点

### 从要件事实反向推导

识别哪些要件可能存在举证困难或争议：

1. **原告举证要件的风险点**：权利发生要件中，哪些事实原告难以证明？
2. **被告可能主张的抗辩方向**：被告可能从哪些角度进行抗辩？（如合同无效、已清偿、诉讼时效等）
3. **标注关键风险要件**：在要件成就分析中明确标注举证风险

### 注意

详细的抗辩分析由独立的抗辩分析模块处理，本模块仅需识别举证风险点，为预测提供依据。

## 第三步：举证责任分配分析

### 罗森贝克法则

举证责任分配遵循"法规要件分类说"：

- **正向框架（权利发生要件）**：主张权利成立的一方举证
  - 例如：合同成立、侵权构成要件、借贷关系成立
- **反向框架（权利障碍要件）**：主张权利受阻的一方举证
  - 例如：合同无效、诉讼时效已过、不可抗力

### 举证分配检查要点

1. **列出要件事实清单**（来源于请求权基础分析）
2. **分类要件性质**：
   - 权利发生要件 → 原告举证
   - 权利障碍要件 → 被告举证
   - 权利消灭要件 → 被告举证
3. **评估举证难度**：关键要件举证困难的一方败诉风险高
4. **检查特殊情形**：是否存在举证责任倒置（过错推定、无过错责任等）

### 举证风险判断

| 举证方举证难度 | 对预测结果的影响 |
|----------------|------------------|
| 低难度（证据充分） | 支持该方主张 |
| 中难度（证据基本够） | 可能支持，但有争议风险 |
| 高难度（证据不足） | 可能不支持，举证失败风险 |
| 极高难度（证据缺失） | 很可能不支持，败诉风险高 |

## 第四步：审查被告抗辩的法律合理性

### 检查用户输入

首先检查用户是否提供了被告方的"事实主张"和"主要理由"。

### 执行条件

- **未提供被告陈述**：基于预判的抗辩清单进行假设性分析
- **提供被告陈述**：参照原告审查结构逐要素分析

### 审查结论选项

与原告审查相同，判断被告抗辩的法律合理性程度。

## 审查框架

```
核心法律要素/争议焦点（来源于请求权基础分析）
        │
        ├──→ 原告主张审查
        │         │
        │         ├── 要件匹配分析
        │         ├── 法律合理性判断
        │         └── 举证难度评估
        │
        ├──→ 举证风险点识别
        │         │
        │         ├── 原告举证要件的风险点
        │         └── 被告可能抗辩方向（概要）
        │
        └──→ 被告主张审查（条件执行，若用户提供被告陈述）
                  │
                  ├── 抗辩要件匹配
                  ├── 法律合理性判断
                  └── 举证难度评估
```

若用户提供被告陈述：（参照原告审查结构）

## 再次强调

此处的"法律合理性"判断：
- 绝对不涉及对证据的采信、评估或考量
- 仅为基于单方或双方所陈述"事实与理由"的纯粹法律逻辑评估
- 是假设性的初步评估，不是最终裁判结论
- 必须融入举证责任分配分析——举证困难的一方败诉风险高
