# 判决趋势预测法

判决趋势预测以要件成就分析为核心，基于涵摄判断推导裁判方向，必须是定性逻辑分析。

## 核心原则

### 定性分析原则

预测必须是基于法律逻辑和事实要素的定性分析，而非定量计算。任务是分析各种可能性，说明导致不同结果的关键变量。

### 禁止量化捏造

严禁凭空捏造或引用无法核实的胜诉率、概率、百分比、赔偿比例等任何量化数据。

### 要件成就导向

预测结论应基于要件成就状态推导——全部要件成就则支持诉请，有要件不成就则驳回诉请。

### 语言风格要求

必须使用严谨、中立且包含可能性的法律预测语言。

## 语言规范

### 必须使用的表达

- "倾向于支持"
- "可能性较大"
- "将取决于……的认定"
- "若……则可能……"
- "有较大可能性"
- "可能会被认定"
- "取决于举证能否达到证明标准"

### 禁止使用的表达

- "有80%的概率"
- "胜诉率XX%"
- "必定会"
- "肯定会"
- "一定不会"

## 要件成就分析框架

### 要件成就状态

基于涵摄判断，每个要件有以下成就状态：

| 状态 | 含义 | 对诉请的影响 |
|------|------|--------------|
| 完全成就 | 事实明确归入要件 | 支持该要件 |
| 部分成就 | 部分事实归入，有争议 | 需进一步举证或认定 |
| 不成就 | 事实无法归入要件 | 阻却诉请 |
| 待认定 | 关键事实有争议 | 取决于举证结果 |

### 要件成就与预测结论的关系

```
全部要件完全成就 → 倾向于支持诉请
    │
    ├── 有要件部分成就 → 可能部分支持
    │       │
    │       └── 争议程度决定支持比例
    │
    ├── 有要件不成就 → 倾向于驳回诉请（该要件阻却）
    │       │
    │       └── 其他要件成就则可能部分驳回
    │
    └── 有要件待认定 → 结论取决于举证结果
            │
            ├── 举证方举证成功 → 成就 → 倾向于支持
            └── 举证方举证失败 → 不成就 → 倾向于驳回
                    │
                    └── 举证困难方承担不利后果
```

## 预测方法

### 关键变量分析法

识别影响判决结果的关键变量（即"待认定"要件），分析不同变量取值下的可能结果：

```
要件A举证成功 → 结果倾向X
要件A举证失败 → 结果倾向Y
要件B存在 → 可能影响结果为Z
```

### 条件推演法

构建条件判断逻辑链：

```
若要件1成就 → 检查要件2
    若要件2成就 → 检查要件3
        若要件3成就 → 倾向于支持
        若要件3不成就 → 倾向于驳回
    若要件2不成就 → 倾于驳回
若要件1不成就 → 倾于驳回
```

### 举证责任分析法

基于举证责任分配预测结果：

```
要件事实A → 原告举证 → 若原告举证困难 → 倾向于驳回
抗辩事实B → 被告举证 → 若被告举证困难 → 抗辩不成立 → 倾向于支持
```

### 证明标准分析法

基于证明标准判断举证风险：

```
特殊事项（欺诈、胁迫等） → 排除合理怀疑标准 → 举证难度更高 → 需要更充分证据
一般事项 → 高度盖然性标准 → 举证难度适中 → 基本证据链即可
```

## 涵摄结论推导

### 推导步骤

1. **汇总要件成就状态**：基于涵摄判断的结果
2. **识别关键争议要件**：哪些要件是"待认定"或"部分成就"
3. **分析举证责任归属**：关键要件由谁举证
4. **评估举证风险**：举证方能否达到证明标准
5. **推导预测结论**：基于要件成就和举证风险的裁判方向

### 推导逻辑表

| 要件成就状态分布 | 关键要件举证方 | 举证风险评估 | 预测倾向 |
|------------------|----------------|--------------|----------|
| 全部完全成就 | — | — | 倾向于支持 |
| 有要件不成就 | — | — | 倾于驳回该部分 |
| 有要件待认定 | 原告 | 高风险 | 倾于驳回（举证失败风险） |
| 有要件待认定 | 原告 | 低风险 | 倾于支持（举证可能成功） |
| 有要件待认定 | 被告 | 高风险 | 倾于支持（被告举证失败） |
| 有要件待认定 | 被告 | 低风险 | 倾于驳回（被告举证可能成功） |

## 注意事项

- 预测结论必须与要件成就分析逻辑一致
- 所有预测必须基于要件成就状态和举证责任分析
- 必须明确说明预测的前提条件（"若XX则YY"）
- 结论应保持适度审慎，避免绝对化表述
- "待认定"要件是预测的关键变量——取决于举证结果
- 举证困难的一方承担不利后果——这是预测的重要判断依据
- 证明标准影响举证难度——特殊事项需要更高证明程度
