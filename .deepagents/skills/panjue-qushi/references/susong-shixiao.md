# 诉讼时效审查法

诉讼时效审查遵循"特别法优于一般法"原则，严禁直接默认适用《民法典》三年普通时效。

## 触发条件

诉讼时效审查并非自动启动，需满足以下条件之一：

1. **当事人明确提出**：被告抗辩理由中明确提及"诉讼时效"相关主张
2. **存在明显时效风险**：案件事实发生时间与当前时间跨度巨大，构成不可忽视的重大程序性风险

## 执行原则

一旦审查被触发，严格遵循"特别法优于一般法"原则，按以下两步强制程序执行，严禁跳步或颠倒顺序。

## 第一步：强制性特别法检索

**任务**：首先执行专门的、只针对本案所属特别法领域的检索。

**检索要求**：
- searchLawTool查询必须包含：(1) 体现案件性质的关键词；(2) "诉讼时效"；(3) "司法解释"/"例外"/"特殊规定"/"不受限制"
- 严禁引用或联想《民法典》任何条款

**常见特别法时效规定**：

| 领域 | 特殊时效规定 |
|------|--------------|
| 公司法 | 股东出资、股权转让等特殊时效 |
| 海商法 | 海上货物运输赔偿请求时效（1年） |
| 保险法 | 人寿保险外的保险金请求时效（2年） |
| 票据法 | 持票人对前手的追索权时效（2年） |
| 拍卖法 | 拍卖成交后权利主张时效 |
| 产品质量 | 产品缺陷损害赔偿时效（2年） |

## 第二步：条件性一般法适用

**启动条件**：当且仅当第一步检索明确返回"在《XX法》中未发现相关特殊规定"结果时，才可检索、引用和适用《民法典》一般规定。

**陈述要求**：若最终适用《民法典》，必须在分析开头明确陈述：

> **经对《XX法》（例如《公司法》）的专门检索，未发现适用于本案情形的诉讼时效特殊规定，故本案应适用《民法典》关于诉讼时效的一般规定。**

## 执行流程

```
审查触发条件判断
        │
        ▼ (触发)
第一步：特别法检索（禁止引用民法典）
        │
        ├──→ 发现特殊规定 → 适用特殊规定
        │
        └──→ 未发现特殊规定
                │
                ▼
第二步：一般法适用（民法典）
        │
        ▼
明确陈述检索结果后适用一般规定
```

## 结论融入方式

诉讼时效审查结论应在"综合结论（原告阶段）"或相关要素的"简要理由"中，以一句话形式自然融入，无需单独列出标题。

## 绝对禁令

- 第一步检索中严禁引用或联想《民法典》任何条款
- 唯一目标是确认特别法中是否存在特殊规定
- 未完成第一步不得进入第二步
