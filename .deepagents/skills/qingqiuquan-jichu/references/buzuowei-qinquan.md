# 不作为侵权四合一检视

不作为侵权中，加害行为、因果关系、不法性、过错的判断合并为一个核心判断。

## 核心判断

**是否有作为义务且违反了该义务**

违反作为义务即同时认定：
- 加害行为：违反作为义务即存在
- 因果关系：由作为义务违反推定（除非即使尽到义务也不可避免）
- 不法性：借助作为义务违反积极确认
- 过错：作为义务违反本身即"疏于尽交往中必要注意"

## 检视程式

```
绝对权受侵害 → 作为义务 → 义务违反（四要件合一） → 被告抗辩
```

## 作为义务来源

| 来源 | 示例 |
|------|------|
| 法定义务 | 安全保障义务、监护义务 |
| 先行行为 | 因先行行为引发危险而负有救助义务 |
| 合同约定 | 合同约定的保护义务 |

## 与作为侵权的区别

| 作为侵权 | 不作为侵权 |
|----------|------------|
| 四要件分别独立判断 | 四要件合并为一判断 |
| 加害行为独立认定 | 违反作为义务即加害行为 |
| 因果关系单独证明 | 由义务违反推定 |
| 不法性可能推定 | 由义务违反积极确认 |

## 执行步骤

1. 确认绝对权受侵害
2. 判断作为义务是否存在
   - 无作为义务 → 不构成，终止
3. 认定作为义务违反
   - 违反 → 四要件同时成立
4. 检视被告抗辩
   - 被告能否证明即使尽到义务损害仍不可避免
5. 结论：明确不作为侵权是否成立

## 不适用情形

- 作为侵权：须分别检视四要件
- 框架权利侵权：不法性不能推定，须法益衡量
