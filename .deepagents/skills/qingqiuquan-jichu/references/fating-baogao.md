# 法庭报告技术

诉讼审理的五阶段框架。

## 五阶段结构

```
程序阶段 → 原告阶段 → 被告阶段 → 证据阶段 → 裁判阶段
```

## 核心原则

**三先原则**：
- 程序先于实体
- 法律先于事实
- 原告先于被告

## 各阶段详解

### 程序阶段

审查程序要件：
- 管辖权
- 当事人适格
- 诉讼时效等

**判停**：程序要件不满足 → 程序性驳回

### 原告阶段（陈述阶段）

**纯法律审查**：假设原告诉述为真，检视其是否满足至少一项请求权基础的适用前提。

- 不考虑陈述真假
- 遍检所有可能的请求权基础
- 用三层四步检视原告诉述的法律合理性

**判停**：原告陈述法律不合理 → 原告败诉

### 被告阶段（陈述阶段）

**纯法律审查**：假设被告诉述为真，检视其是否满足防御规范的适用前提。

- 不考虑陈述真假
- 检视被告抗辩的法律合理性

**判停**：被告抗辩法律不合理 → 原告胜诉（不必举证）

### 证据阶段

仅对双方陈述不一致且构成要件/抗辩要件的事项分配举证。

法律审查阶段已过滤大量无争议事项。

### 裁判阶段

综合法律审查和证据认定结果，形成裁判建议。

## 效率优势

大量案件在法律审查阶段（原告阶段或被告阶段）即可结案，不必进入证据阶段。

| 案型 | 处理 |
|------|------|
| 原告陈述不合理 | 原告阶段即败诉 |
| 原告合理+被告不合理 | 原告不经举证即胜诉 |
| 双方均合理 | 进入证据阶段 |

## 执行步骤

1. 程序阶段审查
2. 原告阶段：假设原告诉述为真，遍检请求权基础
3. 被告阶段：假设被告诉述为真，检视抗辩
4. 证据阶段：仅争议事项举证
5. 裁判阶段：形成裁判建议
