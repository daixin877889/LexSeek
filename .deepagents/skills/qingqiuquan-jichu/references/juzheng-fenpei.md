# 举证分配法

确定每项要件的举证责任分配。

## 基本规则

举证分配由实体法的要件-抗辩结构决定：

| 步骤 | 内容 | 举证主体 |
|------|------|----------|
| 步骤1 | 积极要件 | 原告举证 |
| 步骤2 | 权利未发生抗辩 | 被告举证 |
| 步骤3 | 权利消灭抗辩 | 被敌举证 |
| 步骤4 | 权利行使抗辩 | 被敌举证 |
| 反抗辩 | 抗辩的抗辩 | 原告举证 |

## 条文表述与举证逻辑的冲突

立法条文的表述可能违反举证逻辑：

**典型案例**：《民法典》第235条"无权占有"

| 分析 | 说明 |
|------|------|
| 条文表述 | "无权占有"似为原告应举证的积极要件 |
| 实际逻辑 | "无权"是否定性表述，应为被告举证"有占有本权"的抗辩 |
| 处理 | 通过解释纠正，不被条文措辞误导 |

## 积极要件与消极抗辩的区分

| 类型 | 特征 | 举证主体 |
|------|------|----------|
| 积极要件 | 条文正面表述要件 | 原告举证 |
| 消极抗辩 | 否定性表述（"无权""没有"） | 被告举证存在相反事由 |

## 执行步骤

1. 定位要件在三层的哪一步
2. 确定举证主体：积极要件→原告；消极抗辩→被告；反抗辩→原告
3. 检查条文表述是否与举证逻辑一致
4. 输出举证分配方案：列出要件→举证主体对照表

## 注意事项

- 举证分配是实体法问题，非程序法问题
- 中国法中存在举证责任倒置的法定例外（如环境污染、产品责任）
- 条文措辞可能违反举证逻辑，需通过解释纠正
