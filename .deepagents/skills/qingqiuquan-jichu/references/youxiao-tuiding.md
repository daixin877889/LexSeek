# 合同有效推定原则

合同效力的检视遵循推定结构。

## 核心原则

**已成立的合同推定为有效，对此提出质疑者负担举证之责。**

## 检视结构

```
成立 → 推定有效 → 效力质疑者举证 → 判断
```

### 成立阶段

只需证明合意存在，谁主张合同存在谁举证。

### 推定有效

合同一经成立即推定有效。

### 效力质疑

质疑合同有效者承担举证责任，须证明存在效力瑕疵事由。

## 效力瑕疵分类

按三层四步的逻辑组织：

| 类型 | 内容 | 举证主体 |
|------|------|----------|
| 效力未发生抗辩 | 无行为能力、违反强制规定、代理权滥用、恶意串通等 | 被告举证 |
| 效力消灭抗辩 | 撤销权行使、解除条件成就等 | 主张方举证 |

## 与三层四步的关系

有效推定是三层四步在合同效力领域的专门应用：

| 三层四步 | 合同效力对应 |
|----------|--------------|
| 步骤1：积极要件 | 合同成立 |
| 步骤2：未发生抗辩 | 效力未发生抗辩 |
| 步骤3：消灭抗辩 | 效力消灭抗辩 |

## 执行步骤

1. 确认合同成立（判断是否存在合意）
   - 未成立 → 不存在效力问题，终止
2. 推定合同有效
3. 检视效力未发生抗辩（由质疑方举证）
4. 检视效力消灭抗辩（由主张方举证）
5. 结论：明确合同有效/无效/可撤销/效力待定

## 注意事项

- 成立问题（合意是否存在）是事实判断，不在有效推定范围内
- 消费者保护、劳动合同等倾斜保护领域，有效推定可能需要修正
- 效力性强制规定与管理性强制规定的区分标准尚不统一
