# 预选排序法

面对案件事实，找出所有可能的请求权基础并确定检视顺序。

## 标准检视顺序

```
合同请求权 → 类似合同请求权 → 无因管理 → 物法请求权 → 不当得利 → 侵权请求权
```

**合同排最先的原因**：有效合同可排除任意规范适用，也可能排除其他各类请求权成立。

## 排序原则

| 原则 | 内容 | 说明 |
|------|------|------|
| 合乎逻辑 | 越特别越在先 | 特别法优先于一般法 |
| 诉讼经济 | 构要件越少越在先 | 更容易满足，快速得出结论 |

## 预选步骤

**第一步——预选**

按标准顺序逐类搜索可能的法条：
- 排除明显不成立的选项
- 排除规范排斥竞合（逻辑互斥的规范对，保留特别规范）

**第二步——排序**

- 越特别的请求权基础检视顺位越在先
- 构成要件越少的请求权基础检视顺位越在先

## 规范排斥竞合

多项规范在逻辑上互斥，只需检视特别规范：

| 情形 | 处理 |
|------|------|
| 违约损害赔偿 vs 一般侵权 | 违约为特别规范，优先检视 |
| 特别法 vs 一般法 | 特别法优先 |

## 执行步骤

1. 梳理原告诉请："谁向谁请求什么"
2. 预选请求权基础：按标准顺序逐类搜索
3. 排除规范排斥竞合
4. 确定检视顺序：按"越特别越在先""要件越少越在先"排列
5. 输出检视计划：列出有序的请求权基础清单
