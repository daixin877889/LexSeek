# 专门应用

本文档汇总请求权基础方法在特定领域的专门应用。

---

## 一、不作为侵权四合一检视

详见：[buzuowei-qinquan.md](./buzuowei-qinquan.md)

传统过错侵权的四要件（加害行为、因果关系、不法性、过错）在不作为侵权中合并为一个核心判断：**是否有作为义务且违反了该义务**。

### 简化检视程式

```
绝对权受侵害 → 作为义务（违反即推定四要件合一） → 被告抗辩
```

### 适用场景
- 安全保障义务违反案件
- 医疗不作为侵权
- 监护人未尽义务案件

## 二、合同有效推定原则

详见：[youxiao-tuiding.md](./youxiao-tuiding.md)

合同效力检视遵循推定结构。

### 检视结构

```
成立 → 推定有效 → 效力质疑者举证
```

### 效力瑕疵分类

| 类型 | 内容 | 举证主体 |
|------|------|----------|
| 效力未发生抗辩 | 无行为能力、违反强制规定等 | 被告举证 |
| 效力消灭抗辩 | 撤销、解除等 | 被敌举证 |

## 三、法律漏洞类推填补

详见：[loudong-leitui.md](./loudong-leitui.md)

遍检所有请求权基础后无适用规范时，精确定位法律漏洞，以类推填补。

### 基本方法

在法律评价上相似的案件之间类比，将已适用的规范类推到未规定的案件。

### 执行步骤

1. 确认法律漏洞（遍检后无适用规范）
2. 寻找类似案件/规范
3. 论证类推合理性（法律评价相似）
4. 适用类推结果

## 四、公因式展开与体系构建

详见：[gongyingshi-zhankai.md](./gongyingshi-zhankai.md)

从法律适用视角构建民事规范体系。

### 三层操作

| 层级 | 范围 | 用途 |
|------|------|------|
| 宏观 | 整部法典 | 请求权基础体系化 |
| 中观 | 特定编章 | 规范序列整理 |
| 微观 | 单条规范 | 评注式分析 |

### 核心方法

针对每个请求权基础，层层引入辅助规范，将立法"压缩"的公因式还原为完整的规范图谱。
