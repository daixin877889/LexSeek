# 三层四步检视法

对任何一项请求权做完整检视，依次通过三道门槛：

## 检视结构

```
第一层：请求权已产生
    ├── 步骤1：积极要件是否满足 → 原告举证
    └── 步骤2：是否存在权利未发生抗辩 → 被告举证

第二层：请求权未消灭
    └── 步骤3：是否存在权利消灭抗辩 → 被敌举证

第三层：请求权可行使
    └── 步骤4：是否存在权利行使抗辩 → 被告举证
```

## 判停规则

每层不通过则后续不必检视：
- 步骤1有要件不满足 → 请求权未产生，检视终止
- 第二层通过但请求权已消灭 → 检视终止
- 第三层存在有效抗辩权 → 请求权不可行使

## 第一层详解

**步骤1：积极要件检视**

列出请求权基础的全部构成要件，逐一判断是否满足：
- 满足的：简述理由
- 不满足的：说明原因，标记检视中止
- 有争议的：展示正反理由

**步骤2：权利未发生抗辩**

检视是否存在阻碍请求权产生的抗辩事由：
- 无行为能力
- 违反强制规定
- 意思表示瑕疵等

## 第二层详解

**步骤3：权利消灭抗辩**

检视是否存在使已产生的请求权消灭的事由：
- 清偿
- 免除
- 抵销
- 混同
- 解除条件成就等

## 第三层详解

**步骤4：权利行使抗辩**

检视是否存在阻碍请求权行使的抗辩权：
- 时效抗辩
- 同时履行抗辩权
- 不安抗辩权
- 先诉抗辩权等

## 与举证分配的关系

| 步骤 | 内容 | 举证主体 |
|------|------|----------|
| 步骤1 | 积极要件 | 原告举证 |
| 步骤2 | 未发生抗辩 | 被告举证 |
| 步骤3 | 消灭抗辩 | 被敌举证 |
| 步骤4 | 行使抗辩 | 被敌举证 |

## 执行步骤

1. 确定请求权基础（主要规范）
2. 第一层检视：步骤1+步骤2
3. 第二层检视：步骤3
4. 第三层检视：步骤4
5. 结论：明确请求权在哪一层通过或未通过
