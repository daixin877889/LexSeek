# 鉴定体裁法

规范法律分析写作格式，强制展示推理过程。

## 写作结构

严格递进的四个步骤：

```
假设 → 大前提 → 涵摄 → 结论
```

### 步骤1：假设

以假设性问题为起点："甲得否依第X条向乙请求Y？"

若有多项请求权基础，每项独立假设。

### 步骤2：大前提

列出该请求权基础的全部构成要件：
- 包括辅助规范补充的要件
- 按三层四步结构排列

### 步骤3：涵摄

逐一将案件事实归入各构成要件：

| 情形 | 处理 |
|------|------|
| 满足要件 | 简述理由 |
| 不满足要件 | 说明原因，标记检视中止 |
| 有争议 | 展示正反理由 |

### 步骤4：结论

回答假设问题："甲得/不得依第X条向乙请求Y"

若有后续请求权基础需检视，继续步骤1。

## 与裁判体裁的区别

| 鉴定体裁 | 裁判体裁 |
|----------|----------|
| 先假设→再推理→后结论 | 先结论→再说理由 |
| 展示完整思维过程 | 结论+简要理由 |
| 不跳步，每要件必检视 | 可能跳过明显环节 |

裁判体裁的结论本身必须借助鉴定体裁得出。

## 执行步骤

1. 提出假设："甲得否依第X条向乙请求Y"
2. 列出大前提：全部构成要件
3. 逐项涵摄：事实归入要件
4. 得出结论：回答假设问题
