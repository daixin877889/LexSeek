# 证据和策略梳理方法论

## 简介

本skill提供民事诉讼证据分析和诉讼策略构建的系统方法论，用于：

- 梳理和评估证据材料
- 构建完整的证据链
- 识别证据缺口并制定补强策略
- 制定诉讼策略方向
- 预测庭审争议焦点

## 使用场景

**适用场景**：
- 律师整理案卷证据材料
- 分析证据效力和证明力
- 构建举证方案
- 制定诉讼策略
- 预判庭审争议焦点
- 判决书证据认定分析

**不适用场景**：
- 刑事案件证据分析（证明标准不同）
- 行政诉讼证据分析（举证规则不同）
- 仲裁程序（程序规则差异）

## 快速使用指南

### 最简流程（证据清单）

1. 审阅已有材料
2. 识别证据类型
3. 进行三性审查（关联性→合法性→真实性）
4. 评估证明力
5. 输出证据清单

### 完整流程（策略分析）

1. 判断案件情形（推演/混合/实证/判决分析）
2. 进行三性审查（按质证次序）
3. 评估证明力（单证+组合）
4. 识别待证事实
5. 分配举证责任（区分举证责任与举证必要）
6. 构建证据链（四步法）
7. 识别证据缺口
8. 制定诉讼策略（主线+辅线+突破口）

## 核心原则速查

| 原则 | 说明 |
|------|------|
| 三性缺一不可 | 任一性缺失，证据无证据能力 |
| 先能力后证明力 | 无证据能力的证据不得判断证明力 |
| 举证责任不可转换 | 举证责任法定分配，不可转移 |
| 举证必要可转移 | 证据达高度可能性时举证必要转移 |
| 高度盖然性75%+ | 民事证明标准，非排除合理怀疑 |
| 否认者不举证 | 否认不产生新事实 |
| 抗辩者举证 | 抗辩产生新事实 |

## 常见失败模式速查

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 非法证据判断证明力 | 无证据能力不得判断证明力 | 先判断证据能力 |
| 误用刑事标准 | 用排除合理怀疑判断民事证据 | 民事只需高度盖然性 |
| 举证责任混淆举证必要 | 误以为举证责任可转换 | 举证责任不变，举证必要可转移 |
| 偷拍偷录一律排除 | 手机录音通常合法 | 区分设备类型、侵害程度 |
| 私下承认当作自认 | 诉讼外自认不免证 | 区分诉讼内/诉讼外自认 |

使用流程：
```

2. 使用本skill进行证据分析
3. 反馈证据缺口，调整诉讼策略
```
