# 证据和策略梳理方法论

## 核心原则

证据和策略梳理以"证据→待证事实→法律要件→诉讼请求"为逻辑主线，构建证据分析和诉讼策略的系统框架。

分析方法遵循四个基本原则：

1. **三性审查原则**：每项证据须经过真实性、合法性、关联性三维度审查（详见 [references/zhengju-sanxing.md](./references/zhengju-sanxing.md)）

2. **证据能力先于证明力原则**：先判断证据能力（能否进门），再判断证明力大小（详见 [references/zhengju-nengli.md](./references/zhengju-nengli.md)）

3. **举证对应原则**：举证责任与实体法要件-抗辩结构对应，积极要件由原告举证（详见 [references/juzheng-fenpei.md](./references/juzheng-fenpei.md)）

4. **高度盖然性原则**：民事诉讼证明标准为高度盖然性（75%+），非排除合理怀疑（详见 [references/gaodu-gairanxing.md](./references/gaodu-gairanxing.md)）

## 关键要点

### 一、证据三性审查

详见 [references/zhengju-sanxing.md](./references/zhengju-sanxing.md)

- **真实性审查**：形式真实性（载体真实）+ 实质真实性（内容真实）
- **合法性审查**：主体合法 + 取得方式合法 + 程序合法 + 形式合法（四个维度）
- **关联性审查**：与待证事实的关联程度（直接关联/间接关联）
- **审查顺序**：关联性 → 合法性 → 真实性 → 证明力（详见 [references/zhizheng-cixu.md](./references/zhizheng-cixu.md)）

### 二、证据能力与证明力分离

详见 [references/zhengju-nengli.md](./references/zhengju-nengli.md)

- **证据能力**：能否进入诉讼？质的问题（有/无）
- **证明力**：能否证明待证事实？量的问题（强/弱）
- **两阶段审查**：先判断证据能力，再判断证明力

### 三、质证系统方法

详见 [references/zhizheng-yijian.md](./references/zhizheng-yijian.md)

- **质证次序**：关联性 → 合法性 → 真实性 → 证明力
- **质证意见六类型**：无异议、逾期举证异议、关联性异议、合法性异议、真实性异议、证明力异议
- **组合使用**：可根据具体情况组合发表多种异议

### 四、证明力评估

详见 [references/zhengmingli-pinggu.md](./references/zhengmingli-pinggu.md)

- **单证证明力**：强/一般/弱三档评估
- **组合证明力**：证据间的补强与印证关系
- **高度盖然性判断**：是否达到75%以上可能性

### 五、举证责任分配

详见 [references/juzheng-fenpei.md](./references/juzheng-fenpei.md)

- **举证责任（败诉风险）**：法定分配、不可转换
- **举证必要（举证压力）**：动态变化、可转移
- **否认与抗辩**：否认者不举证，抗辩者举证
- **举证必要转移**：证据达高度可能性（75%+）时转移至对方

### 六、证据链构建方法

详见 [references/zhengju-lian.md](./references/zhengju-lian.md)

- **四步法结构**：核心观点 → 证据组合 → 逻辑阐述 → 风险反驳
- **混合拼图模式**：已有证据与待补证据的结合论证
- **完整性检验**：验证证据链是否完整覆盖待证事实

### 七、推定运用

详见 [references/tuiding-yunyong.md](./references/tuiding-yunyong.md)

- **法律推定**：法律明文规定，强制适用，反驳需"推翻"
- **事实推定**：法官依经验法则推断，自由心证，反驳只需"反驳"
- **事实推定三要件**：基础事实真实可靠 + 高度盖然性联系 + 对方未有效反证

### 八、自认制度

详见 [references/ziren-zhidu.md](./references/ziren-zhidu.md)

- **诉讼内自认**：本案诉讼过程中向本案法官作出 + 对己不利 → 免证
- **诉讼外自认**：私下、另案承认 → 不免证，仅作证据参考

### 九、情形判断逻辑

详见 [references/qingxing-panduan.md](./references/qingxing-panduan.md)

- **情形一（推演模式）**：无证据 → 推演所需证据类型
- **情形二（混合模式）**：零散证据 → 整合已有与待补证据
- **情形三（实证模式）**：充分证据 → 全面审查构建证据链
- **情形四（判决分析模式）**：判决书 → 提取法院证据认定

### 十、取证指引

详见 [references/quzheng-zhiyin.md](./references/quzheng-zhiyin.md)

- **合法性原则**：不严重侵害权益、不违反禁止性规定
- **书证取证**：原件要求、公证要点、公私书证效力差异
- **电子数据取证**：七要素真实性判断、微信聊天记录取证要点
- **偷拍偷录判断**：设备来源、取证场所、取证目的三要素

### 十一、诉讼策略构建

详见 [references/susong-celue.md](./references/susong-celue.md)

- **核心策略定位**：主线、辅线、突破口
- **庭审焦点预测**：3-5个关键焦点
- **证据组织建议**：编排顺序、分组建议、可视化

## 使用流程

```
1. 情形判断 → 判断用户提供材料状况（推演/混合/实证/判决分析）
2. 证据清单 → 列出已有证据和建议补充证据
3. 三性审查 → 评估每项证据的可采性（按质证次序）
4. 证明力评估 → 判断证据的证明价值（单证+组合）
5. 待证事实识别 → 确定需要证明的关键事实
6. 举证分配 → 明确举证责任主体
7. 证据链构建 → 组织证据组合（四步法）
8. 缺口识别 → 发现证据不足之处
9. 策略制定 → 确定诉讼策略方向
```

## 边界说明

**不适用场景**：
- 刑事案件证据分析（证明标准不同）
- 行政诉讼证据分析（举证规则不同）
- 仲裁程序（程序规则差异）

**常见失败模式**：
- 证据形式要件与法律行为要件混淆
- 非法证据判断证明力
- 偷拍偷录合法性判断不精确
- 举证责任与举证必要混淆
- 误用刑事排除合理怀疑标准

## 与请求权基础模块的配合

本skill与请求权基础模块紧密配合：

1. **前置依赖**：证据分析服务于请求权检视中的要件证明
2. **举证对应**：请求权检视确定举证责任分配，证据分析执行举证
3. **策略联动**：请求权成立与否影响诉讼策略方向

配合流程：
```
请求权基础模块 → 确定要件结构和举证分配
     ↓
证据策略模块 → 证据分析和补强
     ↓
请求权检视 → 评估要件证明可能性
```
