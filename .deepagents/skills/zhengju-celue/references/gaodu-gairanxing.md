# 高度盖然性证明标准

民事诉讼证明标准：高度盖然性（75%+），而非排除合理怀疑（90%+）。

## 一、证明标准概念

证明标准是证据证明力需达到的程度才能认定事实为真。

## 二、民事与刑事证明标准的区别

| 标准 | 民事诉讼 | 刑事诉讼 |
|------|----------|----------|
| 名称 | 高度盖然性 | 排除合理怀疑 |
| 程度 | 75%以上可能性 | 90%以上可能性 |
| 说明 | 高度可能即可认定 | 排除一切合理怀疑 |

**关键区别**：民事证明标准低于刑事证明标准

## 三、高度盖然性标准

### 定义

证据证明力达到高度可能性（75%以上）即可认定事实为真。

### 适用范围

民事诉讼一般事实证明

### 判断方法

| 证明力程度 | 高度盖然性判断 | 结论 |
|------------|----------------|------|
| 90%+ | 超过高度盖然性标准 | 可认定事实为真 |
| 75%-90% | 达到高度盖然性标准 | 可认定事实为真 |
| 60%-75% | 未达高度盖然性标准 | 需补强证据 |
| 40%-60% | 真伪不明 | 举证责任方承担不利后果 |
| <40% | 不足以证明 | 举证责任方承担不利后果 |

## 四、真伪不明的处理

### 真伪不明定义

证明力在40%-60%之间，事实存在与否无法判断。

### 处理规则

真伪不明时，举证责任方承担不利后果：

```
真伪不明 → 举证责任方败诉
```

**原理**：举证之所在，败诉之所在。

## 五、倾向性鉴定意见的处理

### 问题

鉴定意见写"倾向性意见"（如"倾向认为签名真实"），能否定案？

### 判断

| 判断 | 说明 |
|------|------|
| 倾向性意见可能已达高度盖然性 | "倾向认为..."可能表示75%+可能性 |
| 结合其他证据判断 | 需综合判断是否满足标准 |
| 民事诉讼中可能已足够定案 | 不需达到排除合理怀疑 |

### 案例

```
鉴定意见："倾向认为签名真实"

判断：
• 倾向性意见可能已达高度盖然性（75%+）
• 结合其他证据判断是否满足标准
• 若达高度盖然性，可认定签名真实

结论：民事诉讼中倾向性意见可能已足够定案
```

## 六、典型案例

### 证据证明力80%可能性定案

```
证据证明力达80%可能性：

判断：
• 证明标准：民事诉讼高度盖然性（75%+）
• 80%可能性已超过高度盖然性标准

结论：可以认定事实为真
结果：不需达到排除合理怀疑（90%+）
```

### 真伪不明举证责任方败诉

```
证据证明力40-60%（真伪不明）：

判断：
• 真伪不明不满足高度盖然性标准（75%+）
• 举证责任方承担举证不力后果

结论：不能认定事实，举证责任方败诉
```

### 倾向性鉴定意见定案

```
鉴定意见写"倾向性意见"：

判断：
• 倾向性意见可能已达高度盖然性
• 结合其他证据判断是否满足75%+标准

结论：倾向性意见可能已足够定案
结果：民事诉讼中倾向性意见可认定事实
```

## 七、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 误用刑事标准 | 用排除合理怀疑标准判断民事证据 | 民事证据只需高度盖然性（75%+） |
| 倾向性意见一律否定 | 倾向性意见可能已达高度盖然性 | 结合其他证据判断 |
| 真伪不明当作举证不足 | 真伪不明时举证责任方承担不利后果 | 区分真伪不明与举证不足 |
| 要求100%证明 | 民事诉讼不需100%证明 | 高度盖然性即可 |

## 八、高度盖然性运用策略

### 主张事实方的策略

| 情形 | 策略 |
|------|------|
| 证据达75%+ | 主张已达高度盖然性标准，可认定事实 |
| 证据60%-75% | 补强证据至75%+ |
| 证据真伪不明 | 调整诉讼策略或和解 |

### 反驳事实方的策略

| 情形 | 策略 |
|------|------|
| 对方证据达75%+ | 提出反证使真伪不明 |
| 对方证据倾向性意见 | 结合其他证据反驳 |
| 真伪不明 | 主张举证责任方承担不利后果 |

## 九、高度盖然性输出格式

```
证明力评估：

证据组合：[证据名称]
证明力程度：[90%/80%/70%/60%/50%等]

高度盖然性判断：
• 是否达到75%以上可能性：[是/否]
• 说明：[判断依据]

结论：[可认定事实为真/需补强证据/举证责任方承担不利后果]
```

## 十、与证据链、举证责任的衔接

**与证据链构建的衔接**：
- 证据链构建 → 组合证明力评估
- 组合证明力达75%+ → 可认定事实为真

**与举证责任的衔接**：
- 真伪不明（40%-60%）→ 举证责任方承担不利后果
- 举证必要转移：证据达75%+ → 举证必要转移至对方
