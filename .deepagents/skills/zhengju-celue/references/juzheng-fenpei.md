# 举证责任分配法

确定每项待证事实的举证责任主体。

## 一、核心原则

**举证责任不可转换原则**：举证责任是法定的，来源于法律的强制性规定，不发生举证责任转换的问题。败诉风险始终由法定承担者负最终责任。

**举证之所在，败诉之所在**：承担举证责任的一方，若举证不能则败诉。举证责任是诉讼的核心风险所在。

## 二、举证责任基本概念

### 谁主张谁举证

民事诉讼法的基本原则：主张事实存在的一方承担举证责任。

### 举证责任的性质

| 类型 | 说明 |
|------|------|
| 行为责任 | 提供证据的责任 |
| 结果责任 | 举证不能时承担不利后果的责任 |

### 举证责任与举证必要的区分

| 概念 | 性质 | 特点 | 适用 |
|------|------|------|------|
| 举证责任 | 法定分配、不可转换 | 始终不变 | 判断最终败诉风险 |
| 举证必要 | 动态变化、可转移 | 可因法官心证转移 | 判断当前举证压力 |

**举证必要转移规则**：若证据已达高度可能性（75%+），举证必要转移至对方。

## 三、举证责任分配规则（法律要件分类说）

### 基本分配规则

与请求权基础的三层四步检视相对应：

| 层级 | 内容 | 举证主体 |
|------|------|----------|
| 第一层 | 请求权产生要件 | 原告举证 |
| 第二层 | 权利消灭抗辩 | 被告举证 |
| 第三层 | 权利行使抗辩 | 被告举证 |

### 详细分配表

| 类型 | 内容 | 举证主体 | 法律依据 |
|------|------|----------|----------|
| 积极要件 | 请求权产生的构成要件 | 原告 | 主张者举证 |
| 权利发生抗辩 | 阻碍请求权产生的事由 | 被告 | 抗辩者举证 |
| 权利消灭抗辩 | 使请求权消灭的事由 | 被告 | 抗辩者举证 |
| 权利行使抗辩 | 阻碍请求权行使的事由 | 被告 | 抗辩者举证 |
| 反抗辩 | 抗辩的抗辩 | 原告 | 主张者举证 |

### 要件事实分类

| 分类 | 特征 | 举证主体 |
|------|------|----------|
| 权利发生要件 | 法律关系成立的要件事实 | 主张权利存在者 |
| 权利妨碍要件 | 权利成立时已存在障碍的事实 | 主张权利变更/消灭者 |
| 权利消灭要件 | 权利成立后消灭的事实 | 主张权利变更/消灭者 |
| 权利受制要件 | 权利行使受限的事实 | 主张权利变更/消灭者 |

## 四、否认与抗辩识别

**关键判断**：看是否产生新事实

| 类型 | 定义 | 特点 | 举证责任 |
|------|------|------|----------|
| 否认 | 否定对方事实主张 | 不产生新事实 | 否认者不举证 |
| 抗辩 | 承认对方事实+提出新事实阻却 | 产生新事实 | 抗辩者举证 |

**否认的类型**：
- 直接否认：直接说"不是"（如"我没借过钱"）
- 间接否认：说对方证据不真实（如"借条是伪造的"）

**抗辩的类型**：
- 权利消灭抗辩：主张权利消灭（如"已还款"）
- 权利妨碍抗辩：主张权利成立有障碍（如"合同无效"）
- 权利受制抗辩：主张权利行使受限（如"已过诉讼时效"）

### 典型案例

| 被告主张 | 类型 | 举证责任 |
|----------|------|----------|
| "没借钱" | 否认 | 被告不举证 |
| "借条是伪造的" | 间接否认 | 被告不举证 |
| "借了但已还款" | 抗辩 | 被告举证还款事实 |
| "借款无效（欺诈）" | 抗辩 | 被告举证欺诈事实 |

## 五、常见案件的举证分配

### 合同纠纷

| 待证事实 | 举证主体 |
|----------|----------|
| 合同关系成立 | 原告 |
| 合同内容 | 原告 |
| 已履行合同义务 | 主张履行方 |
| 合同无效 | 主张无效方 |
| 合同可撤销 | 主张撤销方 |
| 合同已解除 | 主张解除方 |
| 违约事实 | 主张违约方 |
| 免责事由 | 被告 |

### 侵权纠纷

| 待证事实 | 举证主体 |
|----------|----------|
| 侵权行为 | 原告 |
| 损害后果 | 原告 |
| 因果关系 | 原告 |
| 过错（一般侵权） | 原告 |
| 无过错（无过错责任） | 被告（不适用） |
| 免责事由 | 被告 |

### 所有权纠纷

| 待证事实 | 举证主体 |
|----------|----------|
| 所有权归属 | 主张方 |
| 占有权源 | 占有人 |
| 无权占有 | 原告（通过被告证明有权反驳） |

## 六、举证责任倒置

### 法定倒置情形

| 案件类型 | 倒置内容 | 法律依据 |
|----------|----------|----------|
| 环境污染 | 因果关系由被告举证 | 民诉解释 |
| 共同危险行为 | 因果关系由被告举证 | 民诉解释 |
| 医疗纠纷 | 过错及因果关系由被告举证 | 侵权责任编 |
| 产品责任 | 缺陷与损害因果关系由被告举证 | 产品质量法 |
| 高危作业 | 无过错责任+免责事由由被告举证 | 侵权责任编 |

### 倒置后的分配

```
原: 原告举证A → 原告举证B → 原告举证C → 原告举证因果关系
倒置后: 原告举证A → 原告举证B → 原告举证C → 被告举证无因果关系
```

## 七、鉴定申请主体的判断

### 判断框架

举证责任不变，但举证必要可转移：

| 情形 | 举证必要归属 | 说明 |
|------|--------------|------|
| 签名肉眼比对明显存疑 | 原告 | 法官心证倾向签名不真实，原告需申请鉴定 |
| 签名肉眼比对无明显疑点 | 被告 | 已达高度可能性，举证必要转移至被告 |
| 借条真实性已达高度可能性 | 被告 | 被告需申请鉴定推翻 |

### 典型案例

**笔迹鉴定申请主体判断**：
- 原告提交借条，被告否认签名真实性
- 举证责任：原告主张借款，举证责任不变（原告承担最终败诉风险）
- 举证必要：法官肉眼比对签名存疑 → 举证必要转移至原告
- 结论：原告申请鉴定（举证必要转移），但举证责任仍由原告承担
- 若原告不申请鉴定 → 签名真实性未证明 → 原告败诉

## 八、举证不能的后果

### 原告举证不能

- 请求权要件无法证明 → 请求权不成立 → 驳回诉讼请求

### 被告举证不能

- 抗辩事由无法证明 → 抗辩不成立 → 承担举证不能后果

### 真伪不明的处理

- 证明力40-60%（真伪不明）→ 不满足高度盖然性标准 → 举证责任方承担不利后果

## 九、举证分配输出格式

```

• [相关法条]

• [需要证明的具体内容]

举证必要判断：
• 当前证据状态：[已达高度可能性/不足]
• 举证必要归属：[原告/被告]

• [举证方向建议]
```

## 十、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 举证责任与举证必要混淆 | 误以为举证责任可转换 | 举证责任不变，举证必要可转移 |
| 笔迹鉴定申请主体一刀切 | 一概由原告或一概由被告申请 | 视法官心证倾向判断举证必要转移 |
| 否认当作抗辩 | "没借钱"和"已还款"举证责任不同 | 区分是否产生新事实 |
| 间接否认当作抗辩 | "借条伪造"是间接否认，不是抗辩 | 间接否认不产生新事实 |
| 抗辩不举证 | 抗辩产生新事实，抗辩者必须举证 | 抗辩者举证新事实 |

## 十一、与请求权基础模块的衔接

举证分配与请求权基础检视紧密衔接：

```
请求权基础检视          举证分配
─────────────────────────────────
步骤1 积极要件    →    原告举证
步骤2 未发生抗辩  →    被告举证
步骤3 消灭抗辩    →    被告举证
步骤4 行使抗辩    →    被告举证
反抗辩          →    原告举证
```

使用流程：
1. 使用请求权基础模块确定要件结构
2. 识别每个要件的待证事实
3. 使用本方法确定举证主体
4. 结合证据分析评估举证可能性
