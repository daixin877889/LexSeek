# 取证指引法

指导各类证据的合规取证方法和标准。

## 一、取证基本原则

### 合法性原则

- 不得采用严重侵害他人合法权益的方式
- 不得采用违反法律禁止性规定的方法
- 不得采用严重违背公序良俗的手段

**排除标准**：
| 排除要件 | 说明 |
|----------|------|
| 严重侵害合法权益 | 严重侵害隐私权、住宅安宁权等 |
| 违反法律禁止性规定 | 使用间谍器材、闯入住宅等 |
| 严重违背公序良俗 | 取证行为严重违背社会公德 |

### 完整性原则

- 保全证据的完整内容
- 保全证据的形成背景
- 保全证据的关联信息

### 及时性原则

- 发现证据及时固定
- 易失证据及时保全
- 必要时申请法院保全

## 二、书证取证指引

### 原件要求

**原则**：书证应当提交原件

**例外可提交复制件的情形**：
- 原件已不存在且有正当理由
- 原件由对方控制且拒不提供
- 原件因客观原因无法调取

**复制件的补强**：
| 复制件类型 | 补强方式 |
|------------|----------|
| 复印件 | 与原件核对一致 |
| 照片 | 结合原件、其他证据印证 |
| 扫描件 | 保存原始电子文件 |

### 公证要点

**应当公证的情形**：
- 重要书证可能灭失
- 证据可能在域外使用
- 电子证据需增强证明力

**公证注意事项**：
- 公证书应完整记录取证过程
- 附原件或复制件
- 公证员签字、公证处盖章

### 公文书证与私文书证效力差异

| 类型 | 效力规则 | 质证策略 |
|------|----------|----------|
| 公文书证 | 形式合规即推定内容真实 | 推翻推定：证明制作程序违法或内容虚假 |
| 私文书证 | 形式真实≠内容真实 | 直接质疑内容真实性 |

**公文书证形式合规条件**：
- 具有职权机构印章
- 格式符合该机构行文规范
- 内容符合该机构职权范围

### 单位证明材料三要件

- 单位印章
- 负责人签名
- 经办人签名

**三要件缺失后果**：证据资格不被认可

### 书证取证清单

```

取证方式：□原件 □复制件 □公证
取证时间：_______________
取证地点：_______________
取证经过：_______________
保存方式：_______________
备注：___________________
```

## 三、电子数据取证指引

### 电子数据真实性判断七要素

1. **系统环境完整性**：生成、存储、传输的硬件软件环境是否完整可靠
2. **系统运行状态**：是否正常运行，有无故障、崩溃、被攻击
3. **监测核查手段**：有无哈希校验、日志记录、备份机制
4. **保存传输提取完整性**：数据是否完整，有无截取、删减、篡改
5. **形成存储场景正常性**：是否在正常往来活动中形成
6. **操作主体适当性**：保存、传输、提取主体是否适当
7. **其他影响因素**：时间戳可信度、加密强度

### 电子数据类型与取证要点

| 类型 | 示例 | 取证要点 |
|------|------|----------|
| 通信数据 | 微信、QQ、短信 | 身份确认、完整性 |
| 交易数据 | 支付记录、订单 | 平台存证、时间戳 |
| 文档数据 | 电子合同、邮件 | 原始文件、发送记录 |
| 社交数据 | 朋友圈、微博 | 公证、截取完整 |

### 微信聊天记录取证

**取证步骤**：
1. 确认对方身份（微信名、微信号、头像）
2. 完整截取聊天记录（按时间顺序）
3. 保存原始载体（手机）
4. 必要时进行公证

**关键标准**：
- 必须展示完整的聊天记录
- 不得删减、剪辑
- 需证明对方身份
- 建议进行公证增强证明力

**常见问题**：
| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 仅提供截图 | 真实性存疑 | 公证或出示原始载体 |
| 删减部分记录 | 证明力降低 | 完整展示 |
| 对方身份不明 | 关联性存疑 | 引导对方确认身份 |

### 电子邮件取证

**取证步骤**：
1. 登录原始邮箱账号
2. 展示邮件完整内容（发件人、收件人、时间、正文、附件）
3. 下载保存邮件原始文件
4. 必要时公证

**关键标准**：
- 保存邮件原始格式（.eml）
- 保全邮件头信息
- 确认发件人、收件人身份

### 电子数据存证

| 存证方式 | 说明 | 证明力 |
|----------|------|--------|
| 可信时间戳 | 第三方可信时间戳服务中心提供 | 需法院认定 |
| 区块链存证 | 利用区块链不可篡改特性 | 部分互联网法院已认可 |
| 公证存证 | 公证处对电子数据进行公证 | 最强 |

**特殊规则**：电子数据的打印件或其他可显示输出件视为原件

## 四、证人证言取证指引

### 证人资格

**可以作证的人**：
- 知道案件情况的人
- 具有辨别是非能力
- 能够正确表达

### 证人证言真实性判断十要点

**证人自身角度（前6点）**：
1. 利害关系：证人与案件有无利害关系
2. 思顾虑：证人陈述时有无思想顾虑
3. 感知来源：亲历目击？听闻转述？
4. 认知条件：距离远近、光线好坏、听力视力
5. 言行举止：庭上从容镇定？紧张慌乱？
6. 品德：诚信可靠？有无不良记录？

**证言本身角度（后4点）**：
7. 常理逻辑：是否符合常理与逻辑
8. 前后一致：自身前后是否矛盾翻证
9. 与其他证据吻合：是否与其他证据一致
10. 出庭作证：有无出庭接受质询

### 证人证言获取

**取证方式**：
1. 书面证言
2. 录音录像
3. 申请证人出庭

**书面证言要点**：
- 证人身份信息
- 证人与当事人关系
- 作证事项的时间、地点、经过
- 证人签字、日期

**申请证人出庭**：
- 在举证期限届满前申请
- 说明证人基本情况
- 说明证人对案件事实的了解情况

### 证人证言注意事项

| 注意点 | 说明 |
|--------|------|
| 利害关系 | 与当事人有利害关系的证人证言证明力较低 |
| 出庭要求 | 证人应当出庭作证，接受质询，书面证言证明力弱 |
| 证言内容 | 应当是证人的亲身感知，非道听途说 |
| 证言形式 | 书面证言需要证人签字确认 |
| 证人保护 | 证人作证后应退庭，不得旁听庭审 |

### 循环作证陷阱

共同被告互相为对方作证：利害关系存在利益关联，相当于自我作证，证言不被采纳。

## 五、视听资料取证指引

### 录音取证合法性判断

**合法性要求**：
- 在公共场合录音
- 不侵害他人隐私
- 不采用窃听等非法手段

**判断三要素**：
| 要素 | 说明 |
|------|------|
| 设备来源 | 普通手机（合法）vs间谍器材（非法） |
| 取证场所 | 公共场所（合法）vs闯入住宅（非法） |
| 取证目的 | 维权（合法）vs恶意（可能非法） |

**取证要点**：
| 要点 | 说明 |
|------|------|
| 原始载体 | 保留录音的原始设备 |
| 完整性 | 不得剪辑、删减 |
| 身份确认 | 引导对方表明身份 |
| 内容确认 | 引导对方确认关键事实 |

### 单一视听资料可采性判断

| 判断 | 标准 |
|------|------|
| 无疑点→可单独定案 | 完整、清晰、无剪辑痕迹、对方承认真实性 |
| 存疑点→需补强证据 | 对方否认、可能剪辑、无法鉴定 |

### 录像取证

**取证要点**：
- 保留原始录像设备
- 确保录像完整性
- 录像中应有时间标记
- 必要时公证

### 监控视频取证

**取证方式**：
- 申请法院调取
- 与监控管理人协商获取
- 注意保存期限（通常30天）

**取证要点**：
- 确认监控时间与实际时间
- 保全完整视频
- 注意视频格式和清晰度

## 六、鉴定意见取证指引

### 鉴定程序启动

**申请要点**：
- 在举证期限内申请
- 明确鉴定事项
- 提供鉴定材料

### 鉴定意见审查八要素

1. **检材样本审查**：是否具备鉴定条件、是否真实完整、有无污染耗损
2. **原理方法审查**：原理是否科学、方法是否适用
3. **仪器设备审查**：是否适用、是否先进可靠
4. **鉴定时机审查**：是否适当、是否需要现场勘验
5. **论证逻辑审查**：是否严谨、逻辑是否清晰
6. **合理性审查**：是否合理、是否符合常识常理
7. **一致性审查**：与已知事实、其他证据是否一致
8. **鉴定人审查**：是否诚信、有无应回避未回避

### 鉴定程序违法程度判断

| 违法程度 | 具体情形 | 后果 |
|----------|----------|------|
| 严重违法 | 鉴定人应回避未回避、人员组成违规、未现场勘验 | 可申请重新鉴定 |
| 轻微违法 | 签名瑕疵、超时限出具 | 可补正，不影响证据能力 |

### 文书形成时间鉴定注意事项

除圆珠笔书写外，文书形成时间鉴定无认可方法，盲目委托可能导致鉴定无效。

## 七、特殊取证方式

### 申请法院调查令

**适用情形**：
- 证据由国家有关部门保存
- 证据涉及商业秘密、个人隐私
- 当事人因客观原因不能自行收集

**申请材料**：
- 调查令申请书
- 调查证据的名称、内容
- 申请理由
- 调查对象的名称、地址

### 申请书证提出命令

**申请四要素**：
1. 书证名称或内容明确
2. 待证事实及其重要性
3. 对方控制书证的根据
4. 应提交的理由（法定情形）

**申请后果**：对方无正当理由拒不提交 → 法院认定申请人主张的书证内容为真实

### 申请法院保全

**适用情形**：
- 证据可能灭失
- 证据以后难以取得

**申请要点**：
- 在举证期限内申请
- 说明保全理由
- 提供担保

## 八、取证风险提示

### 非法取证风险

| 风险行为 | 后果 | 替代方案 |
|----------|------|----------|
| 窃听、偷拍（间谍器材） | 证据排除+可能触犯刑法 | 公开场合录音录像 |
| 暴力、威胁取证 | 证据排除 | 合法协商 |
| 侵犯隐私权（严重） | 证据排除 | 申请法院调取 |
| 伪造证据 | 法律制裁 | 合法取证 |

### 证据保全风险

| 风险行为 | 后果 | 预防措施 |
|----------|------|----------|
| 电子证据未及时保全 | 数据丢失 | 及时公证或时间戳 |
| 书证复制件未核对 | 证明力弱 | 及时与原件核对 |
| 证人证言未固定 | 证人翻供 | 书面证言+录音录像 |

## 九、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 手机录音一律排除 | 手机录音非间谍器材，通常合法 | 区分设备类型判断 |
| 间谍器材取证 | 不仅证据排除，还可能触犯刑法 | 使用普通设备 |
| 经公证书面证言视为出庭证言 | 公证仅证明形式真实，证人仍须出庭 | 申请证人出庭 |
| 鉴定材料未经质证盲目鉴定 | 检材未经质证程序违法 | 检材先质证后鉴定 |
| 文书形成时间盲目委托 | 无认可方法导致鉴定无效 | 仅圆珠笔书写可鉴定 |
| 书证提出命令滥用 | 需书证确实存在且对方控制 | 确认控制根据 |
