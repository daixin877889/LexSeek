# 诉讼策略构建法

制定诉讼策略和庭审预案。

## 一、策略构建框架

### 策略分析维度

```
            ┌─────────────────┐
            │   核心策略定位   │
            └────────┬────────┘
                     │
        ┌────────────┼────────────┐
        ▼            ▼            ▼
   ┌─────────┐  ┌─────────┐  ┌─────────┐
   │进攻主线 │  │防守辅线 │  │突破口  │
   └─────────┘  └─────────┘  └─────────┘
        │            │            │
        └────────────┼────────────┘
                     │
            ┌────────┴────────┐
            ▼                 ▼
     ┌─────────────┐   ┌─────────────┐
     │庭审焦点预测 │   │证据组织建议 │
     └─────────────┘   └─────────────┘
```

### 策略三要素

| 要素 | 含义 | 分析重点 |
|------|------|----------|
| 主线 | 主要进攻方向 | 最有把握的法律主张 |
| 辅线 | 辅助防守方向 | 次要主张和防御点 |
| 窠破口 | 决胜关键点 | 证据最强、对方最弱处 |

## 二、核心策略确定

### 策略定位方法

**步骤1：法律要件分析**
- 分析请求权构成要件
- 识别已证明和待证明的要件
- 确定举证薄弱环节

**步骤2：证据强弱分析**
- 我方证据优势
- 对方证据优势
- 争议焦点

**步骤3：策略选择**

| 证据情况 | 策略方向 | 具体策略 |
|----------|----------|----------|
| 我方证据充分 | 进攻型 | 主打证据优势，证明要件 |
| 我方证据不足 | 防守型 | 主张举证责任分配，要求对方举证 |
| 证据相当 | 平衡型 | 主打法律解释，争取法官心证 |
| 存在瑕疵 | 补强型 | 先补强证据，再确定主攻方向 |

### 常见策略方向

**程序违规策略**：
- 适格主体问题
- 管辖权问题
- 时效问题
- 程序瑕疵

**合同解释策略**：
- 文义解释
- 体系解释
- 目的解释
- 习惯解释

**证据效力策略**：
- 攻击对方证据三性
- 质疑对方证据证明力
- 提出反证

**事实认定策略**：
- 主张法律推定
- 适用举证责任倒置
- 主张表见证明

### 举证必要与举证责任运用

**举证责任不变，举证必要可转移**：

| 情形 | 策略运用 |
|------|----------|
| 我方举证责任，证据已达高度可能性 | 主张举证必要转移至对方 |
| 对方举证责任，证据不足 | 要求对方举证，主张对方举证不能败诉 |
| 真伪不明 | 主张举证责任方承担不利后果 |

### 推定运用策略

| 推定类型 | 运用策略 |
|----------|----------|
| 法律推定 | 强制适用，对方需推翻基础事实或推定逻辑 |
| 事实推定 | 申请法官依经验法则推定，对方只需反驳（使真伪不明） |

**反驳难度差异**：
| 反驳类型 | 标准 |
|----------|------|
| 推翻（法律推定） | 证明相反事实成立 |
| 反驳（事实推定） | 使推定事实真伪不明 |

### 自认运用策略

| 自认类型 | 策略运用 |
|----------|----------|
| 诉讼内自认 | 直接免证，无需举证证明自认内容 |
| 诉讼外自认 | 仅作证据参考，仍需举证 |

**自认风险提示**：庭审中承认不利事实产生免证效力，谨慎表态。

## 三、庭审焦点预测

### 焦点预测方法

**步骤1：识别争议焦点**
- 双方主张对立的事实
- 双方对法律适用有分歧
- 证据存在矛盾

**步骤2：筛选关键焦点**
- 对裁判结果有决定性影响
- 双方争议激烈
- 法官可能重点关注

**步骤3：预测法官关注点**

| 法官关注点 | 预测依据 |
|------------|----------|
| 证据真实性 | 原件/复制件问题 |
| 法律适用 | 法律解释争议 |
| 责任划分 | 过错程度、原因力 |
| 金额计算 | 计算依据、标准 |

### 焦点预测输出格式

```
庭审焦点预测：

1. [焦点一名称]
   我方立场：...
   对方可能立场：...
   法官可能关注：...
   应对准备：...

2. [焦点二名称]
   ...

3. [焦点三名称]
   ...
```

### 预判数量控制

- 重点预测3-5个关键焦点
- 窝破口应具有决定性
- 避免列举无关痛痒的细节

## 四、证据组织建议

### 编排顺序

| 编排方式 | 适用情形 | 说明 |
|----------|----------|------|
| 时间顺序编排 | 事实发展脉络清晰 | 按证据形成时间排列，便于法官理解案情 |
| 逻辑顺序编排 | 复杂案件 | 按待证事实分组，按证明逻辑排列 |
| 争议焦点顺序编排 | 争议明确的案件 | 每组证据直接对应焦点 |

### 证据分组建议

```
证据册编排建议：

第一组：基础事实证据
├── 证据1：[证据名称]
├── 证据2：[证据名称]
└── ...

第二组：履行情况证据
├── 证据3：[证据名称]
├── 证据4：[证据名称]
└── ...

第三组：违约/侵权证据
├── 证据5：[证据名称]
└── ...

第四组：损失证据
├── 证据6：[证据名称]
└── ...
```

### 可视化建议

| 图表类型 | 适用情形 | 说明 |
|----------|----------|------|
| 时间轴图表 | 事件发展脉络清晰 | 展示事件发展脉络，标注关键时间节点，对应相关证据 |
| 关系图 | 当事人关系复杂 | 展示当事人关系、交易结构、证据关联 |
| 资金流向图 | 涉及款项往来 | 展示款项往来，标注争议金额，对应支付凭证 |

### 证人、鉴定申请建议

**申请证人出庭**：
- 证人证言对案件有重要影响
- 证人能够证明关键事实
- 需要证人接受质询

**申请鉴定**：
- 争议事项需要专业判断
- 符合鉴定条件
- 在举证期限内提出

**申请调查令**：
- 证据由第三方保管
- 当事人无法自行调取
- 证据对案件有重要影响

**申请书证提出命令**：
- 书证在对方控制
- 对方拒不提交
- 书证对待证事实重要

## 五、高度盖然性运用

### 证明标准判断

| 证明力程度 | 高度盖然性判断 | 策略建议 |
|------------|----------------|----------|
| 90%+ | 超过标准 | 可主张事实认定 |
| 75%-90% | 达到标准 | 可主张事实认定 |
| 60%-75% | 未达标准 | 需补强证据 |
| <60% | 真伪不明 | 主张举证责任方败诉 |

### 倾向性意见处理

鉴定意见写"倾向性意见"：
- 可能已达高度盖然性
- 结合其他证据判断是否满足75%+标准
- 民事诉讼中倾向性意见可能已足够定案

## 七、策略调整原则

### 证据变化时的调整

| 情况 | 策略调整 |
|------|----------|
| 发现新证据 | 评估对策略的影响，调整主攻方向 |
| 证据被对方否定 | 准备补强证据，或调整举证策略 |
| 对方出示反证 | 分析反证效力，调整应对方案 |

### 庭审时的调整

| 情况 | 策略调整 |
|------|----------|
| 法官关注点变化 | 调整陈述重点 |
| 对方庭审策略变化 | 调整应对方案 |
| 庭审中出现新情况 | 灵活调整，申请补充证据 |

### 和解时机判断

**考虑和解的情形**：
- 证据存在重大缺陷
- 法律风险较高
- 诉讼成本过高
- 判决执行困难

## 八、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 策略方向错误 | 未分析证据强弱选择错误方向 | 先分析证据强弱再选择策略 |
| 焦点预测过多 | 列举过多无关焦点 | 重点预测3-5个关键焦点 |
| 证据组织混乱 | 证据编排无逻辑 | 按时间/逻辑/焦点顺序编排 |
| 忽视举证必要转移 | 未利用举证必要转移策略 | 区分举证责任与举证必要 |
| 自认风险忽视 | 庭审中随意承认不利事实 | 谨慎表态，避免产生免证效力 |
| 倾向性意见一律否定 | 未判断是否达高度盖然性 | 结合其他证据判断是否满足75%+ |

## 九、实务技巧

### 开庭陈述准备

- 简明扼要陈述案情
- 突出核心主张
- 引导法官关注焦点

### 质证准备

- 按质证次序准备意见：关联性→合法性→真实性→证明力
- 准备质证意见六类型清单
- 预判对方质证意见

### 最后陈述准备

- 总结我方证据优势
- 强调对方证据缺陷
- 提出裁判建议
