# 推定运用框架

运用法律推定和事实推定减轻举证负担的方法。

## 一、推定的概念

推定是根据已知事实推断未知事实的方法，可减轻举证负担。

**核心作用**：免除主张方对推定事实的直接举证，转移举证压力至对方。

## 二、推定类型区分

### 法律推定

| 特点 | 说明 |
|------|------|
| 定义 | 法律明文规定的推定 |
| 依据 | 必须有法律条文明确规定 |
| 适用方式 | 强制适用，法官必须适用 |
| 反驳难度 | 较难——需推翻基础事实或推翻推定逻辑 |
| 反驳标准 | 足以推翻（证明相反事实成立） |

**典型示例**：
- 占有动产推定所有权（《民法典》规定）
- 公文书证形式合规推定内容真实

### 事实推定

| 特点 | 说明 |
|------|------|
| 定义 | 法官依已知事实和经验法则推断 |
| 依据 | 无需法律条文，依经验法则 |
| 适用方式 | 自由心证，法官可决定是否适用 |
| 反驳难度 | 较易——只需使推定事实真伪不明 |
| 反驳标准 | 足以反驳（使推定事实真伪不明） |

**典型示例**：
- 家中摔倒推定有安全隐患
- 多年不主张权利推定已放弃

## 三、事实推定三要件

运用事实推定必须满足三个要件：

### 要件1：基础事实真实可靠

- 基础事实必须有证据证明
- 基础事实是推定的起点
- 基础事实未证明则无法推定

### 要件2：高度盖然性联系

- 基础事实与推定事实之间有高度盖然性联系
- 高度盖然性 = 75%以上可能性
- 该联系基于经验法则（众所周知、反复验证）

### 要件3：对方未有效反证

- 对方没有提出反证
- 或反证不成立（反证不能使推定事实真伪不明）
- 反证标准：足以反驳（使推定事实真伪不明）

**三要件同时满足才能认定推定事实成立。**

## 四、经验法则运用

### 经验法则的概念

经验法则是指人们从生活经验中总结和归纳的关于事物因果关系或事物状态的知识。

**运用要求**：
- 必须是众所周知、反复验证的事实
- 不能用个人经验代替司法经验法则

### 经验法则运用步骤

1. 确认基础事实已证明
2. 判断高度盖然性联系
3. 检验对方是否提出有效反证
4. 认定推定事实

## 五、推定运用策略

### 主张推定方的策略

| 情形 | 策略 |
|------|------|
| 有法律推定依据 | 主张法律推定强制适用 |
| 无法律推定依据但符合经验法则 | 申请法官依经验法则事实推定 |
| 基础事实已有证据 | 主张基础事实已证明，推定成立 |

### 反驳推定方的策略

| 推定类型 | 反驳策略 |
|----------|----------|
| 法律推定 | 推翻基础事实（证明基础事实不成立）或推翻推定逻辑（证明推定关系不成立） |
| 事实推定 | 反驳（使推定事实真伪不明），无需证明相反事实成立 |

## 六、典型案例

### 法律推定案例：占有动产推定所有权

```
问题：谁占有的动产推定谁有所有权？

判断：
• 类型：《民法典》规定——法律推定
• 适用：强制适用
• 反驳：推翻基础事实（证明占有不成立）或推翻推定逻辑（证明占有不等于所有权）

结论：法律推定强制适用，反驳需"推翻"标准
```

### 事实推定成功案例：家中摔倒推定安全隐患

```
问题：原告在被告家中摔倒，能否推定被告有安全隐患？

要件检验：
• 要件1：原告摔倒事实有证据证明（医院记录） ✓
• 要件2：家中摔倒通常因安全隐患（高度盖然性） ✓
• 要件3：被告无反证（未证明地面干燥） ✓

结论：三要件满足，推定成立
```

### 事实推定失败案例：深夜外出推定出轨

```
问题：被告深夜外出，能否推定出轨？

要件检验：
• 要件1：深夜外出有证据证明 ✓
• 要件2：深夜外出与出轨无高度盖然性联系 ✗（原因很多）

结论：要件2不满足，推定失败
```

### 反证成立案例：多年不主张借款推定已还款

```
问题：多年不主张借款，能否推定已还款？

要件检验：
• 要件1：多年不催收有证据证明 ✓
• 要件2：不催收通常已还款（高度盖然性） ✓
• 要件3：原告反证（关系好不催收不罕见），使推定真伪不明 ✓

结论：要件3反证成立，推定失败
```

## 七、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 混淆推翻与反驳 | 推翻需证明相反事实成立，反驳只需使真伪不明 | 区分反驳难度 |
| 事实推定当作法律推定 | 无法律条文依据不能强制适用 | 区分是否有法律条文 |
| 用个人经验代替经验法则 | 经验法则需众所周知、反复验证 | 论证经验法则的普遍性 |
| 高度盖然性判断不严谨 | 仅凭直觉判断 | 论证高度盖然性是否存在（75%+） |
| 反证标准混淆 | 反证只需使真伪不明，无需证明相反事实 | 区分反驳与推翻标准 |

## 八、推定运用输出格式

```
推定主张：[推定事实名称]

推定类型：[法律推定/事实推定]
法律依据：[如有法律条文]

基础事实：[基础事实内容]
基础事实证据：[证据名称]

高度盖然性分析：[基础事实与推定事实的联系]

对方反证：[如有反证，说明内容]
反证效力：[是否使推定事实真伪不明]

推定结论：[推定成立/推定失败]
```

## 九、与举证责任的衔接

推定成立后举证必要转移：

```
推定前：主张方举证推定事实
推定后：举证必要转移至对方（对方需反驳推定）
举证责任：仍由主张方承担最终败诉风险（不变）
```
