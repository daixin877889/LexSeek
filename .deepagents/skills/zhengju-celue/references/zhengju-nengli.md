# 证据能力先于证明力原则

区分证据能力（能否进门）与证明力（能否定案）的两阶段审查原则。

## 一、核心原则

**证明力大小以证据能力为前提。没有证据能力的证据，不得判断其证明力。**

## 二、证据能力与证明力的区分

| 概念 | 问题 | 特点 | 判断标准 |
|------|------|------|----------|
| 证据能力 | 能否进入诉讼？ | 质的问题（有/无），无大小之别 | 三性是否同时满足 |
| 证明力 | 能否证明待证事实？ | 量的问题（强/弱），有大小之别 | 真实程度、关联强度 |

## 三、两阶段审查顺序

**顺序不可颠倒**：

### 第一阶段：判断证据能力（能否进门）

- **问题**：证据能否进入诉讼？
- **标准**：证据三性（真实性、合法性、关联性）同时满足
- **结论**：有证据能力（能进门）或无证据能力（不能进门）
- **特点**：质的问题（有/无），无大小之别

### 第二阶段：判断证明力（能否定案）

- **问题**：证据能否证明待证事实？证明力强弱？
- **标准**：真实程度、关联强度
- **结论**：证明力强/弱/不足
- **特点**：量的问题（强/弱），有大小之别

## 四、审查流程

```
                    开始
                      │
                      ▼
            ┌─────────────────┐
            │ 第一阶段：证据能力 │
            │ （三性审查）      │
            └────────┬────────┘
                     │
           ┌────────┴────────┐
           │三性均满足       │任一性不满足
           ▼                 ▼
     有证据能力          无证据能力
     （能进门）          （不能进门）
           │                 │
           ▼                 ▼
   ┌─────────────────┐   质证结束
   │ 第二阶段：证明力 │   证据排除
   │ （证明力评估）   │
   └────────┬────────┘
            │
            ▼
      证明力强/弱/不足
```

## 五、典型案例

### 非法证据不判断证明力

```
间谍器材偷录的录音证明被告承认欠款：

第一阶段：证据能力审查
• 真实性：录音内容真实
• 合法性：间谍器材偷录，违反禁止性规定 → 非法
• 结论：无证据能力

第二阶段：不得判断证明力
• 无证据能力，不得判断证明力
• 录音不被采纳，原告败诉
```

### 真实性存疑不判断证明力

```
借条签名存疑：

第一阶段：证据能力审查
• 真实性：签名存疑，需鉴定确认
• 结论：真实性未确认，证据能力待定

第二阶段：证明力无法判断
• 真实性未确认，证明力无法判断
• 需先鉴定确认真实性，再判断证明力
```

### 合法证据证明力判断

```
借条三性均满足：

第一阶段：证据能力审查
• 真实性：签名真实 ✓
• 合法性：取得合法 ✓
• 关联性：与借款有关 ✓
• 结论：有证据能力

第二阶段：证明力审查
• 借条证明借款存在
• 结论：证明力强
```

## 六、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 颠倒顺序 | 先判断证明力后判断证据能力 | 先判断证据能力，再判断证明力 |
| 非法证据判断证明力 | 非法证据无证据能力，不得判断证明力 | 无证据能力则证据排除，停止审查 |
| 混淆有/无与大/小 | 证据能力是质（有/无），证明力是量（大/小） | 区分质与量的问题 |
| 真实性存疑仍判断证明力 | 真实性未确认不能判断证明力 | 先确认真实性 |

## 七、与其他方法的衔接

**与三性审查的衔接**：
- 第一阶段（证据能力）→ 三性审查
- 第二阶段（证明力）→ 证明力评估

**与质证次序的衔接**：
- 证据能力阶段 → 关联性→合法性→真实性
- 证明力阶段 → 证明力

## 八、实务要点

### 判断顺序口诀

**先进门，后定案**：
- 先判断能否进门（证据能力）
- 后判断能否定案（证明力）

### 判断结论对应

| 证据能力 | 证明力 | 最终结论 |
|----------|--------|----------|
| 有 | 强 | 可采信且证明力强 |
| 有 | 一般 | 可采信但证明力一般 |
| 有 | 弱 | 可采信但需补强 |
| 无 | — | 应排除，不判断证明力 |
| 存疑 | — | 存疑需补强 |
