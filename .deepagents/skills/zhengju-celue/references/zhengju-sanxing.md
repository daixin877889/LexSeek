# 证据三性审查法

对每项证据进行真实性、合法性、关联性三维度审查。

## 一、核心原则

**证据三性缺一不可原则**：证据必须同时具备真实性、合法性、关联性才能进入诉讼。任一性缺失，证据即无证据能力。

**证据能力先于证明力原则**：先判断证据能力（能否进门），再判断证明力（能否定案）。无证据能力的证据不得判断证明力。

## 二、三性框架

```
真实性
├── 形式真实性：证据载体是否真实存在
└── 实质真实性：证据内容是否真实反映事实

合法性
├── 主体合法：取证主体是否适格
├── 取得方式合法：是否通过合法途径取得
├── 程序合法：取证程序是否合规
└── 形式合法：是否符合法定形式要求

关联性
├── 事实关联：与待证事实是否存在关联
└── 证明价值：关联程度强弱
```

## 三、真实性审查

### 形式真实性

**审查要点**：
- 原件/原物是否存在
- 复制件与原件是否一致
- 电子数据的完整性校验
- 签名/印章是否真实

**判断标准**：
| 情形 | 判断 |
|------|------|
| 提供原件 | 形式真实性强 |
| 仅提供复制件 | 形式真实性存疑，需补强 |
| 对方认可 | 可认定真实性 |
| 对方否认且无其他印证 | 真实性无法认定 |

### 实质真实性

**审查要点**：
- 内容是否完整、未被篡改
- 形成时间是否真实
- 形成主体是否真实

**常见风险**：
- 电子数据被篡改风险
- 录音录像剪辑风险
- 微信聊天记录删减风险

### 特殊证据的真实性审查

**电子数据真实性判断七要素**：
1. 系统环境完整性：生成、存储、传输的硬件软件环境是否完整可靠
2. 系统运行状态：是否正常运行，有无故障、崩溃、被攻击
3. 监测核查手段：有无哈希校验、日志记录、备份机制
4. 保存传输提取完整性：数据是否完整，有无截取、删减、篡改
5. 形成存储场景正常性：是否在正常往来活动中形成
6. 操作主体适当性：保存、传输、提取主体是否适当
7. 其他影响因素：时间戳可信度、加密强度

**特殊规则**：电子数据的打印件或其他可显示输出件视为原件。

**微信聊天记录真实性审查要点**：
- 是否提供原始载体（手机）
- 聊天记录是否完整
- 对方身份是否可确认

**常见问题**：
| 问题 | 后果 | 解决方案 |
|------|------|----------|
| 仅提供截图 | 真实性存疑 | 公证或出示原始载体 |
| 删减部分记录 | 证明力降低 | 完整展示 |
| 对方身份不明 | 关联性存疑 | 引导对方确认身份 |

## 四、合法性审查

### 收集手段合法性

**非法证据排除规则**：
- 严重侵害他人合法权益取得的证据：排除
- 违反法律禁止性规定取得的证据：排除
- 瑕疵取证：可能采信但证明力降低

**排除标准（需满足任一）**：
1. **严重侵害合法权益**：严重侵害隐私权、住宅安宁权等
2. **违反法律禁止性规定**：使用间谍器材、闯入住宅等法律明文禁止行为
3. **严重违背公序良俗**：取证行为严重违背社会公德

**不排除情形**：
- 一般性侵害（非严重侵害）
- 取证目的正当（维权而非恶意）
- 社会接受度高（社会公众可接受）

**判断三要素**：
| 要素 | 说明 |
|------|------|
| 取证目的正当性 | 维权vs恶意 |
| 侵害程度 | 一般侵害vs严重侵害 |
| 社会接受度 | 社会公众能否接受 |

**常见合法性审查**：

| 证据类型 | 合法性审查要点 |
|----------|----------------|
| 录音录像 | 是否在公开场合、是否侵害隐私 |
| 私自拍录 | 是否违反禁止性规定（间谍器材） |
| 私自调取 | 是否侵犯他人权益 |
| 诱取证据 | 是否超出必要限度 |

### 偷拍偷录合法性判断

**设备来源判断**：
- 普通手机/录音笔：通常合法
- 间谍器材（窃听器、针孔摄像头）：违法，排除

**取证场所判断**：
- 公共场所/对方同意的场所：通常合法
- 闯入住宅/秘密场所：严重侵害权益，排除

**典型案例判断**：
| 取证方式 | 合法性判断 |
|----------|------------|
| 手机偷录通话 | 取证目的维权，侵害程度一般，社会接受度高——合法 |
| 间谍器材偷录 | 违反法律禁止性规定——非法排除 |
| 闯入住宅偷录 | 严重侵害住宅安宁权——非法排除 |

### 证据形式合法性

**法定证据形式**：
- 书证
- 物证
- 视听资料
- 电子数据
- 证人证言
- 当事人陈述
- 鉴定意见
- 勘验笔录

**形式要求**：
- 电子数据：需保证完整性、可识别性
- 证人证言：证人需出庭接受质询
- 鉴定意见：鉴定机构和鉴定人需具备资质

**单位证明材料三要件**：
- 单位印章
- 负责人签名
- 经办人签名
三要件缺一则形式要件不合规，证据资格不被认可。

## 五、关联性审查

### 关联性判断

**审查步骤**：
1. 识别证据证明的内容
2. 确定案件待证事实
3. 判断证据内容与待证事实的关系
4. 评估关联程度

**关联程度分级**：

| 等级 | 描述 | 示例 |
|------|------|------|
| 直接关联 | 证据直接证明待证事实核心 | 合同原件证明合同关系 |
| 间接关联 | 证据佐证待证事实 | 履行行为佐证合同存在 |
| 无关联 | 与待证事实无关 | 与本案无关的合同 |

### 证明价值评估

**正面关联**：支持待证事实成立
**负面关联**：否定待证事实成立
**中立**：与待证事实无关

## 六、审查顺序

**质证次序规则**：关联性 → 合法性 → 真实性 → 证明力

1. **先质疑关联性**：关联性是证据的首要资格条件，无关联直接排除
2. **再质疑合法性**：合法性是对有关联证据的价值判断，非法证据排除
3. **再质疑真实性**：真实性审查难度最大，先排除无关联、非法证据后再审
4. **最后质疑证明力**：前三步通过后判断证明力强弱

**顺序逻辑**：由形式到内容，由资格到效力，节约司法资源。

## 七、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 证据形式要件与法律行为要件混淆 | 合同条款违法≠合同作为证据不合法 | 证据合法性审查证据本身，非证据记载内容 |
| 传真件效力等同原件误解 | 合同效力≠证据证明力 | 传真件合同有效不代表传真件等同于原件证据 |
| 偷拍偷录合法性判断不精确 | 误以为所有偷拍偷录都排除或都合法 | 违法取证需满足"严重"要件 |
| 手机录音一律排除 | 手机录音非间谍器材，通常合法 | 区分设备类型、侵害程度 |
| 电子数据原件认定误区 | 用传统书证原件概念套用电子数据 | 电子数据打印件视为原件 |
| 单位证明材料三要件缺失 | 仅盖章无签名 | 需印章+负责人签名+经办人签名 |

## 八、三性审查输出格式

```
证据名称：[证据名称]
证据类型：[类型]

真实性审查：
• 形式真实性：[√/存疑/×] 理由：...
• 实质真实性：[√/存疑/×] 理由：...
• 真实性综合评价：[强/一般/弱]

合法性审查：
• 收集手段：[合法/瑕疵/非法] 理由：...
• 证据形式：[符合/不符合] 理由：...
• 合法性综合评价：[可采信/存疑/排除]

关联性审查：
• 待证事实：[待证事实名称]
• 关联类型：[直接关联/间接关联]
• 关联程度：[强/一般/弱]

三性综合结论：[可采信且证明力强/可采信但证明力一般/存疑需补强/应排除]
```

## 九、与其他方法的衔接

三性审查完成后：
- 若三性均满足 → 有证据能力 → 进入证明力评估
- 若任一性不满足 → 无证据能力 → 证据排除
- 若存疑 → 需补强证据或进一步核实
