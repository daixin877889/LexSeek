# 质证意见六类型清单

质证时系统发表意见的六类标准清单。

## 一、质证意见类型

质证时，律师可系统发表的六类标准意见：

### 1. 无异议

全部认可证据，不提出质疑。

**适用情形**：
- 对方证据对己方有利
- 证据真实性、合法性、关联性均无问题
- 策略上选择认可

### 2. 逾期举证异议（证据资格异议）

证据逾期提交，主张证据失权不予采纳。

**适用条件**：
- 对方在举证期限届满后提交证据
- 己方不同意质证逾期证据

**法律后果**：证据可能失权，不予采纳

### 3. 关联性异议

证据与待证事实无关，不应进入诉讼。

**异议要点**：
- 证据与待证事实无实质联系
- 不能证明待证事实

**质证效果**：无关联证据直接排除，缩小法庭调查范围

### 4. 合法性异议

取证违法——主体不适格、取得方式违法、程序不合规、形式不合法。

**异议要点**：
- 取证主体不适格
- 取得方式违法（严重侵害权益/违反禁止性规定/严重违背公序良俗）
- 取证程序不合规
- 证据形式不合法

**质证效果**：非法证据排除，不得进入诉讼

### 5. 真实性异议

证据伪造变造——载体不真实、内容不真实。

**异议要点**：
- 证据载体不真实（伪造借条）
- 证据内容不真实（篡改记录）
- 需提供反证或申请鉴定

**质证效果**：真实性不成立，证据不被采纳

### 6. 证明力异议

证据证明力不足——真实性较低、关联性较弱、不能证明待证事实。

**异议要点**：
- 证据真实程度较低
- 证据关联强度较弱
- 不能充分证明待证事实

**质证效果**：证明力减弱，需其他证据补强

## 二、组合使用

可根据具体情况组合发表多种异议：

| 组合示例 | 适用情形 |
|----------|----------|
| 真实性异议+证明力异议 | 证据存疑且证明力不足 |
| 合法性异议+真实性异议 | 取证违法且内容存疑 |
| 逾期举证异议+真实性异议 | 逾期提交且证据存疑 |
| 关联性异议+证明力异议 | 与待证事实关联弱 |

## 三、质证意见表达格式

### 标准格式

```
对证据[证据名称]的质证意见：

1. 逾期举证异议：[是否逾期，己方是否同意质证]
2. 关联性异议：[与待证事实的关系，是否有关联]
3. 合法性异议：[取证方式、程序、形式是否合法]
4. 真实性异议：[载体和内容是否真实]
5. 证明力异议：[证明力强弱，能否证明待证事实]

综合意见：[采纳/不采纳/存疑需补强]
```

### 简要格式

```
对证据[证据名称]：[异议类型]。

理由：[异议依据]。

建议：[不采纳/需补强/无异议]。
```

## 四、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 质证意见表述不严谨 | 仅说"有异议"不说明异议类型和依据 | 明确异议类型+说明异议依据 |
| 不区分逾期举证异议与证据资格异议 | 逾期举证异议是证据资格异议的一种 | 明确逾期事实+己方不同意质证 |
| 存疑视听资料一律需补强 | 无疑点的视听资料可单独定案 | 区分存疑与否 |
| 有利证据仍提异议 | 对己方有利证据可选择无异议 | 策略性选择无异议 |

## 五、典型案例

### 微信聊天记录质证

```
对微信聊天记录截图的质证意见：

1. 逾期举证异议：未逾期提交
2. 关联性异议：与借款事实有关联
3. 合法性异议：取证方式合法
4. 真实性异议：仅提供截图，未提供原始载体核对，真实性存疑
5. 证明力异议：若真实性无法确认，证明力不足

综合意见：存疑需补强，要求提供原始载体（手机）现场演示核对
```

### 借条质证

```
对借条的质证意见：

1. 逾期举证异议：未逾期提交
2. 关联性异议：与借款事实有关联
3. 合法性异议：取证方式合法
4. 真实性异议：签名肉眼比对存疑，需笔迹鉴定验证
5. 证明力异议：签名不真实则证明力为零

综合意见：存疑，申请笔迹鉴定或要求对方申请鉴定
```

### 偷录录音质证

```
对偷录通话录音的质证意见：

1. 逾期举证异议：未逾期提交
2. 关联性异议：与欠款承认事实有关联
3. 合法性异议：手机录音非间谍器材，取证目的维权，不严重侵害权益，合法
4. 真实性异议：录音完整无剪辑痕迹，真实性成立
5. 证明力异议：录音内容证明对方承认欠款，证明力强

综合意见：合法且真实，可采纳，证明力强
```

## 六、与三性审查的衔接

先用三性框架分析，再用六类型清单表达：

```
三性审查 → 六类型意见表达
─────────────────────────────
真实性审查 → 真实性异议
合法性审查 → 合法性异议
关联性审查 → 关联性异议
证明力评估 → 证明力异议
逾期问题 → 逾期举证异议
全部认可 → 无异议
```

## 七、质证次序

按法定顺序发表质证意见：关联性 → 合法性 → 真实性 → 证明力
