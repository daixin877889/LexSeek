# 自认制度

诉讼内自认与诉讼外自认的区分及效力判断。

## 一、自认的概念

自认是当事人对不利于己的事实的承认。

## 二、自认类型区分

### 诉讼内自认

| 要件 | 说明 |
|------|------|
| 场合要件 | 在本案诉讼过程中作出（庭审、庭前会议、书面诉讼材料） |
| 对象要件 | 向本案审判法官作出 |
| 内容要件 | 对己不利事实的承认 |
| 意思要件 | 明确作出承认 |

**效力**：直接免除对方举证责任，对方无需举证证明自认内容

### 诉讼外自认

| 特点 | 说明 |
|------|------|
| 场合 | 诉讼外场合作出（私下、在其他案件、在其他场合） |
| 效力 | 不免除举证，仅作为证据参考 |

**效力**：不免除举证，由法庭结合其他事实认定其证明效力

## 三、自认免证要件

诉讼内自认产生免证效力，需四要件同时满足：

### 要件1：场合要件

- 是否在本案诉讼过程中？
- 庭审？庭前会议？书面诉讼材料？

**判断**：若非诉讼场合 → 诉讼外自认，不产生免证效力

### 要件2：对象要件

- 是否向本案审判法官作出？

**判断**：若非本案法官 → 诉讼外自认，不产生免证效力

### 要件3：内容要件

- 是否对己不利事实的承认？
- 承认的是不利事实还是有利事实？

**判断**：若非不利承认 → 非自认，不产生自认效力

### 要件4：意思要件

- 是否明确作出承认？

**判断**：需明确承认，默示不明确

## 四、自认效力

### 免证效力

诉讼内自认成立后：
- 对方无需举证证明自认内容
- 法院直接认定自认事实为真
- 自认方不得反悔（除符合撤回条件）

### 不产生免证效力

| 情形 | 说明 |
|------|------|
| 诉讼外自认 | 私下、另案承认 |
| 非不利承认 | 承认有利事实 |
| 非向本案法官作出 | 向其他人承认 |
| 非明确承认 | 模糊表态 |

## 五、典型案例

### 庭审承认欠款免证

```
被告在庭审中承认欠款10万元：

要件检验：
• 要件1：庭审（本案诉讼过程中） ✓
• 要件2：向审判法官 ✓
• 要件3：欠款（对被告不利） ✓
• 要件4：明确承认 ✓

结论：四要件满足，免证效力成立
结果：原告无需举证，法院直接认定欠款10万元
```

### 私下承认欠款不免证

```
被告私下向原告承认欠款：

要件检验：
• 要件1：私下（非诉讼场合） ✗

结论：四要件不满足，诉讼外自认，不产生免证效力
结果：原告仍需举证证明欠款事实，私下承认仅作证据参考
```

### 在其他案件承认不免证

```
被告在另案中承认欠款：

要件检验：
• 要件1：另案诉讼（非本案） ✗
• 要件2：向另案法官（非本案法官） ✗

结论：诉讼外自认，不产生免证效力
结果：原告仍需举证，另案承认仅作证据参考
```

### 承认有利事实不免证

```
被告在庭审中承认产品质量好：

要件检验：
• 要件3：产品质量好（对被告有利，非不利承认） ✗

结论：非不利承认，不产生自认效力
结果：非不利承认，不产生自认效力
```

## 六、自认风险提示

### 庭审表态风险

| 风险行为 | 后果 | 预防措施 |
|----------|------|----------|
| 庭审中承认不利事实 | 产生免证效力，对方无需举证 | 谨慎表态，避免不利承认 |
| 书面诉讼材料承认不利事实 | 产生免证效力 | 审慎起草诉讼材料 |
| 模糊表态被认定为自认 | 可能产生自认效力 | 明确表达，避免歧义 |

### 自认后不得反悔

自认产生免证效力后，自认方不得反悔，除非符合撤回条件：
- 重大误解
- 受胁迫

## 七、自认运用策略

### 主张自认方的策略

| 情形 | 策略 |
|------|------|
| 对方庭审承认不利事实 | 主张诉讼内自认免证，无需举证 |
| 对方私下承认 | 主张诉讼外自认作证据参考，仍需举证 |

### 可能自认方的策略

| 风险 | 预防 |
|------|------|
| 庭审中可能承认不利事实 | 谨慎表态，避免不利承认 |
| 书面诉讼材料可能承认不利事实 | 审慎起草，避免不利表述 |

## 八、自认输出格式

```
自认判断：

承认内容：[承认的事实内容]
承认场合：[庭审/私下/另案等]

要件检验：
• 场合要件：[是否本案诉讼过程中]
• 对象要件：[是否向本案法官]
• 内容要件：[是否不利承认]
• 意思要件：[是否明确承认]

自认类型：[诉讼内自认/诉讼外自认]
自认效力：[免证/不免证]

举证建议：[是否仍需举证]
```

## 九、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 私下承认当作自认 | 私下场合的承认是诉讼外自认 | 区分场合 |
| 在其他案件承认当作自认 | 另案承认是诉讼外自认 | 区分案件 |
| 非不利承认当作自认 | 承认有利事实不是自认 | 区分是否不利 |
| 自认后反悔 | 自认产生免证效力后不得反悔 | 谨慎表态 |

## 十、与否认/抗辩的衔接

自认与否认/抗辩的关系：

| 类型 | 说明 |
|------|------|
| 自认 | 承认对方主张的不利事实 |
| 否认 | 不承认对方主张的事实 |
| 抗辩 | 承认对方事实+提出新事实阻却 |
