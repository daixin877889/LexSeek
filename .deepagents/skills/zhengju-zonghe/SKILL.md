# 证据分析双模式方法论

## 核心原则

证据分析以"路径判断→框架选择→逐项审查→综合判断"为逻辑主线。

分析方法遵循四项基本原则，依路径切换：

1. **路径优先原则**：案件类型（民事/刑事）+ 任务方向（举证/质证）决定分析框架，先判断路径再执行分析

2. **民事举证以关联性为先**：审查顺序为关联性→合法性→真实性→证明力，证明标准为高度盖然性（~75%）

3. **刑事质证以合法性为门槛**：证据能力（合法性）不过，不审查证明力。审查顺序为合法性→真实性→关联性

4. **举证责任与证明标准随类型切换**：民事谁主张谁举证（~75%），刑事控方独担证明责任（排除合理怀疑~99%）

## 路径矩阵

|  | 民事 | 刑事 |
|------|------|------|
| **举证** | 路径A：民事举证 | 路径C：刑事举证（少用） |
| **质证** | 路径B：民事质证 | 路径D：刑事质证 |

路径判断由agent自行执行，详见提示词步骤0。

## 路径A：民事举证

### 核心规则

1. **情形判断**：根据材料状况选择推演/混合/实证/判决分析模式
2. **审查顺序**：关联性→合法性→真实性→证明力
3. **证明标准**：高度盖然性（75%以上可能性）
4. **举证责任**：谁主张谁举证

### 执行步骤

```
1. 情形判断 → 推演/混合/实证/判决分析
2. 证据清单整理 → 已有证据 + 待补证据（实证模式标注原件/公证状态）
3. 三性审查 → 关联性 → 合法性 → 真实性
4. 证明力评估 → 单证 + 组合
5. 证明力分析报告 → 以事实为纲逐证展开八维度分析，区分本证/反证举证标准
6. 证据链构建 → 四步法（核心观点→证据组合→逻辑阐述→风险反驳）
7. 缺口识别 → 补强建议
8. 策略制定 → 主线 + 辅线 + 突破口
9. 取证计划 → 先行 + 诉讼
```

详细业务规则见 [references/民事举证.md](./references/民事举证.md)。

## 路径B：民事质证

### 核心规则

1. **审查顺序**：关联性→合法性→真实性→证明力
2. **质证意见六类型**：无异议、逾期举证异议、关联性异议、合法性异议、真实性异议、证明力异议
3. **组合使用**：可根据具体情况组合发表多种异议

### 执行步骤

```
1. 逐份审查对方证据
2. 关联性异议 → 合法性异议 → 真实性异议
3. 证明力异议
4. 质证意见汇总
```

详细业务规则见 [references/民事举证.md](./references/民事举证.md) 质证部分。

## 路径C：刑事举证

### 核心规则

1. **仅用于辩方主动提交有利证据**（不在场证明、无罪证据等）
2. **被告人无自证其罪义务**
3. **审查顺序**：真实性优先（辩方证据合法性门槛较低）

### 执行步骤

```
1. 证据清单 → 仅列有利证据
2. 三性审查 → 真实性优先
3. 证明力评估
4. 提交时机建议
```

---

## 路径D：刑事质证

### 核心规则

1. **合法性为门槛**：先审查证据能力（合法性），再审查证明力（真实性、关联性）
2. **控方独担证明责任**：辩方只需提供非法取证线索
3. **存疑有利于被告人**：证据无法排除合理怀疑时，作出有利于被告人的认定
4. **孤证不能定案**：仅适用于对被告人不利的事实
5. **非法证据应当排除**：即使与其他证据相互印证，仍应排除
6. **客观证据优于主观证据**：物证优于言词，录像优于笔录

### 执行步骤

```
1. 判断证据类型 → 实物/鉴定/证言/勘验/辨认/供述
2. 选择审查框架 → 按证据类型选择
3. 审查合法性（证据能力）→ 鉴真、程序合法性、供述合法性
4. 若证据能力不具备 → 建议排除，终止后续审查
5. 若证据能力具备 → 审查真实性 → 审查关联性
6. 综合运用证据规则 → 孤证判断、印证判断、口供补强、证明标准
7. 给出质证结论 → 采信/排除/存疑
```

详细业务规则见 [references/刑事质证.md](./references/刑事质证.md)。

## 关键要点

### 一、情形判断（民事举证）

根据用户材料状况选择分析模式：

| 情形 | 判断标准 | 分析模式 |
|------|----------|----------|
| 情形一 | 未提供具体证据文件 | 推演模式 |
| 情形二 | 提供少量、零散证据 | 混合模式 |
| 情形三 | 提供较充分的真实证据 | 实证模式 |
| 情形四 | 提供法院判决文书 | 判决分析模式 |

### 二、证据三性审查（民事）

详见 [references/民事举证.md](./references/民事举证.md)

- **关联性**：与待证事实的关联程度（直接/间接/无）
- **合法性**：主体合法+取得方式合法+程序合法+形式合法
- **真实性**：形式真实性（载体真实）+实质真实性（内容真实）
- **审查顺序**：关联性→合法性→真实性→证明力

### 三、证据能力与证明力分离

- **证据能力**：能否进入诉讼（质的问题，有/无）
- **证明力**：能否证明待证事实（量的问题，强/弱）
- **两阶段审查**：先判断证据能力，再判断证明力

### 四、证明力评估与证明力分析报告（民事）

详见 [references/民事举证.md](./references/民事举证.md)

- **单证证明力**：强/一般/弱三档评估
- **组合证明力**：证据间的补强与印证关系
- **高度盖然性判断**：是否达到高度可能性标准

**证明力分析报告**：以事实为纲，逐证展开八个分析维度（证据类型/是否原件/证据来源/证明事实/关联性/证明力度/证据风险/补充证据建议），嵌入本证/反证举证标准区分。

**本证与反证举证标准**：

| 类型 | 举证标准 |
|------|----------|
| 原告本证（权利产生要件） | 高度盖然性——须使法庭确信事实存在具有高度可能性 |
| 被告反证（权利妨碍抗辩） | 仅需真伪不明——使待证事实产生合理怀疑即可 |
| 被告本证（权利消灭/限制） | 高度盖然性——被告提出新主张须达到本证标准 |

### 五、证据链构建（民事）

详见 [references/民事举证.md](./references/民事举证.md)

- **四步法结构**：核心观点→证据组合→逻辑阐述→风险反驳
- **混合拼图模式**：已有证据与待补证据的结合论证
- **完整性检验**：验证证据链是否完整覆盖待证事实

### 六、质证意见六类型（民事质证）

详见 [references/民事举证.md](./references/民事举证.md)

1. 无异议
2. 逾期举证异议
3. 关联性异议
4. 合法性异议
5. 真实性异议
6. 证明力异议

### 七、刑事证据类型审查（刑事质证）

详见 [references/刑事质证.md](./references/刑事质证.md)

- **实物证据**→先鉴真（独特性辨认+保管链验证）
- **鉴定意见**→六步质证（主体→检材→程序→方法→内容→关联）
- **证人证言**→四维度审查（利害关系→证言矛盾→印证→常理）
- **勘验笔录**→六项审查（主体→见证人→时间→内容→签名→附件）
- **辨认笔录**→六情形审查（详见 references/程序审查专项.md）
- **被告人供述**→合法性审查（刑讯逼供/威胁/引诱/疲劳审讯）

### 八、综合证据规则（刑事质证）

详见 [references/刑事质证.md](./references/刑事质证.md) 和 [references/证据定案规则.md](./references/证据定案规则.md)

- **孤证判断**：形式判断→实质判断→关联判断
- **印证判断**：寻找重合点→检验相互验证
- **口供补强**：只有被告人供述不能定罪
- **证明标准**：全面性→合法性真实性→排他性
- **存疑有利**：事实存疑→金额存疑→数量存疑

### 九、非法证据排除（刑事质证）

详见 [references/排除规则.md](./references/排除规则.md)

- **强制排除**：来源不明、刑讯逼供、威胁、引诱、欺骗
- **见证人不适格**：辅警、聘用人员、侦查机关人员
- **鉴定超范围**：金额认定、折算等法律裁判事项
- **辨认未混杂**：人<7、照片<10、物品<5

### 十、证据权重规则（刑事质证）

详见 [references/证据权重与优先.md](./references/证据权重与优先.md)

- **客观优于主观**：物证优于言词，鉴定优于供述
- **录像优于笔录**：实质性差异时以录像为准
- **鉴真是前提**：未鉴真无证据能力

## 边界说明

**适用场景**：
- 民事案件证据整理、审查、举证策略
- 民事案件质证对方证据
- 刑事案件辩方质证控方证据
- 刑事案件辩方提交有利证据

**不适用场景**：
- 纯信息查询（"什么是证据"）→直接回答概念
- 与证据分析无关的请求
- 控方视角的刑事证据分析

**常见失败模式**：
- 不判断路径直接分析（民事方法用于刑事或反之）
- 刑事质证跳过合法性直接审证明力
- 民事举证审查顺序错误（应先关联性后合法性）
- 混淆证据能力与证明力
- 以印证采信非法证据
- 误用刑事排除合理怀疑标准于民事案件
- 要求被告人证明刑讯逼供

## 与其他模块的配合

1. **前置依赖**：证据分析服务于请求权检视中的要件证明
2. **举证对应**：请求权检视确定举证责任分配，证据分析执行举证
3. **策略联动**：请求权成立与否影响诉讼策略方向

配合流程：
```
请求权基础模块 → 确定要件结构和举证分配
     ↓
证据分析模块 → 证据审查、链构建、策略制定
     ↓
抗辩分析模块 → 对抗性攻防推演
```
