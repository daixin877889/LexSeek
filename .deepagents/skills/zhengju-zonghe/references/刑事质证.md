# 刑事质证方法论

## 一、核心原则

### P1：合法性为门槛

先审查证据能力（合法性），再审查证明力（真实性、关联性）。证据能力不具备的证据，不得继续审查证明力，应直接建议排除。

### P2：控方独担证明责任

被告人不负自证其罪的责任。控方必须"排除合理怀疑"地证明犯罪事实（~99%）。非法证据排除程序中，辩方只需提供线索，控方承担证明取证合法性的责任。

### P3：孤证不能定案

单个证据不能单独作为认定有罪事实的根据。虽有多份证据但与主要事实有关联的只有一项时，实质仍是孤证。此规则仅适用于对被告人不利的事实。

### P4：存疑有利于被告人

证据无法排除合理怀疑时，作出有利于被告人的认定。事实存疑→认定不存在；金额存疑→以重合部分认定；数量存疑→以最低数量认定。

### P5：非法证据应当排除

刑讯逼供、威胁、引诱、欺骗等非法方法取得的证据必须排除。即使与其他证据相互印证，仍应排除（印证解决真实性，不解决合法性）。

### P6：客观证据优于主观证据

言词证据与物证矛盾时以物证为准。笔录与录像不一致时以录像为准。

## 二、证据类型判断与审查框架选择

### 步骤1：判断证据类型

确认证据属于以下哪一类：
- 物证/书证 → 先鉴真，再审查内容
- 鉴定意见 → 六步质证
- 证人证言 → 四维度审查
- 勘验/检查笔录 → 六项审查
- 辨认笔录 → 六情形审查
- 被告人供述 → 合法性优先审查
- 电子数据 → 鉴真+完整性审查

### 步骤2：选择审查框架

| 证据类型 | 审查框架 | 详细参考 |
|----------|----------|----------|

| 鉴定意见 | 六步质证（主体→检材→程序→方法→内容→关联） | 基础审查框架.md 鉴定部分 |

## 三、合法性审查（证据能力门槛）

### 实物证据鉴真

鉴真解决同一性问题：法庭出示的这份证据，就是当初提取的那份吗？

**鉴真两步法**：

**第一步：独特性辨认**
通过外部特征识别证据同一性：
- 形状、颜色、签名、编号
- 现场照片、标签位置
- 特殊标记

**第二步：保管链验证**
追踪证据从提取到出示的全流程：
提取笔录 → 扣押清单 → 保管记录 → 送检记录 → 鉴定意见 → 法庭出示

审查各环节记录是否完整、时间间隔是否合理、标签位置有无变化、性状有无变化。

**鉴真失败后果**：证据来源不明，无证据能力，强制排除，不可补正。

### 程序合法性审查

**取证主体**：
- 是否侦查人员（≥2人）
- 是否持证
- 是否具有管辖权

**见证人适格性**：
- 是否与案件无关
- 非辅警、非聘用人员、非侦查机关人员
- 见证人应实际到场

**批准手续**：
- 搜查、扣押、技术侦查等是否有批准文件
- 批准文件是否合法有效

### 供述合法性审查

**审查重点**：
- 有无刑讯逼供（肉刑、变相肉刑）
- 有无威胁（以损害本人或近亲属合法权益相威胁）
- 有无引诱、欺骗（以非法利益相引诱）
- 有无非法拘禁（未依法采取强制措施）
- 有无疲劳审讯（连续讯问超过12小时）
- 笔录与同步录音录像是否一致

**审查方式**：
- 申请调取同步录音录像
- 对比笔录记载时间与录像时间
- 形成录像文字稿，逐处对比差异
- 申请侦查人员出庭说明

## 四、各类证据审查要点

### 鉴定意见：六步质证

详见 [基础审查框架.md](./基础审查框架.md) 鉴定部分。

1. **主体审查**：鉴定机构是否在司法鉴定名录，鉴定人专业是否对口
2. **检材审查**：检材是否经过鉴真，保管链是否完整，是否被污染
3. **程序审查**：是否送达被告人，有无复核程序
4. **方法审查**：方法是否科学，抽样数量是否足够
5. **内容审查**：结论是否明确（不含"可能""推断"），是否超出鉴定范围（金额认定、折算等法律事项属超范围）
6. **关联审查**：与案件事实有无关联，与其他证据是否矛盾

### 证人证言：四维度审查

详见 [基础审查框架.md](./基础审查框架.md) 证人部分。

1. **利害关系**：证人与当事人有无利害关系
2. **证言矛盾**：证言前后是否一致，与其他证据是否矛盾
3. **印证情况**：能否与其他证据相互印证
4. **常理判断**：是否符合常识常理（如500米距离、夜晚能否"亲眼看到"）

**传闻证据不可采**："听说""传说"类表述的证据不得作为定案根据。

### 勘验/检查笔录：六项审查

详见 [基础审查框架.md](./基础审查框架.md) 勘验部分。

1. **主体**：是否侦查人员持证，≥2人
2. **见证人**：是否适格，非辅警/聘用人员
3. **时间**：是否及时进行
4. **内容**：是否现场制作（非事后补写）
5. **签名**：相关人员是否实际到场签名
6. **附件**：现场图/照片/录像与笔录是否一致

### 辨认笔录：六情形审查

详见 [程序审查专项.md](./程序审查专项.md) 辨认部分。

辨认存在以下任一情形，不得作为定案根据：
1. 非侦查人员主持辨认
2. 辨认前使辨认人见到辨认对象
3. 未个别进行辨认
4. 未混杂（人<7、照片<10、物品<5）
5. 有明显暗示嫌疑（如辨认照片为入所照片）
6. 其他存疑情形

### 电子数据审查

- 提取程序是否合法
- 是否扣押原始存储介质
- 有无完整性校验（哈希值/MD5）
- 提取笔录是否完整记载

## 五、真实性审查

### 审查内容

- 证据来源是否清楚
- 内容是否完整、有无篡改
- 与其他证据能否印证

### 审查要点

| 审查项 | 审查内容 | 问题信号 |
|--------|----------|----------|
| 来源情况 | 证据从何而来 | 来源不明→真实性存疑 |
| 内容完整性 | 有无删减、篡改、污染 | 有编辑痕迹→真实性存疑 |
| 印证情况 | 与其他证据是否一致 | 矛盾→需分析原因 |

---

## 六、关联性审查

- 证据与待证事实有无实质联系
- 能否直接证明犯罪构成要件事实
- 关联程度（直接关联/间接关联/无关联）

---

## 七、综合证据规则运用

### 孤证判断

三步判断法：
1. **形式判断**：与主要事实有关的证据有几份？
2. **实质判断**：多份证据是否同出一源？
3. **关联判断**：是否只有一份证据与主要事实直接关联？

**结论**：孤证不能定案（仅适用于对被告人不利的事实）。

### 口供补强

- 只有被告人供述，没有其他证据的，不能认定被告人有罪
- 同案犯供述不能互为补强
- 补强证据必须是口供以外的其他证据

### 印证判断

- 证据之间能否相互验证
- 印证解决真实性，不解决合法性
- 非法证据即使与其他证据相互印证，仍应排除

### 一对一证据处理

指控方一证据、辩护方一否认的情形：
- 本质是孤证，概率仅50%
- 存疑有利于被告人，认定该项事实不存在

### 证明标准：三层递进

1. **全面性**：每个待证事实是否有证据证明
2. **合法真实**：证据是否经法定程序查证属实
3. **排他性**：是否排除合理怀疑，结论唯一

死刑案件适用"排除一切合理怀疑"标准。

### 间接证据定案五条件

1. 证据已经查证属实
2. 证据之间相互印证
3. 证据链完整，不存在无法排除的矛盾
4. 根据证据得出的结论足以排除合理怀疑
5. 推理符合逻辑和经验法则

## 八、质证结论

### 结论类型

| 结论 | 适用情形 |
|------|----------|
| 采信 | 合法性具备、真实性可靠、关联性成立、达到证明标准 |
| 排除 | 证据能力不具备（非法取证/来源不明/鉴真失败） |
| 存疑 | 证据能力具备但证明力存疑，或需进一步调查核实 |

### 后续行动建议

质证结论为排除或存疑时，应给出后续行动建议：

| 行动类型 | 适用情形 |
|----------|----------|
| 申请非法证据排除 | 存在刑讯逼供、程序违法等 |
| 申请调取同步录音录像 | 笔录与供述存疑 |
| 申请证人出庭 | 关键证人有必要出庭质询 |
| 申请重新鉴定 | 鉴定意见存在严重问题 |
| 申请侦查人员出庭 | 需说明取证过程 |

## 十、常见失败模式

| 失败模式 | 说明 | 正确做法 |
|----------|------|----------|
| 跳过合法性直接审证明力 | 对来源不明证据仍判断真实性 | 先审查合法性，证据能力不具备即终止 |
| 以印证采信非法证据 | 认为与其他证据印证即可采信 | 印证解决真实性不解决合法性 |
| 忽视鉴真环节 | 直接质证内容，未审查同一性和保管链 | 实物证据必须先鉴真 |
| 举证责任倒置 | 要求被告人证明刑讯逼供 | 辩方只需提供线索，控方证明合法性 |
| 拘泥三性忽视综合规则 | 机械套用三性框架，不运用孤证/补强规则 | 综合运用各项证据规则 |
| 混淆证据能力与证明力 | 质证时混淆"能不能用"和"能证明多少" | 先能力后证明力，层级分明 |
