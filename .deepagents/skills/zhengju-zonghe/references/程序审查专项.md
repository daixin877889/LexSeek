# 程序审查专项

## 一、辨认笔录审查

审查辨认笔录的程序合法性，六个排除情形。

### 六个排除情形

#### 1. 非侦查人员主持

**认定标准**：
- 辨认必须由侦查人员主持
- 非侦查人员主持无效

#### 2. 辨认前见到辨认对象

**认定标准**：
- 在正式辨认程序前让辨认人见到嫌疑人照片或本人
- 污染辨认结果

**后果**：辨认笔录不得作为定案根据

#### 3. 辨认没有个别进行

**认定标准**：
- 多人辨认同一对象时必须逐一单独进行
- 集体辨认无效

#### 4. 辨认对象未混杂或数量不符规定

**数量要求**：
- 人辨认 ≥ 7人
- 照片辨认 ≥ 10张
- 物品辨认 ≥ 5件

**认定标准**：
- 未混杂或数量不足 → 无效

#### 5. 有明显暗示或指认嫌疑

**认定标准**：
- 辨认照片有暗示性背景（如入所照片）
- 侦查人员有暗示性言语

**后果**：辨认笔录不得作为定案根据

#### 6. 其他无法确定真实性的情形

### 执行步骤

1. **审查辨认主持人**
2. **审查辨认前是否见过对象**
3. **审查是否个别进行**
4. **审查混杂数量**
5. **审查有无暗示嫌疑**
6. **给出结论**

## 二、同步录音录像运用

运用同步录音录像审查讯问合法性或供述真实性。

### 核心原则

同步录音录像是最优证据，比笔录更完整、真实、客观。

### 四步运用方法

#### 1. 申请调取

**适用情形**：
- 证明取证合法性
- 证明供述真实性

**操作要点**：
- 向法庭提交调取录像申请
- 说明调取目的和理由

#### 2. 详细观看

**操作要点**：
- 不能只看摘要，要逐字逐句观看
- 录像显示的内容才是真实内容

#### 3. 形成文字稿，对比笔录

**操作要点**：
- 将录像内容逐字记录
- 与讯问笔录逐项对比

#### 4. 发现差异，揭露非法

**审查内容**：
- 笔录与录像是否一致
- 录像是否显示疲劳审讯、威胁、诱导

### 实质性差异判断

**什么是实质性差异**：
- 影响定罪量刑的内容差异
- 如：笔录记载"认罪"，录像显示"否认"
- 如：笔录未记载威胁，录像显示有威胁

**处理原则**：
- 实质性差异 → 以录像为准

### 关键规则

检方不提供录像，推定对控方不利。

### 执行步骤

1. **申请调取录像**
2. **详细观看形成文字稿**
3. **对比笔录内容**
4. **判断差异性质**
5. **揭露非法行为**
6. **给出结论**

## 三、非法证据排除申请

申请排除非法证据的流程。

### 核心原则

证明责任在控方，辩方只需提供线索。

### 五步流程

#### 1. 提出申请

**辩方职责**：
- 提供线索或材料
- 说明怀疑依据

**不需要**：
- 不需要证明非法
- 只需说明时间、地点、何种非法方法

#### 2. 说明理由

**说明内容**：
- 具体说明何种非法方法
- 刑讯逼供、威胁、引诱、欺骗、非法拘禁等

#### 3. 启动调查

**检察院职责**：
- 承担证明取证合法的责任
- 辩方不承担证明非法的责任

#### 4. 调查程序

**原则**：
- 先行当庭调查为原则
- 先解决证据合法性，再进入事实认定

#### 5. 裁决结果

**规则**：
- 控方不能证明合法性 → 排除
- 检方不提供录像 → 推定对控方不利

### 非法取证类型

#### 刑讯逼供

**认定标准**：
- 肉体或精神上的折磨

#### 威胁

**认定标准**：
- 以不利后果威胁被告人
- 如"不认罪就抓你老婆"

#### 引诱

**认定标准**：
- 以利益诱导被告人供述

#### 欺骗

**认定标准**：
- 以虚假信息欺骗被告人

#### 非法拘禁

**认定标准**：
- 限制人身自由

#### 程序违法

**认定标准**：
- 扣押程序违法、见证人不适格等

### 疲劳审讯判断

**边界问题**：
- 连续讯问20小时是否属于非法方法？
- 需具体情况判断

### 执行步骤

1. **判断非法方法类型**
2. **准备线索材料**
3. **起草排除申请**
4. **申请调取录像**（若涉及供述）
5. **准备质证意见**
