# 法律法规搜索功能使用说明

## 功能概述

法律法规搜索功能为用户提供了一个专业的法律检索界面，支持法律法规和法条的多维度搜索与浏览。

## 主要功能

### 1. 统计面板
- 显示法律、行政法规、司法解释、指导意见的数量统计
- 支持加载状态的骨架屏显示

### 2. 筛选功能
- **关键词搜索**：支持按法律名称或文号搜索
- **类型筛选**：法律、行政法规、司法解释、指导意见
- **发文机关筛选**：按发文机关过滤
- **生效状态筛选**：有效、已失效、全部状态
- **日期范围筛选**：按发布日期范围过滤

### 3. 列表展示
- **PC 端**：表格布局，显示名称、类型、发文机关、发布日期、生效状态
- **移动端**：卡片布局，适配触摸操作
- **分页功能**：支持页码跳转和页面大小设置

### 4. 法律全文预览
- **条文导航**：按层级结构（编、章、节、条）展示
- **内容高亮**：支持搜索关键词高亮显示
- **响应式设计**：桌面端侧边栏，移动端全屏模式

### 5. 法条搜索
- **语义搜索**：使用向量搜索技术，支持语义相关的法条查找
- **结果展示**：显示法条内容、所属法律名称和章节层级
- **筛选功能**：支持按法律类型和生效状态筛选

## 使用流程

### 1. 基本搜索
1. 在搜索框中输入法律名称或文号
2. 点击"搜索"按钮或按回车键
3. 查看搜索结果列表

### 2. 高级筛选
1. 点击筛选面板中的各个筛选条件
2. 系统自动更新搜索结果
3. 使用"重置"按钮清空所有筛选条件

### 3. 查看法律全文
1. 点击列表中的法律法规项目
2. 在预览面板中查看法律全文
3. 使用条文导航快速跳转到指定条文

### 4. 法条搜索
1. 在法条搜索面板中输入相关描述
2. 系统返回语义相关的法条
3. 点击法条结果查看完整法律条文

## 移动端特殊功能

### 浮动操作按钮
- **筛选按钮**：打开筛选抽屉
- **法条搜索按钮**：打开法条搜索面板

### 抽屉面板
- **筛选抽屉**：底部滑出，包含所有筛选选项
- **法条搜索抽屉**：右侧滑出，提供法条搜索功能
- **预览全屏**：法律全文以全屏模式展示

## 技术特性

### 性能优化
- 骨架屏加载状态
- 分页加载，避免一次性加载大量数据
- 响应式图片和内容

### 用户体验
- 搜索关键词高亮
- 友好的空状态提示
- 错误状态重试功能
- 触摸友好的移动端交互

### 无障碍支持
- 语义化 HTML 结构
- 键盘导航支持
- 屏幕阅读器友好

## 数据来源

- **法律法规数据**：来自 `legalMain` 表
- **法条数据**：来自 `legalArticles` 表
- **向量搜索**：基于 `lawEmbeddings` 表的向量数据

## API 接口

- `GET /api/v1/legal/statistics` - 获取统计数据
- `GET /api/v1/legal/list` - 获取法律法规列表
- `GET /api/v1/legal/:id` - 获取法律法规详情
- `POST /api/v1/legal/search-articles` - 法条搜索
- `GET /api/v1/legal/issuing-authorities` - 获取发文机关列表

## 注意事项

1. **菜单权限**：需要执行 `scripts/add-legal-menu.sql` 脚本添加菜单权限
2. **数据依赖**：需要确保法律数据已导入到数据库
3. **向量搜索**：法条搜索功能依赖向量数据，需要先进行向量化处理
